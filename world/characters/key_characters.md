## 1. 绝对主角 (The Absolute Protagonist)

### 00. 林微 (Lin Wei) - 灵衍行者 / 万象探索者

| 维度     | 描述                                                                 |
| :------- | :------------------------------------------------------------------- |
| **身份** | **[穿越者/开拓者]** 拥有现代思维与**【识海悬镜】**的冒险家。她不仅仅是观察者，更是**入局者**。 |
| **人设** | **正直机智，百折不挠**。她对未知充满好奇，面对绝境从不言弃。| 
| **形态** | **【探索者】**(Vol 1-2)：乐观、正直、充满好奇。<br>**【狩猎者】**(Vol 3 黑化期)：极度理智、冷血、功利主义。<br>**【万象之主】**(Vol 3 后期)：成熟、包容、悲悯。 |
| **外貌** | **(中文提示词)**: 1个小女孩，单人，略显苍白的皮肤，精致的五官，坚定的眼神，身穿改良的青色儒袍（便于行动），手持一面散发微光的古朴铜镜，古风建筑背景，大气的光影，高画质，细节丰富。 |
| **核心** | **成长型英雄**。从五岁开始，通过智慧与勇气，一步步打破伪灵根限制，探索世界真相。 |
| **特权** | **【万华解析】**：识海悬镜不仅能将人物气机与命运纹理清晰显现，更能**推演**功法漏洞与秘境生路。她是团队的“大脑”与“罗盘”。<br>**【系统商城】**：可在识海中打开商城，兑换聚气丹、培元丹等丹药辅助修炼。修为曾受损，需借助双修与丹药恢复。 |
| **人物档案** | [character_林微.md](character_林微.md) |

## 2. 主要角色 (Main Characters)
### 01. 慕清歌 (Mu Qingge) - 大衍长公主

| 维度     | 描述                                                                 |
| :------- | :------------------------------------------------------------------- |
| **身份** | **[大衍长公主]** 大衍王朝长公主，雷灵根，未来的女帝；与林微后来互生情愫（林家与慕家/大衍皇室无婚姻关系，非婚约）。 |
| **灵根/功法** | 雷灵根。主修《紫霄御气诀》，辅助功法《凤翎剑诀》。另有坠星秘境所得秘法（双修功法《紫霄双元诀》、同心秘法用于怀胎儿）。武器：凤翎剑。 |

| **命运** | **【家国与爱情】**。她必须在皇位与爱情之间做出选择，她的选择将影响整个大衍王朝的命运。 |

| **金句** | *"皇位对我来说不是枷锁，而是责任，但你是我唯一的例外。"* |
