# 取名备选库 (Naming Bank)

本目录提供《灵衍万华录》写作时可用的**古风取名备选**，便于快速选用坊市、驿站、城池、村落、地形、小镇、商铺、组织、势力、宗门及 NPC 名称。

## 文件说明

| 文件 | 说明 |
| :--- | :--- |
| **取名备选库_001.md、_002.md、…** | 每次运行脚本生成**新编号**库，不覆盖旧文件，便于多次生成后人工筛选 |
| **generate_name_bank.py** | 生成脚本（Python 3）；可反复运行，每次产生新库；可修改脚本内 `COUNT_PER_CATEGORY`、用字库与词根 |

## 类别与数量（每库）

- **一、坊市**：1000 条，**3–5 字**（如东云坊、青云玉坊、青云碧落坊）
- **二、驿站**：1000 条（如长风驿、青霞栈）
- **三、城池**：1000 条（如天京、碧落关、青城郡）
- **四、村落**：1000 条（如叶家庄、竹里村）
- **五、地形**：1000 条（如青峰、龙溪、丹霞谷）
- **六、小镇**：1000 条（如长乐镇、云集）
- **七、商铺**：1000 条（如宝云斋、玉锦阁）
- **八、中小组织**：1000 条（如丹盟、器阁）
- **九、中小势力**：1000 条（如青云帮、紫霄堂）
- **十、中小宗门**：1000 条（如灵剑谷、玄阵观）
- **十一、NPC 备选名字**：3000 条（二字/三字随机组合）

## 使用方式

1. **生成新库**：在 `world/naming` 下运行 `python generate_name_bank.py`。首次生成 `取名备选库_001.md`，再次运行生成 `_002.md`，依此类推，**不会覆盖**已有库。
2. **人工筛选**：打开任意 `取名备选库_XXX.md`，在对应章节选用或改字组合；可从多份库中挑出可用的名，再合并到自己的笔记或设定里。
3. **调整数量与风格**：编辑 `generate_name_bank.py` 顶部 `COUNT_PER_CATEGORY`（默认 1000）、`COUNT_NPC`（默认 3000），以及用字库与词根，可改变每类条数和古风风格。

## 与设定的一致性

名称仅为**备选库**，不自动写入地点档案或 factions；选用后请按项目命名规则写入对应 `location_*.md` 或势力/角色设定，并保持与 `world_map.md`、`factions.md` 一致。
