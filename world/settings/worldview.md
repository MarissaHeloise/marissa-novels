# 灵衍世界世界观设定 (Worldview Settings) - 世界碎片版

**时间线**：世界历史分期与剧情时间锚点见 `world/settings/timeline.md`。**剧情当前时间**：大衍景帝三十五年春（坠星湖异象指引秘境开启、特别异象重宝出世；林微得见万象镜）。  
**力量体系**：凡人武者、灵修、浊修、阴修、功德修行、天道/规则体系及灵根与体质详见 `world/settings/power_systems.md`。

## 1. 核心概念

### 世界碎片 (World Fragment)

灵衍世界是万千世界的碎片之一。**凡界、灵界、浊界并非物理上彼此割裂的位面**，而是**同一方天地内的不同区域**，靠**结界与天地规则**划分与区隔，并存在**交接地带**。三界之别主要在于**灵力性质与浓度**：灵界以灵气为主、浓度高；浊界以浊气为主（浊气即被七情六欲或其他因果污染的灵气）、浓度高；凡界灵气与浊气皆有但浓度都不高，影响相对温和。灵界与浊界之间另有**灵浊争夺区域**，边界随势而移。跨界需穿越结界或满足规则（通行令、修为等），或经交接地带过渡。**阴界**在空间与规则上独立，有**专门通道**（黄泉路、鬼门关等）与三界相连。这些碎片散落在宇宙中，各自演化出不同的规则和生命。

### 灵气与浊气失衡 (Spirit and Turbid Qi Imbalance)

由于人和修者的七情六欲，导致灵气与浊气失衡，威胁着整个世界的稳定。**浊气**本质上是**被七情六欲或其他因果污染的灵气**；浊气会侵蚀生灵的生存空间，影响修者的心智。灵界、浊界、凡界的区分即依灵气与浊气的浓度与主导性而定。

## 2. 三界详细设定

### 【凡界 - Mortal Realm】

**能量特征**：**灵气与浊气皆有**，但**浓度都不高**，不像灵界、浊界那样一方独大、影响剧烈；天地法则又对高阶修为有所压制，灵气在此显得**惰性化**、难以被大量调动。
**天道桎梏**：凡界存在一层无形的“天花板”。
**修为上限**：**练气大圆满 (Qi Cultivation Perfected)**。

- 无论天赋多么妖孽，在凡界突破至筑基期的概率无限接近于零（除非有逆天机缘或古法）。

**核心规则**：

- **境界压制**：外来高阶修士（筑基及以上）进入凡界，修为会被**强制压制**至练气大圆满。强行运转高阶灵力会招致天罚（雷劫）。
- **术法削弱**：由于灵气惰性，所有法术消耗翻倍，威力只有灵界的十分之一。
- **武道盛行**：因为灵术难用，凡界发展出了极致的**武道**。顶尖宗师的肉身战力，近身可搏杀练气巅峰修士。

**修者死亡归宿**：死亡后灵魂进入**地府轮回**（三界共通，详见 `underworld_soul_system.md`），修为消散于天地间。灵魂依托**魂力**（一种特殊的**阴能**，三界皆存的特殊力量），在**特殊状态下**（如开天眼、观魂术）才可为生者所见；亦可被灭杀而**魂飞魄散**，无法轮回。

**所属势力**：

- **大衍王朝**（凡界共主，据中州及四方边州）
  - 都城：天京（京畿）；政区：中州·东莱·云梦·陇西·朔方五州
  - 人口：2亿+
  - 特点：科举制度、州郡县镇村五级政区、纯凡人统治
  - **林微所在**：东莱州青阳城（近坠星湖）
- **地脉九司**：凡界秩序与治理机构，与地府、凡人界治理挂钩（镇妖司、钦天监、守陵人、渡界使、问心局、灵植司、铸币司、藏书楼、影卫）；其中镇妖司**总部常驻苍梧封印关口**，凛渊域设前哨。
- **地府**：管辖三界灵魂与轮回（阴界势力），与地脉九司在妖魂、厉鬼处理上协作；灵魂体系与天眼设定见 `world/settings/underworld_soul_system.md`。
- **影卫**：凡界皇室直属刺客与密探。

**市井九流**（凡界地下网络；门类与称谓以 `world/factions/factions.md` 为准）：
- **惊门·天机阁**：贩卖情报，关注坠星秘境。**疲门·百草堂**：行医制药，为秘境试炼者提供丹药。
- **飘门·霓裳坊**：青楼戏班、易容刺杀。**爵门·纵横楼**：情报与暗杀，与血煞盟（血影门凡界分支）有脉络。
- **要门·万货帮**：丐帮网络；**云州商会**为其分支，控制云州经济命脉。
- **册门·玄机殿**、**风门·地师会**、**火门·丹鼎盟**、**彩门·幻戏宗**：古董风水炼丹幻术等。
- **天师府**：捉妖除魔民间组织，与青丘国狐妖事件有关。**万宝阁**：天工偃月在凡界商贸分支，举办万宝盛会。

**屏障通道**：

- **朔方州·天门关**：凡界—灵界通道，需通行令，通往灵界凛渊域**烛阴国**（人族政权）；驿道自天京北通天门关。
- **云梦州南疆·迷雾林**：自然屏障，常人进入必死，为凡灵交界险地。
- **东莱州东海·龙门**：海底通道，连接灵界沧溟域东海龙宫。

### 【灵界 - Spirit Realm】

**界域定位**：与凡界、浊界**同处一方天地**，以**结界与规则**及**灵气浓度**为界；**灵气为主、浓度高**，浊气相对被压制或稀释。进入灵界需突破筑基或经关口、交接地带。

**能量特征**：**灵气复苏与灾变共存**。

- 世界正处于“枯竭期”向“复苏期”的剧烈转折点。灵气不再温和，而是呈**潮汐状爆发**。
- **灵潮 (Spirit Tides)**：不定期爆发的灵气洪流，既能带来一夜突破的机缘，也能撑爆经脉，甚至引发山崩海啸、妖兽狂化。

**主要居民**：修仙者、妖族、灵族、龙族
**修为上限**：**元婴期 (Nascent Soul)**。

- 此乃灵界所能容纳的极限。元婴之上，需突破至化神期。

**核心规则**：

- **灾变修炼**：环境险恶，安逸闭关已成奢望。修士必须在动荡中争夺那些新诞生的“灵眼”。
- **适者生存**：灵潮会随机改变地形（如平原突起灵峰，湖泊化为雷池），只有适应力最强的不仅能活下来，还能变得更强。
- **凡人禁区**：由于灵气狂暴，普通凡人若无阵法庇护，会在灵潮中直接爆体或异化。

**修者死亡归宿**：**身死道消**（肉身与修为化为灵气回馈天地），**灵魂仍入地府轮回**（除非魂飞魄散）。

### 【浊界 - Turbid Realm】

**能量特征**：浊气浓郁，灵气被污染  
**主要居民**：魔修、半魔、堕落修士、异兽  
**修为体系**：魔息→魔基→魔丹→魔婴→魔魂→魔尊

**核心规则**：

- **腐蚀特性**：浊气会侵蚀普通人的理智，长期居住需特殊功法
- **修真者危险**：灵界修士若误入浊界，灵力会被浊气污染，轻则走火入魔，重则异化成魔
- **魔修限制**：魔修进入凡界或灵界，魔气同样会被压制

**修者死亡归宿**：**身死道消**（修为化为浊气回馈浊界天地），**灵魂仍入地府轮回**（除非魂飞魄散）。

**所属势力·浊世九渊**（浊界本土势力，详见 `world/factions/factions.md`）：
- **魔罗殿**：魔修核心，夜琉璃所在
- **血影门**：邪修，血煞盟为凡界云州分支
- **万毒窟**：毒修
- **幽冥教**：鬼修，拘役亡魂为资粮，与地府轮回秩序对立
- **渊墟殿**：崇拜深渊的堕落者
- **蚀骨渊**：骨修/尸道
- **噬魂殿**：魂修
- **焚心盟**：狂修
- **无相窟**：诡道/谍报

**主要区域**：
- **苍梧**（灵浊交界）：百草谷为灵界侧，毒瘴魔林为浊界侧；苏青黛所在百草谷属灵界。
- **暗影区域**（浊界腹地，无国家）：斗兽场、黑市、魔修据点；夜琉璃所在魔罗殿·地下斗兽场。

**浊界本质**：与凡界、灵界**同处一方天地**，以**结界与规则**及**浊气浓度**为界；浊气为主、浓度高（浊气即被七情六欲或其他因果污染的灵气）。与灵界之间存在**交接地带与争夺区域**（如苍梧毒瘴魔林侧、凛渊域极北裂缝一带），灵浊边界可随势而移。居住着被放逐的灵界修士与本土魔修，依赖浊气修炼。

**浊气净化方法**：
1. **补全世界本源**：收集世界碎片，补全世界本源，增强世界的自我净化能力
2. **建立灵气循环**：建立灵气与浊气的循环系统，使浊气能够被自然转化为纯净的灵气
3. **七情六欲的平衡**：通过教导修者平衡自己的七情六欲，减少浊气的产生
4. **特殊阵法**：创造特殊的阵法，能够吸收和转化浊气

**浊界结局**：
- **灵浊和解**：实现灵界与浊界的和解，建立新的平衡生态系统
- **浊气转化**：浊气被转化为纯净的灵气，形成良性循环
- **浊界重塑**：浊界不再是被放逐的地方，而是成为灵界的一部分
- **居民归宿**：浊界居民与灵界居民共同生活，形成多元化的世界

**阻止浊气侵蚀的方法**：
1. **世界壁垒的修复**：通过补全世界本源，修复世界壁垒，阻止浊气的外泄
2. **灵气守护者的设立**：建立专门的灵气守护者组织，负责观测与调衡灵气与浊气的平衡
3. **浊气转化装置**：在灵界与浊界的交界处设立浊气转化装置，将浊气转化为纯净的灵气
4. **修者的自我约束**：通过教育和引导，使修者学会控制自己的七情六欲，减少浊气的产生
5. **世界碎片的整合**：通过整合世界碎片，增强世界的稳定性，减少浊气的侵蚀

**时间**：**大衍景帝三十五年春**（凡界年号；与统一时间线对应见 `timeline.md` §5 当世）。

**大衍景帝三十五年春，坠星湖异象**

坠星湖异象**指引坠星秘境即将开启**；此番有**特别异象**，预示**有重宝出世**。灵衍系统**天命指引**指向秘境——其内可提升林微资质。

（林微识海得见万象镜、灵衍系统于引气入体后激活，见本段「万象镜与灵衍系统设定」，与坠星湖异象为不同事件。）

### 万象镜与灵衍系统设定

- **万象镜**：仅林微可见，表现为识海中悬浮的古朴铜镜，灵纹如烟如雾。
- **未激活前（现阶段）**：镜中名目仅两项——**人物属性**（可查看自身境况，如姓名、年龄、身份、修为、灵根、体质、功法等；**天命值**随剧情在面板中显现，**上限 9999**）、**天命指引**（与机缘命数相关，镜面浮过"天命未启、机缘未至"等，尚无具体任务）。灵衍系统尚未激活，无语音、无完整任务与机缘感应。
- **天命值达到 100**：灵衍系统激活（如女声提示"灵衍系统激活中，已激活"等），届时可开启**天命任务、资质提升、功法传承、机缘感应**等完整功能；与 world/settings/system_mechanics 中「灵衍系统核心功能」一致。
- **镜中人**：隐藏在万象镜/灵衍系统内的天道意识碎片，前期保持冰冷的工具状态，后期逐渐觉醒。
- **最终形态**：万象镜（灵衍镜）重新回归为天道化身，给与林微回到现实世界的机会。

## 5. 坠星秘境 (Starfall Secret Realm)

**性质**：三界交汇的异空间试炼场  
**开启周期**：每十年开启一次，持续三个月  
**入口位置**：青阳城外坠星湖（凡界与灵界的天然裂隙）

**秘境特性**：

- **独立空间**：秘境内部是一个独立的小世界，面积相当于十座城池大小
- **地形多变**：每次开启时地形地貌都会改变（森林、沙漠、雪山、湿地等随机生成）
- **能量混合**：秘境内灵气、浊气、凡气三者混合，呈现不稳定状态
- **实力压制**：所有进入者修为被强制压制至**练气大圆满**
  - 灵界金丹/筑基修士→降至练气大圆满
  - 魔修→降至同等魔息境界
  - 凡界宗师→保持肉身优势，且因习惯了低灵环境，反而如鱼得水

**压制机制详解**：

- 修真者失去大部分神通，只能使用基础术法（火球、水盾等）
- 魔修无法完全魔化，理智值强制锁定不下降
- 所有人的肉身强度被拉平到相近水平
- 这确保了**真正的公平竞争**——拼的是战斗技巧、智慧、意志，而非修为差距

**秘境规则**：

1. 进入条件：10-15岁之间，拥有灵根或特殊体质；**皇室或钦定名额**可获特许（如长公主代天子观星、历练），不受年龄硬限
2. 试炼内容：夺取「**潜龙令**」（秘境中随机刷新的令牌，总数100枚）
3. 危险因素：秘境内有野生妖兽、机关陷阱、以及其他参赛者
4. 时间限制：三个月后秘境关闭，未获得令牌者自动淘汰

**潜龙令与皇家的渊源**：

- **由来**：大衍立国后，皇室与坠星秘境试炼、地脉九司（含钦天监）达成旧例——秘境为「潜龙选拔之地」，**潜龙令**象征「得天道与皇运所钟」的潜质，皇室每届派员「**代天子观星**」，主持或背书盛会开幕，并记录夺令者名录。
- **皇家权责**：开幕由钦差或长公主「代天子」宣旨，宣示潜龙令与国运、选才的关联；夺令者可有资格入京觐见、获皇家赏赐或纳入朝廷/地脉九司选才名册，具体恩典由当朝裁定。
- **象征**：潜龙令既为试炼目标，也被视为「得皇室认可之才」的凭证；长公主亲临坠星湖时，常以「观星选才」「历练」之名，与潜龙令试炼绑定，强化皇权在凡界修真与人才选拔中的存在感。

**历史背景**：

- 坠星秘境是上古修真者留下的试炼场
- 每十年开启一次，已持续数千年
- 凡界、灵界、浊界三方势力共同承认其选拔价值
- 获得令牌者可获得各界宗门争相招揽；**凡界一侧**与**大衍皇室**渊源尤深，夺令者多愿留名以便日后入京或受赏

## 6. 修为体系与寿命

**完整设定**：凡人武者、灵修、浊修、阴修、功德修行、天道/规则体系及**灵根与体质**详见 `world/settings/power_systems.md`。以下为摘要。

### 修为等级（摘要）

- **凡界**：练气期（1-10层）→ 练气大圆满；**凡人武者**另有外劲→内劲→化境→宗师，顶尖宗师近身可搏杀练气巅峰。
- **灵界**：筑基期（1-3层）→ 金丹期（1-3层）→ 元婴期（1-3层）→ 化神期（1-3层）。
- **浊界**：魔息期 → 魔基期 → 魔丹期 → 魔婴期 → 魔魂期 → 魔尊期。
- **阴界/地府**：阴神不修灵气，修阴德与阴司阶品（城隍约筑基～金丹，勾魂使约金丹～元婴等）；见 `underworld_soul_system.md`、`power_systems.md` §4。

### 灵根与体质（摘要）

- **灵根**：五行灵根、变异灵根、伪灵根、无灵根、**混沌灵根**（最稀有，林微拥有）。
- **体质**：混沌体、**九转金丹**（林微）、先天阴体、阴阳眼、媚骨天成/玉人体（合欢宗）、魔种体质（夜琉璃）、妖修血脉（青丘、龙宫等）。详见 `power_systems.md` §7–§8。

### 寿命限制（灵修）

- 练气期：150-200年；筑基期：300-500年；金丹期：800-1000年；元婴期：1500-1800年；化神期：约2000年。

**注**：最高修为为化神，化神修士寿命约两千年；成神需唤醒上古神邸或补全世界本源，神已沉睡，再造极难。天道/规则体系见 `power_systems.md` §6。

## 7. 世界碎片与本源

### 世界碎片

灵衍世界是万千世界的碎片之一，由上古时期的世界破碎后形成。这些碎片散落在宇宙中，各自演化出不同的规则和生命。

### 世界本源

世界本源是维持世界运转的核心能量，由所有生命的信念和天地灵气构成。当世界本源受损时，世界会逐渐走向毁灭。

### 补全世界本源的方法

1. **收集世界碎片**：寻找散落在宇宙中的其他世界碎片，将其整合到灵衍世界中
2. **唤醒上古神邸**：上古神邸是世界碎片的守护者，唤醒它们可以获得修补世界本源的力量
3. **平衡灵气与浊气**：通过建立灵气循环系统，使浊气能够被自然转化为纯净的灵气
4. **减少七情六欲**：通过教导修者平衡自己的七情六欲，减少浊气的产生

## 7.5 婚姻与生育（社会风俗）

**婚姻与伴侣**：灵衍世界中，**女女、男女、男男**均可结为婚姻或伴侣关系，称谓包括夫妻、妻妻、夫夫、情侣、道侣等，世俗与修真界皆认可。

**生育规则**：

- **夫妻（男女）**：可自然怀孕生子。
- **妻妻（女女）**：无子宫一方不能自然怀孕；可通过某些**天材地宝或秘法**实现孕育后代。
- **夫夫（男男）**：无子宫，不能自然怀孕；亦有**天材地宝**可用两人精血或一人精血直接孕育后代，但**极难获得**，且需**大量灵气滋养**。该规则与性别无关，对所有人适用（即凡能获得此类机缘者，皆可借精血与灵气孕育）。

## 10. 灵气守护者与浊气掌控者

### 灵气守护者

守护天地灵气平衡的古老势力，负责观测与调衡灵气与浊气的平衡，防止灵气枯竭。

### 浊气掌控者

控制浊气力量的神秘组织，避免浊气泛滥，尝试与灵界和解；与灵气守护者共同维系灵浊平衡。

### 世界碎片管理者

维护世界碎片稳定的存在，确保世界碎片不会再次破碎，同时寻找其他世界碎片的下落。
