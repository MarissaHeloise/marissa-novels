## 一、修真界（正道）常见修士类型

### 1. 主战系

| 类型 | 特点 | 灵衍世界对应 |
| :--- | :--- | :--- |
| **剑修** | 以剑为道，杀伐第一，速度快、攻击高。 | **天枢剑宗**（莫离）；赤烬域 |
| **体修** | 炼体成圣，肉身强横，近战无敌。 | **金刚寺**；凛渊域苦寒炼体 |
| **刀修** | 刚猛霸道，大开大合，爆发力极强。 | 散修、蛮荒战部、边军 |
| **枪修** | 锐不可当，长攻突进，战场王者。 | 边军、凛渊域前哨、蛮荒战部 |

| **火修** | 狂暴攻伐，焚山煮海。 | 赤烬域、丹鼎盟、炼器坊 |
| **风修** | 速度第一，飘逸灵动。 | 九霄域、云中鹤国、散修 |

## 六、各修士类型·有利体质与不利体质

以下为写作人设时可直接采用的**有利体质**（利于该类型修行或战斗）与**不利体质**（易被克制或进境受阻）。体质名录与说明见 `灵根与体质全分类.md` 第二部分。同一类型可搭配多种有利体质之一或组合使用；不利体质可选其一增加剧情张力。

### 6.1 正道·主战系

| 修士类型 | 有利体质（可选） | 不利体质（可选） |
| :--- | :--- | :--- |
| **剑修** | 先天剑骨、剑心通明体、百脉俱通体、罡煞体、根骨清奇、混沌道体、万古圣体 | 残脉体、破法体、凡胎体 |
| **体修** | 万古圣体、不灭武体、战魂体、金刚不坏体、先天道体 | 残脉体、易劫体 |
