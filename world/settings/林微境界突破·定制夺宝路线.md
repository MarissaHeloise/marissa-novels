# 林微 · 境界突破定制夺宝路线

**用途**：在 `境界突破天材地宝参考.md` 的通用表基础上，为 **林微**（伪灵根、偃术/炼器专精、万象镜宿主、天工偃月宗与灵蚀域线）串联一条 **可写进正文** 的夺宝顺序与叙事抓手。  
**原则**：材料名称与功能 **不脱离** 参考文档；地点与势力名 **贴合本书**（苍梧、天工偃月宗、灵蚀域、浊灵交界等），争夺方式突出 **偃偶、阵盘、万华解析、极限反杀**，弱化“纯境界碾压”。

---

## 一、路线总览

| 大境界 | 定制重心 | 丹药主轴（参考表内名称） |
| :--- | :--- | :--- |
| **天道筑基** | 伪灵根需 **提纯度、拉平衡**；决裂后需 **压心魔、固道心**；资源多在苍梧卫星秘境/浊灵边缘 | 天道筑基丹、清心丹 |
| **金丹** | **金性灵材** 与偃兵丹海共鸣；宗门贡献换方、外派挣资源 | 凝金丹、固金丹、聚气丹（高阶） |
| **元婴** | 神魂负担加重，万象与天命的 **心智代价** 显性化；争夺升级为宗门任务+秘境排位 | 化婴丹、定婴丹、心魔丹 |
| **化神** | 神格与“天道/灵衍规则”对接，与卷末 **天道守护者** 长线伏笔衔接 | 化神丹、稳神丹 |

## 二、天道筑基段（林微专属顺序与写法）

> 参考表中 6 种天材地宝 **建议全部齐集**（伪灵根+天道筑基，宁多勿少）；下列顺序按 **剧情张力** 与 **能力成长** 排列，可视篇幅合并章节约夺。**正文草案**：第43章可 **同章** 取得 **晓光露** 与 **玄冰乳**（余材按卷再夺）。

### 1. 沙魂晶

- **功能**（同参考表）：滋养丹海、稳固根基，中和筑基时灵气紊乱——伪灵根最怕“气乱”，放第一站合适。  
- **本书落地地点**：**苍梧灵界侧 · 沙陨秘涡**（流沙与残阵叠压的古战场边缘，可比照“永宁州沙陨秘境”）。  
- **获得方式**：  
  - 万象镜标出 **涡心窗口期**（限时，引争夺）；  
  - 以 **低阶偃偶** 探沙、本体扼守阵脚，勾引 **千年沙妖** 与另一伙散修互耗，**坐收渔利或补刀反杀**；  
  - 可安排 **百草谷流言外泄** 仅作背景（她本人**不联络**谷内）。

### 2. 晓光露

- **功能**：涤经脉杂质、提灵气纯度——对 **五行杂乱** 伪灵根为 **核心辅材**。  
- **本书落地地点**：**苍梧北缘 · 雾隐秘境寒潭**（月夜才凝露，天亮即散）。  
- **获得方式**：  
  - **万华解析** 算准月相与潭底寒流走向；  
  - **冰纹灵龟** 由偃偶与困阵遛开，本体 **限时采集**；  
  - 遭遇 **偷袭** → 用 **烟幕符/暗弩** 反制，突出“打不过就跑、跑不了就换命算计”。

### 3. 玄冰乳

- **功能**：寒灵凝基、抬灵力上限；与后项 **赤焰果** 成 **寒热钉**，防道基偏枯。  
- **本书落地地点**：**岩髓秘境冰脉支岔**（苍梧与某浊脉辐射带的 **冷楔地带**）。  
- **获得方式**：  
  - 冰脉阵纹与鬼谷式 **秘境夺宝** 节奏：她 **缺蓝缺硬修为**，靠 **破阵节点推演 + 一次性阵盘**；  
  - **冰鳞兽** 可设计为与 **小队修士** 混战时 **同归于尽边缘**，她取乳撤离。

### 4. 赤焰果

- **功能**：火灵破壁、对冲玄冰乳阴寒，拉平筑基阴阳。  
- **本书落地地点**：**炎火山脉外环**（不入核心禁区，避免早期战力崩坏）。  
- **获得方式**：  
  - **炎纹兽** 强杀或引至熔隙借地形；  
  - **火炕+抢掠**：多名练气圆满 / 筑基初期劫修环伺 → **环境杀+偃偶诱敌** 的 **极限反杀**（与细纲第43–44章气质一致）。

### 5. 玉龙髓

- **功能**：调和灵力、压心魔——适合 **决裂与弑杀劫后** 的心境描写。  
- **本书落地地点**：**寒渊冰原外围 · 冰鳞兽旧巢**（与玄冰乳线可分两次进入，避免重复感）。  
- **获得方式**：  
  - 宗门低级外派 / 悬赏 与 **独行夺髓** 二选一；  
  - 强调 **斥候、妖兽、第三方** 三线压力，她靠 **情报与苟** 取胜。

### 6. 青岚玉

- **功能**：固道心、抗心魔侵扰——叙事上可作 **筑基前“是否仍走绝情绝义”** 的物化锚点。  
- **本书落地地点**：**竹海灵脉 · 青竹秘境**（苍梧东南，瘴浅竹深）。  
- **获得方式**：  
  - **翠纹灵蛇** + **隐匿阵纹**：最适合 showcase **万华解析**；  
  - 可安排 **天工偃月宗外门初试** 前夕夺玉，直接 **衔接入宗** 动机。

### 筑基丹线（丹药，非天材地宝但必写）

| 名称 | 功能 | 地点/来源 | 林微定制获得方式 |
| :--- | :--- | :--- | :--- |
| **天道筑基丹** | 提成功率、调和诸材属性冲突 | 无固定产地 | **拍行竞价穷途** → 转 **丹师工换物**（细纲44章钩）或 **天工偃月宗贡献配额**（入宗后第一次大出血） |
| **清心丹** | 筑基当刻镇心神 | 坊市通用品 | 反复备药；可在某次厮杀中 **丹瓶碎裂/剩最后一粒** 加压 |

## 六、设计备注（写作用）

1. **伪灵根**：同类资源里凡是 **纯度、调和、固壳、省耗** 向的，在表中 **优先夺、多夺一档**（叙事上多一次失败或折损也可）。  
2. **偃术**：**矿脉、阵纹、机关遗迹、水下/沙下** 类副本给她主场；**正面boss斩首** 交给队友或计杀。  
3. **万象镜**：只解决 **信息差与生路**，不代替 **道德代价与灵石账单**；高阶后可出现 **“机缘=劫数”** 的反转。  
4. **与慕清歌零联系期**：筑基金丹段 **不得** 用“托人问谷”“传音符”取药；若需情绪压迫，用 **心魔丹、青岚玉、玉龙髓** 在体内自证，不用跨线对话。  
5. **灵蚀域**：金丹中后期起，表中部分产地可平行替换为 **灵蚀域变异产区**（浊染灵植、灵傀残渣矿），功能不变，争夺升级为 **盟会+散修+灵傀潮** 三线。
