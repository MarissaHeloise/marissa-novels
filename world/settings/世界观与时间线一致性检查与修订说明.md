# 世界观、地图、势力、时间线一致性检查与修订说明

**版本**：1.0  
**最后更新**：2026-02-28  
**说明**：对世界观（worldview）、世界地图（world_map）、势力（factions）、灵界七域（spirit_realms_seven_domains）、时间线（timeline）进行统一性与合理性检查后的修订记录与结论。

---

## 1. 检查范围与目标

- **范围**：`worldview.md`、`world_map.md`、`timeline.md`、`spirit_realms_seven_domains.md` 及与屏障通道、地理归属、势力称谓、时间跨度相关的交叉引用。
- **目标**：消除矛盾表述，统一关键概念，使设定自洽、自成一界，便于后续写作与扩展。

---

## 2. 发现的问题与修订结论

### 2.1 天门关位置与天京表述（地理统一）

| 问题 | 原表述 | 修订 |
| :--- | :--- | :--- |
| 天门关所在地 | worldview 写「天京"天门关"」，易误解为天门关在天京 | 明确：**天门关在朔方州·烛阴郡**，凡界—灵界通道；天京仅为驿道南端，北通天门关 |
| 京兆郡 | 「天门关在此郡北缘」 | 改为「驿道北通朔方州天门关，天门镇为北道枢纽」 |
| 天京条 | 「天门关在此连通灵界」 | 改为「驿道北通朔方州天门关（凡界—灵界通道，通往灵界烛阴国）」 |

**涉及文件**：`worldview.md`（屏障通道）、`world_map.md`（京兆郡、中州·天京、重要节点表）。

### 2.2 坠星湖归属（地理统一）

### 2.3 浊界本质与三界关系（设定统一）

| 问题 | 原表述 | 修订 |
| :--- | :--- | :--- |
| 浊界本质 | 「灵界的一部分，由腐化的灵气形成」/「独立位面、物理隔离」 | 与「三界物理隔离」易冲突；用户要求三界不物理隔离 |
| 修订后 | — | **现状**：凡、灵、浊**同处一方天地**，以**结界与规则**及**灵气/浊气浓度**区隔；浊界为浊气为主、浓度高的区域，与灵界有**交接地带与争夺区域**（苍梧、极北裂缝等）；居民为被放逐灵界修士与本土魔修，依赖浊气修炼。阴界独立，有专门通道与三界相连。灵魂为**魂力（阴能）**，三界通用，特殊状态下可见。 |

**涉及文件**：`worldview.md`（浊界本质）。

## 4. 建议后续维护

- 新增「通道」「关口」时，同时在 worldview 屏障通道、world_map 重要节点、对应域/州小节三处统一名称与归属。
- 新增地理节点（如湖泊、秘境）时，先定州/域归属，再在总览与分节中一致书写。
- 势力称谓（道修/法修/剑修等）以 `power_systems.md` 与 `factions.md` 为准，worldview 与七域表保持同词。
