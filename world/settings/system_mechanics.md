# 灵衍世界系统机制 (System Mechanics)

**载体**：**万象镜**（林微识海中的悬镜，仅林微可见）。  
**系统**：由万象镜搭载的**灵衍系统**，以“天命感应”“古意灵纹”等形式呈现，不显化为数据流，与宪法主线（天命、觉醒与世界重塑）紧密绑定。

---

## 0. 万象镜与灵衍系统 (Wanxiang Mirror & Lingyan System)

### 万象镜

- **形态**：识海中悬浮的古镜虚影，灵纹如烟如雾，可显化为天命面板、命盘等界面。
- **本质**：与天道意识碎片相连，是补全世界本源、重塑世界秩序的一环；其存在与三界屏障、世界碎片的稳定相关。
- **镜中人 / 镜中灵**：万象镜器灵，天道意识碎片之化身。前期呈冰冷工具态，仅传递任务与指引；随天命值提升与剧情推进逐渐觉醒，最终可回归为天道化身，予林微“回归现实”或“留世守护”之选。

### 灵衍系统核心功能（服务宪法主线）

| 功能 | 说明 | 与主线关系 |
| :--- | :--- | :--- |
| **天命任务** | 发布指引任务，引导林微获得机缘、接触关键人物与地域 | 推动凡尘觉醒→感情与权力→世界重塑 |
| **资质与功法指引** | 指引灵根觉醒与提升，发现适合的功法（如《灵衍·筑基卷》） | 突破凡人界限，为跨凡入灵铺路 |
| **机缘感应** | 感知灵脉、宝物与道韵波动；与特殊地点“灵韵”共鸣 | 引导探索古迹、宗门、世界碎片线索 |
| **天命值** | 完成天命任务、改变关键人物命运、参与世界重塑而增长 | 达阈值可解放天道碎片意识，获得重塑/创世相关权限 |
| **世界碎片与本源感应** | 对世界碎片方位、上古神邸线索的模糊感应 | 为收集碎片、唤醒神邸、补全本源服务 |

## 2. 属性体系 (Attributes)

### 五维属性 (The Five Dimensions)

1. **根骨 (Root Bone)**：影响气血上限、防御、炼体效率。*来源*：先天、洗髓丹、炼体功法。
2. **悟性 (Understanding)**：影响修炼速度、领悟功法与阵法/炼丹成功率。*来源*：读书、观想、顿悟。
3. **福源 (Luck/Fate)**：影响暴击、掉宝与奇遇触发。*来源*：极难提升，多靠玉佩、祭祀等。
4. **神识 (Spirit/Mind)**：影响法术命中、感知范围、操控法宝数量。*来源*：境界、炼神功法。
5. **魅力 (Charisma)**：影响他人好感初值、交涉成功率。*来源*：外貌、称号、衣饰。

### 特殊数值

- **灵压 (Spiritual Pressure)**：凡界法则压制。调动外界惰性灵气时阻力极大，强行施法易伤经脉。
- **理智值**：接触浊界/魔气会削减，过低易陷入【疯狂】或魔化。
- **同步率**：主角与原身/世界的融合程度。过低易有排斥，过高则彻底融入此世。

## 3. 命盘系统 (Fate Compass)

*(万象镜显化的古意灵纹命盘)*

- **视觉**：灵光幻化而成的罗盘虚影，悬浮于视野一隅。
- **功能**：
  - **星位**：对应三界与灵界七域（或世界碎片）等关键节点；某星亮起表示该域/该线已解锁或推进，与主线探索、世界碎片收集呼应。
  - **模拟推演**：消耗大量灵韵/灵气，对近期可能发生的危机做一次模糊【预知模拟】，用于抉择与避险。

---

## 4. 与宪法主线对应关系

- **初期（凡尘觉醒）**：万象镜觉醒、新手大礼包与天命任务引导林微从无灵根废材走向机缘；《灵衍·筑基卷》突破凡界界限。
- **中期（感情与权力）**：命盘星位、机缘感应与天命任务交织于宫廷与灵界挑战。
- **终局（世界重塑）**：世界碎片与本源感应、天命值满解放天道意识，收集碎片、唤醒上古神邸、平衡灵浊，补全世界本源；镜中人回归天道，林微获得去留之选。
