# 万象镜机制 (Wanxiang Mirror Mechanics)

**唯一金手指**：**万象镜**（林微识海中的悬镜，仅林微可见）。  
**原则**：本作**不存在任何“系统”**。万象镜只以**镜面字迹、残影、天命感应**等方式给出信息与指引，不出现女声播报、签到礼包、任务列表、现代 UI 等设定。

---

## 0. 万象镜概述

- **形态**：识海中悬浮的古镜虚影，灵纹如烟如雾；必要时镜面可“浮字”或“映影”。
- **本质**：与天道意识碎片相连，是“天命/世界重塑”主线的媒介之一，但前期表现克制、冷淡、少言。
- **呈现方式（写法约束）**：
  - 只写：镜面一闪、字迹浮现、凉意/灼意牵引、残影一晃、心口发紧的“天命感”。
  - 不写：系统弹窗、任务面板、机械女声、奖励发放、签到/礼包/倒计时。

---

## 1. 镜中名目（功能）与用法

### 1.1 人物属性

- **用途**：给林微提供“自知之明”，确认当前状态（如姓名、年龄、身份、灵根、修为、体质、所学等）。
- **写法**：不必做成表格 UI，可用“镜面浮出一行行古字”的方式点到即止。

### 1.2 天命指引

- **用途**：在关键节点给出“去往何处/避开何事/取舍何人”的方向性提示。
- **写法**：更像“指路”而非“派任务”。可是短句、古意箴言、或一个强烈的方向牵引。

### 1.3 万华解析（可选名目）

- **用途**：用于“识人/辨物/破局”的更深层能力：看出体质异禀、器物关窍、阵法破绽等。
- **约束**：不做万能鉴定，不做随叫随到；只在“关乎生死或大势”的时刻显露其锋芒。

---

## 2. 天命值（叙事数值）

- **定位**：天命值是作者用来把握“改命力度”的叙事标尺，可在镜面字迹中**偶尔**隐约呈现，但不必频繁报数。
- **增长逻辑**：夺回机缘、扭转关键人物命运、在大势里做出正确选择、推动世界重塑等，都会让天命“更偏向她”。
- **写法建议**：优先用意象替代数字（如“镜纹更明”“裂痕稍愈”“字迹更稳”）。

---

## 3. 镜中人（天道意识碎片）

- **定位**：并非“系统助手”，而是万象镜深处的意识碎片。
- **表现**：前期几乎不说话，只留“残影/一瞥/一句模糊提示”；中后期才逐渐有情绪与抉择。
