# 灵衍世界·灵根与体质全分类

**版本**：2.0  
**最后更新**：2026-02-28  
**说明**：本文档为灵根与体质的**全分类设定**，与 `power_systems.md` §7–§8、`worldview.md` 一致。**灵根**不再扩充，保持现有名录；**体质**按界域与用途分类，并增加不利体质、更多可选项。各**修士类型**对应的有利/不利体质见 `修士类型全分类.md` **第六节**（§6）。  
**三界**：凡、灵、浊同处一方天地，以结界与灵气/浊气浓度区隔；阴界独立。灵修主灵气，浊修主浊气，阴修/功德修主魂力（阴能）与阴德。

**文档结构**：  
- **第一部分** 灵根：一 修真界（五行/先天/异灵根）、二 浊界、三 阴界/魂修·功德修。  
- **第二部分** 体质：一 修真界（顶级道体/主战/丹器符阵/稀有/不利）、二 浊界、三 阴界、四 妖修、五 佛修。  
- **第三部分** 与剧情及设定对应。  
- 各修士类型的**有利/不利体质**见 `修士类型全分类.md` **§6**。

## 第一部分：灵根（不变动名录，仅统一说明）

### 一、修真界（正道）灵根

#### 1.1 五行与复合灵根

| 类型 | 说明 | 常见适配 |
| :--- | :--- | :--- |
| **金/木/水/火/土单灵根** | 锋锐、生机、柔韧、暴烈、厚重；单灵根较优 | 剑修、丹修、法修、阵修、体修等见修士类型文档 |
| **双灵根** | 两种属性，修炼速度中等 | 需选主修属性 |
| **三灵根** | 杂而不精，入门易、进阶难 | 散修常见 |
| **四灵根/五灵根（伪灵根）** | 五行混杂、亲和极弱，几乎难入大道 | 凡界“废材”设定；林微初始可为此类 |
| **混沌灵根** | 可吸收与融合各种属性灵气；灵修侧最稀有 | **林微**（经灵衍镜觉醒后） |

#### 1.2 先天顶级灵根（五行极致）

先天金灵根（无垢金髓）、先天木灵根（万古长生木）、先天水灵根（先天真水）、先天火灵根（先天净火）、先天土灵根（先天息土）。适配剑修、丹修、体修、阵修等对应属性。

#### 1.3 稀有异灵根

风灵根、雷灵根、冰灵根、**毒灵根（灵界侧）**（丹修毒脉、抗毒；与浊界万毒窟毒修相对）、空间灵根、时间灵根（传说）、光明灵根。毒灵根在正道为变异五行、偏医毒；在浊界为浊气毒功，见下。

### 二、浊界灵根

| 类型 | 说明 | 适配 |
| :--- | :--- | :--- |
| **浊灵根** | 吸收浊气修炼，最常见 | 魔罗殿等 |
| **血灵根** | 吞血进阶 | 血影门 |
| **暗灵根** | 阴影、隐匿、刺杀、诅咒 | 无相窟 |
| **毒灵根（浊界侧）** | 万毒不侵，毒功大成 | 万毒窟 |
| **炎魔灵根** | 魔火，焚魂蚀骨 | 焚心盟、魔罗殿 |
| **天魔灵根** | 最顶级，可引天魔之力 | 魔罗殿顶尖 |
| **幽冥灵根** | 连通幽冥，魂魔 | 幽冥教、噬魂殿 |
| **吞噬灵根** | 可吞灵气、修为、法宝 | 噬魂殿、渊墟殿 |
| **破灭灵根** | 专破道法、防御 | 与正道克制 |

---

### 三、阴界/魂修·功德修

魂修、功德修**不依赖灵根**或灵根不显亦可修行，重**体质与魂魄天赋**。佛修侧有菩提灵根、佛光灵根、功德灵根、罗汉灵根、净世灵根（见 `妖修与佛修设定.md`）。

| **药灵体** | 与灵药亲和，辨药、种药、引药性入体 | 丹修、医修、花修 |

#### 1.4 稀有特殊体质

| 体质 | 特性 | 备注 |
| :--- | :--- | :--- |

| **涅槃仙体** | 可死而复生（有条件、次数有限） | 极稀有 |

#### 1.5 正道·不利体质（写人设可用）

| 体质 | 特性 | 说明 |
| :--- | :--- | :--- |
| **残脉体** | 经脉残缺或淤塞，修炼极慢、易走火 | 与“废材”设定搭配 |
| **凡胎体** | 无任何特殊加成，易被同阶压制 | 大多数凡人修士实为隐性格 |
| **易劫体** | 天劫比常人更猛，渡劫失败率更高 | 可作剧情虐点 |
| **破法体** | 自身法术易被同阶破解，不适合主修术法 | 宜转体修/剑修 |
| **心魔体** | 心魔易生，定力要求极高 | 心修、佛修可克，否则不利 |
| **滞灵体** | 灵力运转迟钝，施术慢、恢复慢 | 不利法修、符修、丹修 |
| **短寿体** | 同阶寿元明显偏低 | 需机缘延寿或提前突破 |
