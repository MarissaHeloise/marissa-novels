# 万宝试艺 · 考核内容设计（叙事用）

**用途**：供第46–47章及后续幽机域山门线对照；与 `faction_天工偃月.md`、`faction_天工偃月宗_山门组织.md` 中「万宝阁承办、考工司回传」一致。  
**定位**：外埠一体式选拔，**器修、偃师、机关师** 同台；**散修俗称「天工大比」**，正式文书称 **万宝试艺**。

---

## 一、设计原则（为何要「巧」）

1. **不是堆料炫富**：若只比「谁砸的灵材贵」，散修永无机会，与宗门「纳外埠才」矛盾。故关卡必有 **同一起跑线** 或 **信息不完备**，逼考生显 **思路、手稳、识见**。  
2. **一题多解，但可复核**：每场必有 **验伪阵、计时漏、匠籍刻印**（见山门组织档），结果可回溯，防代打、防事后换件。  
3. **器与偃不可偏废**：天工偃月以 **炼器与偃甲** 为干，机关阵禁为辅；试艺须让 **三条路都能得分**，但 **头名必须露一手「统合」**，否则只像分项匠人，不像宗门要的人。  
4. **外埠考工感**：题目常带 **半旧残件、坊市常见疵病、战损样**，贴近散修经验，却用 **宗门范式** 验收——**考得巧，正在于「你当散修时练出的野路子，能不能被宗门语法读成」**。

---

## 二、届制与场次（可随域微调）

| 层级 | 名称（对内常称） | 功能 |
| :--- | :--- | :--- |
| **海选** | **匣中取巧** / **补阙** 等（分场抽签） | 快筛；淘汰大半，留 **正赛名额** |
| **正赛** | **分项夺点**（器、偃、机 交错） | 积分累加；末位淘汰或划线晋级 |
| **终战** | **一械三用**（或当届命题变体） | 决 **头名** 与 **内门履诺名帖** |

七域分阁 **题目同源、材料同批、时限同刻**，成绩 **回传幽机域考工司** 存档，必要时总阁 **调卷复核**（照雪仙子押印的意义在此：不是来站台，是 **背书可追溯**）。

---

## 三、海选：两道可并行采用的题型（巧妙处）

### 甲、「匣中取巧」（限材盲盒）

- **规则**：每人领 **同一编号的灵材匣**（封缄、同批），开匣前 **不知** 内里配比；开匣后 **限时**（如漏刻两炷），须产出一 **可交付的成品或小件**。  
- **巧**：考 **分配与取舍**——匣内必有 **鸡肋料、相冲料、勉强够一件主料**；炫富无用，**谁先把「能交卷」与「少暴殄」平衡住** 谁占优。  
- **器修**：易走向「炼成即胜」；**偃师**：可走「微型傀枢」；**机关师**：可走「单阵盘可验」。**三类皆可有高分路径**，评判按 **完成度、禁制自洽、验伪一次过** 加权。

### 乙、「补阙」（残件限时接续）

- **规则**：发 **半件**——或断纹法器、或缺枢偃甲、或残阵一角；**不得换料**，只允许 **补、焊、嵌、改纹**；限时内须 **通过验伪与启灵/试机**。  
- **巧**：考 **读题能力**——残件上往往留有 **宗门小范式**（纹路习惯、榫卯语言），野路子散修若只会硬焊，**验伪阵会记一笔「匠风不合」**；读懂范式者，材料平平也能 **「像宗门手笔」**。  
- **叙事钩子**：林微可走 **流沙渊偃经 + 破锋偃将** 的实战语感，与 **范式** 碰撞，有戏。

**海选晋级**：两式取一域当届主裁定，或 **两场取高分**。人数压至 **正赛圈**（具体比例正文可写「十取一」「三取一」等）。

---

## 四、正赛：分项夺点（积分制）

建议 **三场**（顺序可抽签轮换），每场 **百分制**，加权后总榜；**每场淘汰末位若干** 或 **总分划线**。

| 分项 | 暂定命题方向 | 考什么 |
| :--- | :--- | :--- |
| **器** | **淬锋/成胚** | 在 **指定火侯与锤序** 下成器；或 **辨材** 后选一途；重 **手稳、纹顺、无隐裂**。 |
| **偃** | **傀应** | 给 **简傀架** 或 **傀线题**，限时 **三令七动**（具体指令当场公布）；重 **丝序、响应、不缠线**。 |
| **机** | **阵盘/机括** | **小范围机关**——如 **连环锁灵、一触三应**；重 **可复现、可检修**，忌一次性花架子。 |

**巧妙整合**：  
- 正赛中间可插 **「同题异构」**——同一题目说明，器/偃/机 **三种合法交付物**，评审判 **完成度 + 风险分**（例如过激结构扣分）。  
- **对手戏**：不必场场单挑；可采用 **相邻工位互相抽检**（万宝阁公证），**挑错得分、被挑错扣分**，防闷头抄近路，增 **现场张力**。

---

## 五、终战：「一械三用」（头名之战）

**命题思想**：头名不能只偏科，须证明 **一件器物上，器、偃、机 至少各满足一条最低线**，且 **可实战或可依规程检修**。

**示例结构（可当届变形）**：

1. **交付物**：一件 **主体**（如短兵、盾、飞梭类载体其一），须含：  
   - **器**：材质处理达标、可承灵、无验伪警报；  
   - **偃**：至少 **一可动关节/一傀线接口**，能完成 **规定动作**；  
   - **机**：至少 **一机关**（锁灵、变阵、蓄力释放其一），**可重复触发** 不少于 N 次。  
2. **限时与场地**：终战常设 **试艺台** 与 **小型压灵环境**（模拟战损或灵压波动），逼考生 **留检修口**，不能只秀一次性爆发。  
3. **巧**：最忌做成「三件拼盘」；高分在于 **三者共用同一套应力与灵路**，**改一处而不全盘崩**——这正是天工偃月要的 **统合匠才**。

**变体备选**（防读者审美疲劳）：  
- **「逆拆」**：给一件完整宗门下品样器，限时 **拆解并复原**，考 **理解与手底干净**；  
- **「应劫」**：当场公布 **一条随机疵病**（如灵路微蚀），须 **现场修补** 并过验伪。

## 六、评判、防作弊与「万宝」之名

- **验伪阵**：材料来源、刻痕时序、是否中途换件——**关键场次必开**；散修最怕这关，叙事上可制造 **心理压力与反转**。  
- **计时漏**：统一时限，**迟交即废** 或 **档位降格**；同刻七域，防托关系延长时间。  
- **匠籍与试艺牌**：自报名起 **真名绑定**，成品 **烙试艺编号**，与日后 **履诺名帖** 合一——第45章执事台词已落地，此处不重复。  
- **「万宝」**：除 **法宝** 外，可偶设 **识宝** 小关（辨伪灵材、断代、估损），**不占大头**，但可 **破纯砸钱流**，增加 **巧妙感**。

---

## 七、名次与奖励（与既有设定对齐）

- **头名**：**内门履诺名帖** + 灵石/法宝（届别与域别可具体化）；万宝阁 **接引礼仪**，赴幽机域履诺。  
- **前列非头名**：可给 **外门推荐、灵石、材料包** 等，正文可选用。  
- **考工司回传**：分项成绩、验伪记录、争议裁定书——**日后山门小比、升阶** 可查档，利于长线伏笔。

---

## 八、与林微线的贴合点（写作提示）

- **筑基初期**：对手宜含 **筑基初中期** 混场；终战 **压灵** 可略抬难度，但不跨大境界碾压。  
- **强项**：傀线、偃将、实战修补 → **补阙**、**傀应**、终战 **统合** 易出彩。  
- **弱项**：五灵根、资源史 → **匣中取巧** 心理压力、**辨材** 若出现需 **提前埋线**。  
- **忌**：赛博弹窗感、纯数值口嗨；**关卡本身即故事**，分数少提，**动作与裁夺多写**。
