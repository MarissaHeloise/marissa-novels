## 一、妖修（妖族・万灵修行）

### 1. 妖修核心逻辑

- **本体**：妖兽、精怪、草木、灵物；非人而开灵智、纳灵气修行。
- **化形前**：炼筋骨、开灵智、吐纳日月精华与天地灵气。
- **化形后**：修人形、修妖丹、修神通、修法则；部分保留半化形或本相战斗。
- **力量来源**：血脉、天赋、本体属性、天地灵气（灵界妖修主灵气；若入浊气浓区则可能魔化，见邪妖）。
- **与灵衍世界**：荒域九部（青丘、东海龙宫、云中鹤国、碧落海族、荒火部落、苍梧灵族等）、幻花域、沧溟域、荒狩域为妖修聚集地；万兽山庄为人族御兽修，与妖修协作或驯服妖兽。

### 2. 妖修基础分类

#### 2.1 按本体分

| 类别 | 举例 | 灵衍世界对应 |
| :--- | :--- | :--- |
| **兽妖** | 虎、狼、龙、凤、麒麟、蛇、狐、熊、豹、猿等 | 青丘（狐）、蛮荒战部/荒火部落（火脉蛮族可含兽妖）、荒狩域妖兽 |
| **禽妖** | 鹰、雕、孔雀、鹤、鹏、鸾鸟等 | **云中鹤国**（鹤族）、九霄域浮岛 |
| **鳞妖** | 龙、蛟、龟、鱼、蜃等 | **东海龙宫**（龙族，敖漓）、**碧落海族**（鲛人/海妖）、沧溟域 |
| **虫妖** | 蝶、蜂、蝎、蛛、蜈蚣、天蚕等 | 万毒窟对立面有灵蛊；南疆、苍梧毒瘴侧可有虫妖 |
| **植妖** | 花、树、藤、竹、灵草、古木 | **苍梧灵族**（木灵/药修）、幻花域花海、百草谷外围 |
| **器妖** | 法宝、古物开灵 | 极稀有；天工偃月、上古遗迹可遇 |
| **异妖** | 特殊灵物、先天异种 | 陨星遗民、坠星海、上古遗族 |

#### 2.2 按属性分（与正道五行/异灵根对应）

金翅妖、青木妖、水鳞妖、火焰妖、土甲妖；风翼妖、雷纹妖、冰魄妖、暗影妖。可与灵根属性对应，便于写斗法相克。

### 3. 妖修天赋能力（天生就会）

| 能力 | 说明 |
| :--- | :--- |
| **本体神通** | 如狐妖魅惑、虎妖蛮力、鹏妖极速、龙族控水；因种族而异。 |
| **化形** | 化人、化兽、半化形；化形后仍可现本相搏杀。 |
| **血脉觉醒** | 越纯越强，上古血脉可压一切低等妖；青丘九尾、龙族真龙血脉等。 |
| **妖丹** | 妖修根本，碎则身死；相当于人族金丹，但来源与形态不同。 |
| **本命皮毛/爪牙/鳞甲** | 天生法宝，可炼为本命器。 |
| **灵嗅/灵视/灵听** | 感知远超人类，追踪、预警、破隐。 |

### 4. 妖修常见流派

| 流派 | 特点 | 灵衍世界对应 |
| :--- | :--- | :--- |
| **纯血妖修** | 靠血脉碾压，少修外法 | 东海龙宫、青丘嫡系、云中鹤国 |
| **蛮力妖修** | 不修法术，只炼肉身 | 蛮荒战部、荒火部落、荒狩域兽妖 |
| **幻术妖修** | 狐妖、蝶妖、蜃妖为主 | **青丘**（白涂涂）、幻花域 |
| **音波妖修** | 妖笛、妖琴、啸声攻敌 | 云中鹤国（与妙音坊有往来）、荒狩域 |
| **控植妖修** | 树妖、花妖、藤妖 | 苍梧灵族、幻花域 |
| **御水/控火妖修** | 鳞妖、火鸦、炎兽 | 东海龙宫、碧落海族、荒火部落 |
| **蛊妖双修** | 妖身养蛊，极阴毒 | 南疆、浊界万毒窟有邪道对应 |

## 二、佛修（佛门・正道・渡世・镇压邪魔）

### 1. 佛修核心逻辑

- **修**：禅定、慈悲、功德、戒律、心性、佛光。
- **目标**：明心见性、证罗汉果、菩萨果、佛陀果（与灵衍世界化神之上、成神难对应，可作佛门表述）。
- **天生克制**：魔、邪、阴魂、尸、蛊、诅咒；与镇妖司、问心局在“斩妖除魔”上可协同。
- **灵衍世界对应**：**金刚寺**为佛修代表势力（玄门九宗，体修+佛修）；凛渊域守边、伏魔、金刚不坏。

### 2. 佛修基础流派

#### 2.1 正统佛修

| 流派 | 特点 |
| :--- | :--- |
| **禅修** | 打坐入定，悟佛法、开慧眼。 |
| **律修** | 严守戒律，功德无量，佛法自生。 |
| **慧修** | 修智慧、辩经、破妄、破魔。 |
| **净土修** | 念诵佛号，往生净土，灵魂不灭；与地府轮回可衔接。 |

#### 2.2 战斗佛修（最常见，金刚寺主力）

| 流派 | 特点 | 灵衍世界对应 |
| :--- | :--- | :--- |
| **金刚佛修** | 肉身如金刚，近战无敌 | **金刚寺**核心（体修+佛修） |
| **伏魔佛修** | 专门斩妖除魔、镇压邪祟 | 金刚寺、镇妖司协作、凛渊域守边 |
| **药师佛修** | 疗伤、解毒、渡化、救苦 | 金刚寺分支、百草谷协作 |
| **护法佛修** | 护宗门、护佛法、护众生 | 金刚寺、各域险地驻守 |
| **真言佛修** | 一字真言，言出法随 | 金刚寺、散修佛门 |

### 3. 佛修能力体系

| 能力 | 说明 |
| :--- | :--- |
| **佛光** | 净化、破邪、疗伤、渡魂；克魔气、阴魂、诅咒。 |
| **佛印** | 金刚印、降魔印、菩提印；镇压、封印、攻伐。 |
| **经文** | 诵念即生威力，可攻可守可渡化。 |
| **舍利** | 佛修精华，死后遗留，可助人修行。 |
| **慧眼** | 看破虚妄、幻术、心魔、本体。 |
| **金身** | 刀枪不入、万法不侵；与金刚不坏体对应。 |

### 4. 佛修灵根/天赋

| 灵根/天赋 | 说明 |
| :--- | :--- |
| **菩提灵根** | 佛修第一灵根 |
| **佛光灵根** | 天生带净化之力 |
| **功德灵根** | 行善即可涨修为 |
| **罗汉灵根** | 战斗天赋极强 |
| **净世灵根** | 可净化一切污浊；克浊气、毒、蛊 |

### 5. 佛修特殊体质

**完整名录与说明**见 `灵根与体质全分类.md` 第二部分·五（佛修体质）。此处仅作速查：

先天佛胎、金刚不坏体、菩提圣体、功德金身、渡厄佛体、**禅心体**（禅修）、**戒律体**（律修）。各佛修流派对应的**有利/不利体质**见 `修士类型全分类.md` **第六节·6.8** 正道·佛修。

## 三、浊界 / 魔修 对应的「伪佛、邪妖」

### 1. 伪佛（魔佛・邪佛・最危险）

| 类型 | 特点 | 灵衍世界对应 |
| :--- | :--- | :--- |
| **血佛修** | 以血诵经、杀人为功德 | 可与血影门、渊墟殿交叉 |
| **魔佛修** | 外表佛，内心魔，渡化即控制 | 堕落佛门、渊墟殿 |
| **欢喜佛（邪）** | 邪淫入道，采补修行；与合欢宗正道双修对立 | 浊界邪修，合欢宗对立面 |
| **地狱佛** | 修地狱业火，炼魂为食 | 噬魂殿、幽冥教 |
| **咒佛** | 以佛经下诅咒，一字一灾 | 无相窟、诅咒修 |

### 2. 邪妖 / 魔妖 / 血妖

| 类型 | 特点 | 灵衍世界对应 |
| :--- | :--- | :--- |
| **噬灵妖** | 吞吃魂魄、灵根 | 噬魂殿相关、渊墟殿 |
| **血妖** | 吸精血、炼血功 | 血影门、血煞盟 |
| **尸妖** | 死尸成妖，不生不灭 | 幽冥教、蚀骨渊 |
| **魔化妖** | 被魔气污染，狂暴嗜血 | 浊界侵蚀后的妖兽、灵浊交界 |
| **灾厄妖** | 带来瘟疫、干旱、死亡 | 万毒窟、灾厄类妖兽 |
