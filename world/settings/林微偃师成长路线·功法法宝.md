# 林微 · 偃师成长路线（筑基—化神）

**文档用途**：专表 **偃术递进**（功法、重器、神通、**操傀规模**），用语贴合 **苍梧 / 天工偃月宗 / 灵蚀域**。天材地宝夺序与丹药主轴 **一律见** `林微境界突破·定制夺宝路线.md`，**本文不重复叙事节点**。  
**总根功法**：仍 **`《万象归宗诀》`**；下文诸篇为 **偃道别传**，作残简、功勋换经或秘境碑刻即可。  
**旧器承接**：**破锋偃将** 在 **第43章** 起进入 **轻量化、灵巧化** 迭代（战中觉笨重、战后改枢），再逐步 **熔入** 后续主战傀；**栖云偃庐** 后期可升格为 **铸枢工坊 / 偃城阵眼**（化神段再定终形）。  
**正文锚点**：**《偃钥启枢篇》《悬丝照窍诀》** 于 **遗府/灵涡线** 入手；**神识傀丝** 与 **悬丝断机** 等神通成长同步。

---

## 气质与战法（阴柔 · 狠戾 · 霸气）

**写作口径**：她的偃道 **不靠吼、不靠擂鼓造势**——**声轻则丝重**，**笑淡则局死**。  

- **阴柔**：灵丝 **如绣、如溺、如雾中缚**；多从 **腕、踝、颈侧、气窍** 下手；战局里常见 **慢半拍的温柔**（诱敌深入、养骄、蚕食之）。  
- **狠戾**：一旦落子，**不兴无谓的虐**，但 **该废则废**——**轨断灵脉、枢碎道基、傀啖残识**（可对罪修、灵傀潮使用）；俘敌可 **丝穿肩髃为窍**，**走得脱算她失手**。  
- **霸气**：不在音量，在 **一帷之内我即律**——**万丝同叩、千傀齐喏**；她站定处，**退路只有她许的那条**；化神后 **罗帷一落**，**山河机关皆须俯听**。  

> 与同阶女修对比：可写 **“她不似剑修劈面，却似把整座战场绣进一张慢慢收紧的罗幕。”**

## 灵傀数量 · 递进（写作用硬指标）

> 数量为 **常态可稳操** 上限；**短爆** 可逾一档 **须付神魂/血气代价**。量多 **不** 等于每只都有结丹战力，**卒、弩、织、瞑** 分级见各境。

| 境界 | 主战 / 高智 | 灵卒 & 功能卒（合计可同操） | 气质注 |
| :--- | :--- | :--- | :--- |
| **天道筑基** | **1** 敛青级主战 | **外加 2～4**（探沙、牵丝、占位、投毒囊的 **纸轻傀 / 木卒**） | **蚕步**：小钱叠大局 |
| **金丹** | **2** 敛锋级并列 或 **1 重甲 + 1 快臂** | **外加 6～12** **暝卒**（裁识未开全，只执行 **缀、绊、诱、射** 四令） | **魇步**：影多声少 |
| **元婴** | **1** 暝识主傀 + **半甲** | **外加 12～36** **暝络卒**（三路裁识可 **分队**）；必要时 **收卒扩主** | **缀命**：四面缀杀，中央落子 |
| **化神** | **1** 万象裁枢尊 **为律标** | **阵内 300～1200+** **律傀**（共一条 **偃律**；城外另备 **轮换胚傀**） | **罗帷天倾**：一帷之威 |

## 递进铁律（神通合理性）

| 境界 | 偃道在做什么 | 故意 **不做** 什么 |
| :--- | :--- | :--- |
| **筑基** | **丝 surgical** + **卒叠势**；主傀承伤，卒 **送死不可惜** | 卒无灵智、**不可** 当第二个“你” |
| **金丹** | **双线主枢** + **多卒魇步**；丝网 **疏而不漏** | 卒 **不** 共享她完整境界；**爆操** 逾量 **眩络** |
| **元婴** | 婴气 **洗主核 + 卒络**；卒 **分队自洽小动作** | 卒 **不夺舍、不代悟道**；**狠招** 后 **必有空窗** |
| **化神** | **万枢同律**；域内 **罗帷加权重**；单点 **天工一着** | **非** 言出法随；**域剖** 则 **帷破神伤** |

**化神翘楚**：**天工偃月宗镇派级 + 三界偃道三甲一席**；价值在 **一人即半座偃城**（灵蚀域抗潮），非一人一拳废了天下。

## 一、天道筑基 — 初炼傀

**别传功法**

| 篇目 | 定位 |
| :--- | :--- |
| **《偃钥启枢篇》** | 筑 **木、石二等** 与 **纸轻卒** 图谱；**卒造价贱、枢浅**，专堆 **数量与炮灰**。 |
| **《悬丝照窍诀》** | 丝 **走窍**；一掌可 **牵三卒薄操**（再多则 **指僵**）；主战仍吃 **大半心神**。 |

**重器**

| 名称 | 要旨 |
| :--- | :--- |
| **敛青·战傀** | 蛇骨脊、玉屑神、寒露洗；**阵心坦**。 |
| **牵机寒髓符** | 续半息丝；烙 **卒位**——**一次性**。 |
| **髓铁定枢砧** | **批量钉卒枢**；无攻伐。 |

**神通**

- **悬丝断机**：灵丝 **卡在敌术 “起式未完”** 的节点，**迟半拍**；仅对 **已看破** 的术式有效；**钝冲** 无效；连施 **逾九息** 则 **指尖僵痛半日**。  
- **替死契**：**主傀** 迎面承一击，本体 **借斥横移**；主傀距 **≤七丈**；**一主一契 / 日**；卒 **不替死**（卒毁只损丝、可再撒）。  
- **堕星占角**：预落 **傀钉** 占 **阵一角**，与阵旗 **凑残缺星位**，迷踪 **≤十息**；**卒可代钉**；钉卒毁则 **角塌一半**。  
- **丝溺**：灵丝 **绕颈踝如柔水**，**三息内收寸**；专破 **骄进**；**无窍炼体** 效减；**留手** 则擒，**收狠** 则 **绞息 / 碎踝**。

## 二、金丹 — 控傀熟

**别传功法**

| 篇目 | 定位 |
| :--- | :--- |
| **《金枢衍形录》** | **双主战枢**；**暝卒** 图谱（**无灵，只四令**）；丹息 **灌核** 可 **短时全卒齐射一式**（罢后 **卒滞三息**）。 |
| **《同心印髓章》** | **多瞳轮切**（非同时全开）：**一息扫一角**；乱切则 **神眩**。 |

**重器**

| 名称 | 要旨 |
| :--- | :--- |
| **敛锋玄甲傀** ×2 或 **重甲 + 快臂** | 双峰 **扼喉** 之势。 |
| **纳枢游匣**（可叠 **子匣**） | **操战 2 + 暝卒至十二** 轮换；丹格 **一息** 取。 |
| **千丝绊龙绦** | **分岔数十缕** 控 **前锋卒**；雷罡仍 **灼丝**。 |

**神通**

- **双环演武**：两主战 **双轨**；**卒缀轨外** 掷弩 **搔扰**。  
- **折锋合拍**：**主双** 合一点；卒 **不配沾Centre**（沾则 **碎**）。  
- **千缕魇步**：卒 **移位如潮痕**，**真身只在丝源**——**识破丝源者** 可破；持续 **逾十五息** 则 **丹息见底**。  
- **借瞳一瞬**：闭左目 **借一卒瞳** **半息**，够 **一针**；逾半息 **神池刺痛**；**日不过七借**。  
- **裁脉缀**（偏狠）：丝尖 **入敌气窍外缘** **不深插**——**修为不掉阶**，但 **三月内行气如吞针**（**阴柔官司**）。

## 三、元婴 — 融傀生

**别传功法**

| 篇目 | 定位 |
| :--- | :--- |
| **《婴络偃甲篇》** | **婴络分叉**：**一络主、多络卒**；卒 **识浅** 可 **自队**。 |
| **《种暝裁识书》** | 每卒 **裁三识**；三队 **进 / 退 / 诱** **可同时对操**——再分则 **婴颤**。 |

**重器**

| 名称 | 要旨 |
| :--- | :--- |
| **暝识青岚傀** | **主帅**；可 **暂摄三十六卒魂火**（摄久 **主傀暝浊**）。 |
| **婴络同痕玺** | 锁相位；**狠用**：可 **令一卒代承甲噬**（卒 **废**）。 |
| **副魄藏机珠** | **卒智冷备**；**连珠** 可扩至 **两队各十二** 轮换（炼制与时间成本极高）。 |

**神通**

- **半甲叠锋**：一臂或 **背脊偃化**，得 **婴级三招**；撤甲 **该处空窗十二息**，**遇刺必伤**。  
- **万蚕缀命**：**卒如蚕**，**四面缀咬同一敌**；敌悍则 **先啮翼、辅、器**；遇 **泼天罡** 则 **卒废一片**。  
- **婴络并流**：人傀 **灵池并联四至七息**；罢后 **双虚**，**半柱香不可再战**；**月不过三并**。  
- **暝行三替**：高智主傀 **进退诱** 自替；**主帅核不替**；乱 **心印频** 则 **卒僵**。  

**空间法则**：挪卒、换位；**可一串挪三卒** ——逾则 **婴络打结一盏茶**。

## 四、化神 — 化傀神（罗帷）

**别传功法**

| 篇目 | 定位 |
| :--- | :--- |
| **《万枢裁天录》** | **万枢一律**：**三百～千余** 律傀 **同踏一字**（滞 / 重 / 错序 择主律）；**灵屑暂傀** 不计入 **律** 只计 **填缝**。 |
| **《神络同冥契》** | 神格烙脊；**狠境**：**烙深** 则 **律傀嗜主残识** ——她若败，**帷内无全魂**。 |

**重器**

| 名称 | 要旨 |
| :--- | :--- |
| **万象裁枢尊** | **律标**；**罗帷心**。 |
| **天炉铸枢鼎** | **战时涌泉修卒**；**鼎偏城倾**。 |
| **太络束神绦** | 束相散；**解束仪轨半日**。 |

**神通（域 / 次 / 祭）**

- **罗帷天倾**：**万枢阵** 内 **偃相** 如 **低垂天罗** ——敌 **先觉呼吸困难、后觉轨涩**（写法上 **阴柔至极**）；**每场一举**；**阵剖** **相噬神台一寸**。  
- **万枢归流**：**千傀齐喏一步** ——一步之内 **地脉亦须错半拍**；叠 **罗帷** **须燃神髓** **一局唯一**。  
- **天工一着**：对 **单机关 / 单灵脉节点** **一次** 开阖撬拨；成则 **三月不复灵**；败则 **损一裁**（随机一项偃艺跌落须重修）。  
- **谒陵**（可选用 **极狠**）：**律反** 一瞬，**指定一片律傀** **自碎贯敌** ——**卒灭三成**；仅用 **于灵傀潮 / 灭宗级公敌**，**用后须闭关铸卒**.

**栖云偃庐**：证 **偃城** 后，**天炉 + 万枢** 为食；**非战** 不露罗帷。
