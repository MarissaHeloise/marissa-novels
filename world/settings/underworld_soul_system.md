# 地府与灵魂体系设定 (Underworld & Soul System)

**创建时间**：2026-03-01  
**状态**：Active  
**说明**：灵魂与轮回为独立于灵气/浊气的另一套规则与能量体系；地府为三界共通的阴面秩序。**阴修/阴神体系与功德修行**的完整表述见 `world/settings/power_systems.md` §4–§5。

---

## 1. 灵魂与魂力（阴能）(Soul & Soul-Force / Yin-Energy)

### 1.1 灵魂的本质

- **灵魂**是生灵的“真灵”所寄，与肉身、修为相互独立。
- **魂力（阴魂之力/阴能）**：维系灵魂存在的能量，是一种**三界通用的特殊力量**，与天地间的**灵气、浊气**不同源，属**阴面法则**。凡、灵、浊三界众生皆具魂力；魂力可在阴界（地府所在）得到补充，在阳间会随时间流逝而消散，故亡魂久留阳间会变弱或化为厉鬼。
- **三界众生**（凡人、修士、妖族、魔修等）皆具灵魂；死后**肉身与修为**按各自界域规则消散（灵气/浊气回馈天地），**灵魂**则受阴面法则牵引，经阴界**专门通道**（黄泉路、鬼门关等）入地府轮回。

### 1.2 灵魂的两种终结

| 结局 | 条件 | 结果 |
| :--- | :--- | :--- |
| **入轮回** | 灵魂未被灭杀，受地府规则或勾魂使接引 | 魂入阴界，经审判、洗业后可转世（凡/灵/浊皆有可能，依业力与机缘） |
| **魂飞魄散** | 灵魂被专门术法、天罚、噬魂类功法或禁器击散 | 真灵与魂力彻底消散，**无法轮回**，等同于“存在被抹去” |

**魂飞魄散的具体条件**（满足其一即可能造成，依强度与抗性而定）：

| 类型 | 条件/手段 | 说明 |
| :--- | :--- | :--- |
| **灭魂术法** | 针对魂体的杀伐术（如噬魂殿「抽魂炼魄」、部分禁咒「散魂诀」） | 施术者需能“触达”魂体（开天眼或阴眼、或术法自带锁魂）；魂力弱于术法强度则直接崩散。 |
| **天罚/天劫** | 业力过重者在雷劫或天谴中被劈中魂体 | 天道降罚时若判定“罪不容诛”，会连魂带魄一并抹杀，不入地府。 |
| **噬魂类功法** | 吞噬、炼化他人神魂以壮己身（噬魂殿、部分鬼修） | 被吞尽的灵魂不再存在，无法轮回；若只被“撕下一部分”可能重伤但未必立刻魂飞魄散。 |
| **禁器/诅咒** | 上古禁器、血脉诅咒、大能临死反扑的“魂咒” | 需明确针对“灭魂”或“咒杀真灵”；普通致命伤只毁肉身，不必然灭魂。 |
| **自愿散魂** | 极少数禁术允许自我燃尽魂力、永不入轮回 | 多为赎罪或绝誓，地府视为“放弃轮回权”，不予接引。 |

- **判定**：魂力归零、真灵无法被地府感应，即视为魂飞魄散；地府「孽镜」可照出某魂是否仍存于天地，用于追查失踪亡魂。

## 2. 灵魂的可见性：天眼与阴眼

### 2.1 常人不可见魂

- 灵魂/鬼魂存在于与阳间**重叠但不同层**的“魂相”中，依托**魂力（阴能）**而存；肉眼凡胎**无法直接看见**，唯有在**特殊状态下**（如开天眼、观魂术、阴神显化）才可为生者所见。
- 未开天眼者只能通过**阴风、托梦、附身、异象**等间接感知亡魂存在；部分符箓、阵法可短暂显形或沟通。

### 2.2 开天眼 / 阴眼 / 观魂术

- **天眼**：部分修士或特殊体质者可通过修炼或机缘开启“天眼”，直接看见魂体、阴气与地府鬼差。天眼与神识不同：神识感知灵气与生机，天眼感知魂力（阴能）与阴面存在。
- **阴眼**：地府阴神、鬼修或长期接触阴魂者所具的“见魂”之能，本质与天眼同源，多用于阴司公务或鬼道修行。
- **观魂术**：符修、鬼修、天师府等传承的术法，可临时或持久地让人“见魂”，代价或条件各异（如耗寿、需符箓、需在特定时辰等）。

**天眼修炼门槛**：

| 条件 | 说明 |
| :--- | :--- |
| **修为下限** | 至少**筑基期**且神识稳固，否则无法承载“阴面法则”的映照，轻则头痛欲裂，重则识海受损。练气期仅能借外物（符箓、阵法）短暂见魂。 |
| **功法/传承** | 需修习**观魂类心法**或**天眼诀**（天师府、部分符修/鬼修宗门、地府外派城隍传承中有残卷）；无传承者只能靠机缘（如阴地顿悟、阴神托梦赐予）。 |
| **体质/机缘** | **先天阴体**、**阴阳眼**血脉、或曾死里逃生、魂魄离体再归者，更容易开天眼，门槛可放宽至练气后期；普通人需筑基后按部就班修习。 |
| **代价与限制** | 非阴神开天眼会持续消耗**魂力或神识**，久开会疲惫；部分旁门开眼法以**折寿**或**阴气侵体**为代价，地府不提倡。 |
| **与阴眼的区别** | 阴眼多为地府编制内授予或鬼道功法自带，不占“阳间修士”修为；天眼是阳间人主动修出的“见魂”之能，二者在功能上等价。 |

### 2.3 与世界观的关系

- **灵气/浊气**：主阳间修行与肉身、修为；**魂力（阴能）/阴面法则**：主灵魂、轮回与地府，**魂力为三界通用的特殊力量**。两套系统并行，修士“身死”后灵气归天地，灵魂走阴面，互不替代。
- 幽冥教、噬魂殿等**拘役或吞噬灵魂**，与地府“接引—审判—轮回”的秩序直接冲突，故与地府为对立关系。

## 3. 地府的位置与性质

### 3.1 阴界（冥土）

- **地府**所在为**阴界**（又称冥土、幽都），与凡界、灵界、浊界**并列**，不在三界任一界的“地表”或“空间”内，而是**规则层**上的另一面。
- 进入阴界的方式：
  - **死者**：魂灵受阴面法则牵引，经**黄泉路**或由**勾魂使**接引，自然“沉”入阴界。
  - **生者**：极少数秘法、禁术或地府特许（如还阳公务、大能闯阴）可经**鬼门、阴阳井、奈河渡**等节点进入，属例外。

### 3.2 与三界的连通节点

- **凡界**：各地城隍庙、乱葬岗、古战场等阴气汇聚处，常有**鬼门**或**黄泉岔口**，勾魂使由此出入；大衍境内无固定“地府衙门”，地府不直接统治凡界，仅司轮回与阴律。
- **灵界/浊界**：修士、妖族、魔修死后，魂灵同样受阴面法则牵引，可由对应区域的阴气节点或勾魂使跨界接引至阴界。灵界、浊界并无“地府分部”，地府统一在阴界办公，勾魂使可跨界出差。

## 4. 地府架构：地点与机构

### 4.1 阴界主要地点

| 地点 | 职能/说明 |
| :--- | :--- |
| **酆都** | 阴界主城，十殿与各司衙门所在，相当于“地府都城”。 |
| **鬼门关** | 魂灵进入阴界的正式关口，有阴兵把守，生人勿近。 |
| **黄泉路** | 连接阳间与阴界的魂灵通道，勾魂使带魂经此路入阴。 |
| **奈何桥** | 跨忘川的必经之桥，孟婆在此施汤，饮后忘尽前尘方可入轮回。 |
| **忘川** | 阴界大河，洗去业障与执念；坠入忘川者将沉沦历劫，难以立刻轮回。 |
| **望乡台** | 魂灵可最后回望阳间故土与亲人的地方。 |
| **十殿** | 见下表 §4.3 十殿职责。 |
| **轮回井/转生池** | 通过审判、饮过孟婆汤的魂灵由此投入轮回，转世至凡/灵/浊。 |

### 4.3 十殿职责（酆都十殿）

魂灵入阴界后，依生前业力与死因分派至不同殿受审、受刑或直入轮回；十殿顺序流转，并非每魂都走遍十殿。

| 殿序 | 殿名（俗称） | 职责 | 所辖/备注 |
| :--- | :--- | :--- | :--- |
| **一殿** | 秦广殿 | **初审与分流**：查生死簿、孽镜照前业，断寿终/横死/功过；无罪或功德高者可直接送转轮殿或还阳；有罪者发往二至九殿受刑。 | 孽镜台在此；若魂飞魄散者被追查，也由此殿核验“魂是否仍存”。 |
| **二殿** | 楚江殿 | **寒冰地狱**：惩偷盗、拐卖、伤人肢体、谋财害命等。魂灵于寒冰狱中受冻刑，刑满再转下一殿。 | 主掌“伤生害命”类业报。 |
| **三殿** | 宋帝殿 | **黑绳地狱** | 惩忤逆尊长、挑拨离间、诬陷诽谤等；绳缚刀锯，刑满流转。 |
| **四殿** | 五官殿 | **血池/血脓地狱** | 惩滥杀、虐杀、见死不救等；与“血”相关的业障在此清算。 |
| **五殿** | 阎罗殿 | **望乡台与核心审判**：阎罗王亲审大案要案（如帝王将相、大修、跨界重犯）；可调阅各殿案卷，定刑期与来世去向。 | 慕清歌等身份特殊者多经此殿；望乡台在五殿外，魂灵过此可最后一望阳间。 |
| **六殿** | 卞城殿 | **枉死城**：收容横死、自杀、冤死而未得超度之魂；令其暂居枉死城，待因果了结或超度后再判。 | 与阳间“超度”“还愿”衔接；沈梦蝶类冤魂可先入此城。 |
| **七殿** | 泰山殿 | **热恼/蒸笼地狱** | 惩口业（妄语、恶口、两舌）、毁人清誉、邪淫等。 |
| **八殿** | 都市殿 | **大热恼地狱** | 惩不孝、不义、虐亲、欺师等；重者与九殿衔接。 |
| **九殿** | 平等殿 | **阿鼻/无间**：极恶之魂（屠城、叛道、毁轮回秩序等）在此受最长刑期，或永世不得轮回。 | 噬魂殿、幽冥教中罪大恶极者若魂入地府，多判入此殿。 |
| **十殿** | 转轮殿 | **轮回发放**：各殿刑满或直判之魂，汇总至转轮殿，依业力与因果分配**轮回井/转生池**的池口（凡/灵/浊、胎卵湿化等），饮孟婆汤后投入轮回。 | 与奈何桥、孟婆汤、转生池为一套流程；转轮王掌“最后一签”。 |

- **简化用法**：剧情若不需要细写十殿，可只提“经某殿审判”“入枉死城”“转轮殿投胎”；需要时再具体到殿名与刑罚。

### 4.4 职位与修为（阴司阶品）

地府阴神**不修灵气/浊气**，而是修**阴德与阴司阶品**，实力以“阶品”与“神职”挂钩，与阳间境界大致对应关系仅作参考（非严格换算）：

| 职位 | 职责 | 阶品/实力参考 |
| :--- | :--- | :--- |
| **阎君/阎罗** | 主掌一殿或数殿审判，地府最高决策层 | 相当于阳间化神级或更高，阴界规则加持 |
| **判官** | 审案、录籍、断善恶、定刑期与来世 | 相当于元婴～化神，文职为主 |
| **黑白无常（勾魂使）** | 赴阳间接引亡魂，押解至阴界 | 相当于金丹～元婴，可跨界行走 |
| **牛头马面** | 阴兵统领、押解重犯、镇守关口 | 相当于筑基～金丹 |
| **孟婆** | 守奈何桥，施孟婆汤，掌“忘尘”一环 | 修为深不可测，多视为“规则化身”或上古阴神 |
| **城隍** | 驻阳间某城或某域，协理本地亡魂与阴事，向地府汇报 | 相当于筑基～金丹，常为功德成神 |
| **阴兵/鬼差** | 巡逻、押送、守关 | 相当于练气～筑基 |

- **说明**：阴神“修为”不以灵气论，而以**阴德、神职、阴界权柄**为主；与阳间修士交手时，在阴界地府主场占优，在阳间则受规则压制。

## 5. 轮回规则（与三界的关系）

- **凡界、灵界、浊界**众生，死后灵魂**皆可**进入地府轮回（除非魂飞魄散）。
- **身死道消**：仅指**肉身与修为**的消散（灵气/浊气回馈天地），**不**影响灵魂入轮回。
- 来世投胎至**凡界、灵界还是浊界**，由地府依业力、因果与轮回井的“池口”规则决定，并非凡界死者只能投凡界。
- **厉鬼、怨灵**：未及时被接引、或执念过深拒绝过桥的魂灵，会滞留阳间或阴间边缘，需超度或镇压；地府与问心局、镇妖司在“妖魂归处”“厉鬼收押”上有协作。

## 6. 与现有设定的衔接

- **地脉九司**：守陵人、问心局、镇妖司与地府在“尸变亡魂”“妖魂”“厉鬼”处理上分工协作；地府不隶属地脉九司，而是**特殊势力**，与天道、法则相关。
- **幽冥教/噬魂殿**：拘役或吞噬灵魂，破坏轮回秩序，与地府为**对立**关系。
- **天师府/超度**：凡界天师府等势力可通过超度帮助亡魂化解执念、顺利进入地府轮回；开天眼或观魂术为相关人员的常见能力。
- **慕清歌结局**：传位后自杀，灵魂入地府轮回，符合“三界皆可轮回”的设定。
