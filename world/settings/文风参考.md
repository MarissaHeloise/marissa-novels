# 文风参考（修真唯美）

**说明**：写作正文与润色时，文风宜**偏修真唯美**；首要参考《入海》，references 目录下另两本作补充。本文档与 `.novelkit/memory/constitution.md` §8.1、作家档案「文风基调与参考」一致。

---

## 基调

- **修真唯美**：用词雅致、意境留白、情感含蓄，具古典仙侠审美；避免直白网文腔和一味爽点堆砌。
- **节奏**：可缓可留白，不每句都信息拉满；重要情节点用细节与氛围传达，少直白心理喊话。

---

## 首要参考：《入海》（月本渡）

- 仙侠修真、**唯美治愈**、古典仙侠审美；景致与情感描写细腻。
- 可借鉴：节奏缓急、**光影/水月/山海**等意象、通过神态与环境传达情绪的方式。
- 联想方向：修炼与斗法时的光影、气息、水月、山海；人物情绪用神态、动作、环境反衬多于直白独白；句子长短错落，短句制造停顿与余韵。

## 可操作要点

| 方面 | 建议 |
|------|------|
| 景致/环境 | 偏意境化、画面感；山水、灵气、洞府、秘境可带氤氲、潋滟、杳渺等雅词 |
| 修炼/斗法 | 不只写「打了/突破了」，可带光影、气息、剑意/术法的形与势，留一点余白 |
| 情感 | 多用神态、动作、环境反衬；少「她心想：……」式直白心理 |
| 句子 | 长短错落；适当短句作停顿与收束，长句铺陈时注意节奏 |
| 用词 | 雅致不粗粝；可多用凝、纳、铺展、揉进、氤氲、潋滟、杳渺等，避免过于口语或网文套话 |
