# 灵衍世界·法宝套路全览

**版本**：1.2  
**最后更新**：2026-02-28（补全计划第二批：浊修/妖修/武道/阴修各补50条，浊修56→106、妖修60→110、武道92→142、阴修52→102）  
**说明**：本文档为不同修行路线的修士提供完整法宝套路，设计目标为**每类修士的法宝各 500 种**（即剑修 500 种、法修 500 种、体修与佛修 500 种、符阵丹器 500 种、浊修 500 种、妖修 500 种、武道 500 种、阴修·功德修 500 种；八类各自独立计数）。含法宝等级、装备栏分类、适用境界及成长性。**命名层次**分普通（一听就平常）、中高（比较有来头）、顶级（非常厉害），写作时可一听名称即辨档次。当前各节条目数尚未铺满 500，可随写作逐步扩充。

## 法宝等级与装备栏总表

### 一、法宝等级（由低到高）

| 等级 | 对应境界（建议） | 说明 |
| :--- | :--- | :--- |
| **凡器** | 凡人/外劲～内劲 | 无灵性，凡铁、凡甲、凡饰；丹鼎盟主产，与灵器分野明确。 |
| **下品灵器** | 练气 | 初具灵性，可认主、需灵力催动。 |
| **中品灵器** | 筑基 | 灵性稳固，可简单御使、小幅成长空间。 |
| **上品灵器** | 金丹 | 灵性充沛，可御空、与神识契合。 |
| **下品宝器** | 金丹～元婴 | 宝光内蕴，同阶中顶尖，多可炼为本命。 |
| **中品宝器** | 元婴 | 宝器中的中坚，器灵雏形或强灵性。 |
| **上品宝器** | 元婴～化神 | 宝器巅峰，器灵可显化或与主沟通。 |
| **道器** | 化神 | 蕴含道则，一击可牵动天地规则，极稀有。 |
| **仙器/神器** | 化神及以上/传说 | 上古遗留或世界本源相关，当前世界屈指可数。 |

- **成长性**：部分法宝可随主人境界提升而淬炼进阶（如本命剑从灵器→宝器→道器），表中以「成长」标注。
