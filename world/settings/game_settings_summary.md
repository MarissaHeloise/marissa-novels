# 《灵衍万华录》改游戏设定总方案 (Game Adaptation Summary)

**版本**：2.0  
**状态**：方案/思路  
**前提**：以**林微**为绝对主角与唯一主控视角，游戏体验紧扣小说世界观与主线成长弧。

---

## 一、定位与可行性原则

### 1.1 游戏类型建议

- **主推形态**：**单主角剧情向 RPG**（林微固定主角，无自建角色/多主角选择）。
- **可选形态**：**叙事冒险 + 轻度养成/战斗**（偏视觉小说或线性关卡），体量更小、更易落地。
- **不推荐**：多主角、多初始背景、开放阵营选择——会稀释「林微一人改命、天命与世界重塑」的核心命题，且与原著强绑定设定冲突。

### 1.2 与小说的对齐原则

- **视角**：全程林微视角（含内心、万象镜意象、与镜中人的互动）；重要配角以「伙伴/对手/剧情 NPC」出现，不抢戏。
- **主线**：凡尘觉醒（凡界）→ 炼器崛起与慕清歌线（凡界）→ 秋蝉之死与逃离 → 筑基证道（灵界）→ 金丹/元婴 → 世界桎梏与化神 → 万华归一（收集碎片、补全本源、选择留下）。
- **金手指唯一性**：万象镜仅林微拥有，作为叙事与玩法的核心机制（天命指引、机缘提示、天命值意象），不做「多角色各自金手指」。

---

## 二、核心系统设计思路

### 2.1 万象镜机制（游戏化）

| 小说设定 | 游戏化思路 | 可行性要点 |
| :--- | :--- | :--- |
| 天命任务 | **主线/关键支线任务**：以「天命感应」形式发布，指引下一阶段目标（如「前往坠星湖」「寻找偃师传承」）。 | 用任务列表 + 简短古文/韵文描述，保持「天命感」而非现代 UI。 |
| 资质提升 | **灵根/修炼效率成长**：完成关键节点（秘境、传承、试炼）后触发「灵根蜕变」或「瓶颈突破」，数值上体现为修炼速度、突破概率提升。 | 与修为阶段绑定，避免无节制的数值膨胀。 |
| 功法传承 | **功法/技能树**：万象归元诀、偃师传承、空间法则等作为林微专属技能线解锁；万象镜可提示「某地有机缘」。 | 炼器+傀儡+空间法则三条线可做成技能树或装备/傀儡蓝图。 |
| 机缘感应 | **探索提示/高亮**：在场景中标记「天命所钟」的宝物、灵脉、隐藏剧情点；可做成罗盘/镜面涟漪等视觉反馈。 | 控制提示强度，避免完全剥夺探索感。 |
| 天命值 | **隐藏叙事数值**：完成主线、改变关键角色命运、做出符合「重塑世界」的选择可增加；可用「镜面渐明」「裂痕渐愈」等意象暗示进度，不必强调“激活某系统”。 | 不直接显示具体数字，避免数值化破坏氛围。 |
| 镜中人 | **旁白/剧情角色**：前期冷淡提示，中后期逐渐有台词与情绪，终局揭示天道意识身份；可参与关键抉择的「第二意见」。 | 台词少而准，避免“系统播报感”。 |

### 2.2 修为与战斗

- **境界阶段**：练气（1–10 层）→ 筑基 → 金丹 → 元婴 → 化神；与**三界地图解锁**挂钩（凡界练气、灵界筑基至元婴、终局化神）。
- **战斗定位**：林微以**炼器 + 傀儡 + 空间法则**为主战手段，非纯剑修/法修；可设计为「策略回合」或「即时 + 傀儡编队」：本体站位/技能 + 傀儡承担伤害与控场。
- **凡界特殊**：凡界灵气惰性、术法削弱，可突出**武道/身法 + 傀儡**的配合；进入灵界后法术与空间手段增强。

### 2.3 三界与地图节奏

| 界域 | 小说作用 | 游戏化思路 |
| :--- | :--- | :--- |
| **凡界** | 林微出身、觉醒、炼器崛起、慕清歌与秋蝉线 | 主城（青阳城、天京）、坠星湖/秘境、云州、天门关等；练气期主舞台，境界压制规则可做成「外来高阶被压到练气」的剧情/机制。 |
| **灵界** | 筑基后主舞台，七域、玄门九宗、荒域九部 | 分域解锁（如凛渊、沧溟、九霄等），灵潮可做成「随机事件/副本」或阶段剧情；元婴为灵界上限。 |
| **浊界** | 夜琉璃、苏青黛线，灵浊冲突，终战 | 中后期进入，浊气侵蚀可做成「环境 debuff/净化任务」；终局三界联军与浊界大战。 |
| **阴界/地府** | 灵魂轮回设定，慕清歌结局 | 可选：仅剧情提及或短篇「过阴」章节；若做关卡可做成线性剧情副本（黄泉路、某殿审判）。 |

## 三、剧情与角色在游戏中的位置

### 3.1 主线分段（对应卷）

1. **卷一·凡尘镜影**：觉醒万象镜、坠星秘境、偃师传承与练气成长、初遇慕清歌 → 游戏**第一章/新手期**。
2. **卷二·千机百炼**：炼器坊、傀儡、万宝盛会、慕清歌重逢与感情线、辅佐称帝、囚禁与逃亡、秋蝉牺牲 → **第二章/凡界终章**，情感与选择密度高。
3. **卷三·筑基证道**：百草谷、灵潮病与夜琉璃、筑基、叶红绡/敖漓（空间法则于金丹期领悟）→ **第三章/灵界开局**。
4. **卷四–六**：金丹、元婴、世界桎梏、化神、世界碎片、镜中人觉醒、补全本源、选择留下 → **后续章节/终局**。

### 3.2 伙伴与 NPC（不夺视角）

- **秋蝉**：林微侧固定伙伴，战斗/剧情双线；**秋蝉之死**为必发生关键节点，可做分支（如死前选择影响林微后续心态或天命值）。
- **慕清歌**：重要剧情与情感线；游戏内可为「关键抉择对象」（如是否深入合作、是否接受囚禁前的某条支线），影响慕清歌结局是否入轮回等，但不做可攻略多结局的「选谁」式设计，保持林微为唯一主角。
- **叶红绡、敖漓、白涂涂、苏慕晚、苏青黛、夜琉璃等**：按剧情阶段登场，可做**助战/剧情伙伴**或关卡内临时队友，关系通过对话与支线深化，不喧宾夺主。

### 3.3 势力与声望（简化）

- 不采用「多势力任选加入」；以**林微行动结果**自然影响势力态度（如镇妖司、天工偃月、百草谷、青丘等）。
- 声望可影响：商店价格、任务是否开放、部分剧情分支；不做「选边站」大阵营战，终局仍是三界联军共抗浊界。

## 四、特色玩法与表现建议

### 4.1 炼器与傀儡

- **炼器**：材料收集 + 图纸/配方解锁（含万象镜机缘提示）；炼制出的法器、傀儡影响战斗与剧情（如「玄机」系列傀儡）。
- **傀儡编队**：若做策略/回合，可设傀儡槽位，不同傀儡承担坦/控/输出；林微本体负责关键技能与空间法则。

### 4.2 天命与选择

- **关键抉择**：在重大剧情点（如是否全力辅佐慕清歌、秋蝉牺牲前的行动、是否接受某份传承）给出选项，影响关系线或天命值，进而影响结局走向（真结局需天命值满）。
- **避免**：与原著核心结局（林微留下、补全本源）相悖的「毁灭世界」「投靠浊界」等纯恶搞线；可做「坏结局」但标注为非正统。

### 4.3 表现与风格

- **万象镜 UI**：古朴文字、烟云雾气、镜面裂纹/修复意象，少用现代科技感词汇（与 constitution 禁忌词一致）。
- **三界差异**：凡界市井与武道感；灵界灵潮、宗门与妖族；浊界浊气侵蚀与魔修风格——用美术与音乐区分。

---

## 五、体量与实现优先级（思路）

- **最小可玩**：卷一主线 + 万象镜基础能力（天命指引、简单机缘提示）+ 练气期战斗与炼器入门 + 坠星秘境与慕清歌初遇。  
- **完整单机**：六卷主线压缩为 5–8 章大关卡，三界地图分阶段开放，天命值 + 镜中人 + 多结局（含真结局）。  
- **长线/番外**：伙伴支线、地府短篇、七域独立小副本等，在不改变「林微绝对主角」前提下扩展可玩内容。
