# 灵衍世界力量体系与修行设定 (Power Systems & Cultivation)

**创建时间**：2026-02-28  
**最后更新**：2026-02-28  
**说明**：本文档统一凡人武者、灵修、浊修、阴修、功德修行、天道/规则体系及灵根与体质设定，与 `worldview.md`、`underworld_soul_system.md`、各势力档案保持一致。

## 1. 凡人武者 (Mortal Martial Artists)

### 1.1 定位与能量

- **界域**：凡界；因灵气稀薄且惰性化，灵术难用，凡界发展出**极致武道**，以肉身与内劲为主。
- **能量来源**：不依赖天地灵气主动吐纳，而以**气血、筋骨、内劲**为根基；部分顶尖功法可微量引灵入体（仍受天道桎梏，无法突破练气上限）。
- **与灵修关系**：凡界修为天花板为**练气大圆满**；**顶尖武道宗师**的肉身战力，近身可搏杀练气巅峰修士；高阶修士入凡界会被压制至练气大圆满，故武道在凡界具有实际威慑力。

### 1.2 武道境界（凡俗划分）

| 境界 | 别称 | 特征 | 与练气修士对比 |
| :--- | :--- | :--- | :--- |
| **外劲** | 练皮骨 | 筋骨强健，力大身捷，可敌寻常壮汉与低阶妖兽 | 低于练气初期 |
| **内劲** | 练气入体 | 内劲贯通经脉，可隔空发力、轻身提纵；寿命略长于常人 | 约等于练气前期～中期 |
| **化境** | 化劲 | 内劲化形、收放由心，一方豪强水平 | 约等于练气后期 |
| **宗师** | 武道宗师 | 肉身与武意合一，近身可搏杀练气巅峰；凡界武道顶点 | 约等于练气大圆满战力（肉身侧） |

- **说明**：武道境界与练气**非一一对应**，修士有术法与神识，武者纯靠肉身与武技；在凡界灵气被压制的前提下，宗师与练气大圆满修士可形成制衡。
- **寿命**：外劲/内劲与凡人接近或略长；化境约百二十年；宗师可达一百五十年上下，仍远低于练气大圆满的 150～200 年（若修士在凡界不突破则同受凡界寿元限制）。

### 1.3 与剧情的关系

- 林微在凡界阶段可接触武道势力（如天师府、江湖门派）；秋蝉、慕清歌线中可出现宗师级武者。
- 坠星秘境内修为被压制至练气大圆满，**凡界宗师**因习惯低灵环境、肉身优势保留，在秘境中具有竞争力（见 `worldview.md` §5）。

## 2. 灵修 (Spirit Cultivators)

### 2.1 定位与能量

- **界域**：主舞台为**灵界**；凡界仅能修至**练气大圆满**，突破筑基需入灵界或逆天机缘。
- **能量来源**：**灵气**；吐纳天地灵气，淬体凝丹、化婴化神。
- **死亡归宿**：身死道消（修为化灵气回馈天地），**灵魂入地府轮回**（除非魂飞魄散）。

### 2.2 灵修境界与寿命

| 境界 | 层级 | 寿命（约） | 说明 |
| :--- | :--- | :--- | :--- |
| **练气期** | 1～10 层 → 练气大圆满 | 150～200 年 | 凡界上限；灵界为入门 |
| **筑基期** | 1～3 层 | 300～500 年 | 灵界常驻门槛，可御器飞行 |
| **金丹期** | 1～3 层 | 800～1000 年 | 金丹为“道基”凝缩，可结九转金丹等特殊境 |
| **元婴期** | 1～3 层 | 1500～1800 年 | 灵界当前容纳上限 |
| **化神期** | 1～3 层 | 约 2000 年 | 世界当前顶点；化神之上“神”已沉睡，成神极难 |

- **灵界上限**：元婴期为灵界所能容纳的极限；元婴之上需突破至化神，化神修士寿命约两千年，再往上“成神”需唤醒上古神邸或补全世界本源，已近乎传说（见 `worldview.md` §6、§7）。

### 2.3 突破与压制

- **凡界天道桎梏**：在凡界突破筑基的概率无限接近于零（除非逆天机缘或古法）；外来筑基及以上修士进入凡界会被**强制压制**至练气大圆满，强行运转高阶灵力会招致天罚（雷劫）。
- **坠星秘境压制**：所有进入者修为被强制压制至**练气大圆满**（灵界金丹/筑基→降至练气大圆满），拼的是战斗技巧、智慧与意志（`worldview.md` §5）。

### 2.4 与灵根、体质的关系

- 灵根决定**吸收灵气的属性与效率**；体质可提供**特殊能力、瓶颈突破或异象**（如九转金丹）。详见下文 §7、§8。

## 3. 浊修 (Turbid Cultivators)

### 3.1 定位与能量

- **界域**：**浊界**；灵界修士若误入浊界，灵力易被浊气污染，轻则走火入魔，重则异化。
- **能量来源**：**浊气**；以浊气淬体凝丹、化婴化魂，与灵修对应而相克。
- **死亡归宿**：身死则修为化浊气回馈浊界天地，**灵魂仍入地府轮回**（除非魂飞魄散）。

### 3.2 浊修境界（与灵修大致对应）

| 境界 | 别称 | 与灵修对应 | 说明 |
| :--- | :--- | :--- | :--- |
| **魔息期** | 纳浊入体 | 练气 | 引浊气入体，易侵蚀心智，需功法或意志压制 |
| **魔基期** | 浊基 | 筑基 | 浊基稳固，可御浊气而行 |
| **魔丹期** | 浊丹 | 金丹 | 浊丹凝成，战力与金丹修士相当 |
| **魔婴期** | 浊婴 | 元婴 | 夜琉璃线可至 |
| **魔魂期** | 浊魂 | 化神前阶 | 魂魄与浊气合一 |
| **魔尊期** | 魔尊 | 化神级 | 浊界顶点 |

- **心魔与侵蚀**：浊气会侵蚀理智，长期修炼需特殊功法或坚定心志；灵修若被浊气侵体，需净化或压制，否则易堕入魔道。
- **跨界压制**：魔修进入凡界或灵界，魔气同样会被压制，与灵修入凡界规则对称。

### 3.3 浊世九渊与修行流派

- 魔罗殿（魔修）、血影门（邪修/血道）、万毒窟（毒修）、幽冥教（鬼修/尸傀）、渊墟殿（堕落者）、蚀骨渊（骨修/尸道）、噬魂殿（魂修）、焚心盟（狂修/执念）、无相窟（诡道/谍报）；各派主修浊气，侧重点不同，详见 `factions.md` §3b。

## 4. 阴修与阴神 (Yin Cultivators & Underworld Divinity)

### 4.1 定位与能量

- **界域**：**阴界**（地府所在）及地府派驻阳间的**城隍、勾魂使**等。
- **能量来源**：不修灵气/浊气，而修**魂力（阴能）**、阴德与阴司阶品；魂力为三界通用的特殊力量。实力以“阶品”与“神职”挂钩，在阴界主场占优，在阳间受规则压制。

### 4.2 阴神阶品（与阳间境界参考对应）

| 职位/类型 | 阶品/实力参考 | 说明 |
| :--- | :--- | :--- |
| **阎君/阎罗** | 化神级或更高 | 地府最高决策层，阴界规则加持 |
| **判官** | 元婴～化神 | 审案、断善恶、定刑期与来世 |
| **勾魂使（黑白无常）** | 金丹～元婴 | 跨界接引亡魂 |
| **牛头马面** | 筑基～金丹 | 阴兵统领、押解重犯 |
| **孟婆** | 深不可测 | 多视为规则化身或上古阴神 |
| **城隍** | 筑基～金丹 | 驻阳间，常为**功德成神** |
| **阴兵/鬼差** | 练气～筑基 | 巡逻、押送、守关 |

- **与阳间修士交手**：在阴界地府主场占优；在阳间则受规则压制，多通过庙宇、托梦、阴差协办等形式显化。详见 `underworld_soul_system.md` §4。

### 4.3 鬼修（幽冥教等）与阴修之辨

- **阴神**：地府编制，修阴德与阶品，司轮回与阴律，**不**以吞噬或拘役亡魂为业。
- **鬼修**：浊界幽冥教等，以尸傀、拘役亡魂为资粮，与地府轮回秩序**对立**；其“修魂”为邪道，与地府阴神之“阴修”本质不同。

## 5. 功德修行 (Merit & Virtue Cultivation)

### 5.1 功德与业力

- **功德**：善行、护世、超度亡魂、维护秩序等所积累的“阴德”或“阳德”；在轮回与天劫中有用。
- **业力**：恶行、杀孽、破坏轮回（如噬魂、拘役亡魂）等所积累；地府审判时孽镜可照前业，业力过重者受刑期延长、来世贬谪或天劫时魂体同诛。

### 5.2 功德成神（城隍等）

- **城隍**多由生前功德卓著者死后受地府敕封，为**功德成神**；阶品相当于筑基～金丹，驻阳间某城或某域，协理亡魂与阴事。
- 功德修行不直接提升“灵气境界”，但可影响**轮回待遇、天劫难度、地府录用**；部分天师府、符修传承中有“积功德以减天劫”的说法。

### 5.3 与剧情的关系

- 慕清歌结局入地府轮回，其生前功过由地府审判；林微补全世界本源、平衡灵浊可视为大功德，与终局选择衔接。
- 开天眼/观魂术在地府外派城隍传承中有残卷，修士可从城隍一脉获得开天眼机缘（`underworld_soul_system.md` §2、`faction_城隍.md`）。

## 6. 天道力量体系与规则体系 (Heavenly Dao & Rule Layer)

### 6.1 天道意识

- **性质**：**世界意志/规则显化**，非可加入势力；通过灵衍镜与镜中人与林微沟通，为世界规则层最上层。
- **职能**：引导林微完成使命（收集碎片、补全本源、平衡灵浊），维护世界平衡；镜中人为天道意识碎片之化身，终局可回归天道化身，予林微去留之选。
- **与其它规则层**：不直接指挥灵气守护者、浊气掌控者、世界碎片管理者、地府；彼等为自古传承的使命或阴面秩序，与天道**规则一致**而非上下级。详见 `faction_天道意识.md`。

### 6.2 规则层势力（与“力量”的关系）

| 存在 | 力量/权柄来源 | 说明 |
| :--- | :--- | :--- |
| **天道意识** | 世界意志、灵衍镜/天命面板 | 不直接出手，通过林微与镜中人推动 |
| **灵气守护者** | 灵脉与灵浊平衡、观测与调衡 | 防灵气枯竭、灵脉崩坏 |
| **浊气掌控者** | 浊气遏制与转化、灵浊交界 | 遏浊气泛滥，与灵气守护者协作 |
| **世界碎片管理者** | 三界壁垒与通道稳定 | 维护世界碎片与壁垒，与渡界使、上古遗迹相关 |
| **地府** | 阴面法则、魂力（阴能）、轮回权柄 | 司三界灵魂接引、审判、轮回；阴神以阴德与阶品论“修为” |

- **天罚/天劫**：业力过重者在渡劫或天谴中可被劈中魂体，导致魂飞魄散；天道降罚时“罪不容诛”者可连魂带魄一并抹杀（`underworld_soul_system.md` §1.2）。

### 6.3 世界本源与成神

- **世界本源**：由所有生命的信念与天地灵气等构成；受损则世界走向毁灭。补全方法：收集世界碎片、唤醒上古神邸、平衡灵浊、减少七情六欲失衡等（`worldview.md` §7）。
- **化神之上**：当前世界最高修为为化神，化神修士寿命约两千年；“神”已沉睡，成为修补世界本源的一部分，再造神极难；林微终局可成为“天道守护者”一级存在，与补全世界本源挂钩。

## 7. 灵根设定 (Spirit Roots)

**全分类文档**：修真界（正道）、浊界（魔修）、阴界（魂修/功德修）的灵根与体质**全分类**见 `world/settings/灵根与体质全分类.md`。以下为灵修侧摘要及与剧情衔接。

### 7.1 灵根的作用

- 灵根决定**能否吸纳灵气**（灵修）或**浊气**（浊修）及**吸纳的属性与效率**；无灵根或伪灵根者在凡界难以晋入练气，或进境极慢。
- 灵界修士普遍具备至少一种灵根；灵根属性与功法匹配则事半功倍，与体质结合可产生特殊异象（如九转金丹）。

### 7.2 灵根类型

| 类型 | 说明 | 稀有度 |
| :--- | :--- | :--- |
| **五行灵根** | 金、木、水、火、土单灵根或双灵根、多灵根 | 单灵根较优，多灵根杂而不纯、进境多较慢 |
| **变异灵根** | 雷、冰、风、毒等，由五行变异或机缘所致 | 较稀有，与特定功法与地域匹配时极强 |
| **伪灵根** | 五行混杂、亲和极弱，几乎无法引气入体 | 常见于凡界“废材”设定 |
| **无灵根** | 无法感应灵气，纯靠武道或外物 | 林微初始为“无灵根废材”，后经系统觉醒/蜕变 |
| **混沌灵根** | 可吸收与融合**各种属性**灵气，不拘一格 | **最稀有**；林微拥有此体质（与体质合并表述见下） |

### 7.3 与剧情的衔接

- **林微**：初始无灵根/伪灵根，经灵衍镜觉醒与机缘（如天命任务、筑基卷）获得**混沌灵根**或灵根蜕变，从而突破凡界界限。
- **坠星秘境**：进入条件为 10～15 岁且拥有灵根或特殊体质（`worldview.md` §5）。

## 8. 体质设定 (Special Physiques)

### 8.1 体质的作用

- 体质提供**被动能力、突破异象、抗性、或与灵根/功法的联动**；可与灵根叠加（如混沌灵根+九转金丹体质）。
- 部分体质与**开天眼、阴修、鬼修**门槛相关（如先天阴体、阴阳眼）。

### 8.2 常见体质类型

| 体质 | 说明 | 典型角色/势力 |
| :--- | :--- | :--- |
| **混沌灵根/混沌体** | 融合各属性灵气，修炼不拘功法属性；最稀有 | **林微** |
| **九转金丹** | 金丹期特殊境界，金丹品质极高，战力与潜力远超同阶 | **林微**（结丹时） |
| **先天阴体** | 易感魂力（阴能）与阴气，开天眼门槛可放宽至练气后期 | 地府外派、天师府相关 |
| **阴阳眼** | 血脉或机缘所致，可见魂体与阴气 | 与先天阴体类似，多用于“见鬼”设定 |
| **媚骨天成/玉人体** | 魅惑与亲和增强，偏辅助与外交 | **合欢宗**部分弟子 |
| **魔种体质** | 易吸纳浊气或与魔气共鸣，易被浊界势力盯上 | **夜琉璃**（苏青黛研究其体质） |
| **妖修血脉** | 狐族、龙族、鹤族等，血脉觉醒与种族天赋 | 白涂涂（青丘）、敖漓（龙宫）等 |

### 8.3 与各修行体系的关系

- **灵修**：混沌灵根、九转金丹、媚骨/玉人体等主要服务灵修线；妖修血脉在荒域九部中常见。完整名录见 `灵根与体质全分类.md` 第一节。
- **浊修**：魔种体质、部分“堕落”体质与浊气亲和；浊界灵根与体质全分类见 `灵根与体质全分类.md` 第二节。
- **阴修/功德**：先天阴体、阴阳眼降低开天眼与阴神沟通门槛；功德成神不依赖体质，依赖生前功德与地府敕封。魂修/鬼修/功德修体质见 `灵根与体质全分类.md` 第三节。
