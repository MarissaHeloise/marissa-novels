# 势力档案 (Faction Archives)

## 命名规则 (Naming Convention)

**文件名**：`faction_` + **势力中文名**（与档案内「名称」一致）

- 格式：`faction_<势力名>.md`
- 示例：`faction_天机阁.md`、`faction_天枢剑宗.md`、`faction_魔罗殿.md`
- 档案内 **ID** 与文件名保持一致：`**ID**：faction_<势力名>`

**与地点档案对应**：地点用 `location_<地点名>.md`，势力用 `faction_<势力名>.md`；二者均与 `world/settings/world_map.md`、`worldview.md` 及 `factions.md` 总纲一致。
