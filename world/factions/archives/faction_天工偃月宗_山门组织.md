## 一、宗门气质与地域绑定

幽机域 **玄冥幽机**：地表荒、灵脉与工坊 **深藏地下**，古机关城遗迹与暗河矿道纵横。山门依 **匠城** 而筑，**升降井、轨车、符阵灯、傀儡巡夜** 为日常。门风可概括为：

| 维度 | 表述 |
| :--- | :--- |
| **匠德** | 作品可验伪、工序可溯源；剽窃、以次充好为大忌。 |
| **实绩** | 贡献、考工名次、驻外战功与 **可交付的器/偃/傀** 并重。 |
| **地脉** | 内门洞府沿 **矿道灵脉** 开凿；外门聚 **匠城表层**；**禁地** 与古遗迹接驳处由 **镇渊一脉** 协同 **绳墨司** 共管。 |
| **通宝** | **万宝阁** 为宗门商贸组织（见下节）；**符行司** 主外事与驻点文书，**铢府司** 主大宗灵材入山；**域际传送** 在 **匠城·通宝台**（见 `location_幽机域.md`）。 |

### 1.1 修士构成：器修、偃修为干，辅脉并存

天工偃月是 **玄门大宗**，**不可能** 仅由单一传承构成。宗内以 **器修**（炼器、锻胚、灵火温养）与 **偃修**（偃甲、灵傀、机关战阵）为 **绝对主干**，人数与资源 **占七八成**；其余为 **辅脉与客修**，人数少、但 **职能不可缺**——多为 **服务匠道、护山、通商、矿道与遗迹作业**。

| 类型 | 在宗门中的地位 | 常见落点（示例） |
| :--- | :--- | :--- |
| **器修** | 主轴之一；**砺火峰** 为核心。 | 开炉、淬灵、器胚、灵器评鉴。 |
| **偃修**（含机关偃甲） | 主轴之一；**偃月峰** 为核心。 | 灵傀、战阵傀、驻外偃城。 |
| **阵修** | 辅脉；与九宫阵道 **协同** 者多从此出。 | **镇渊峰**、护山大阵、遗迹封印、试场 **验伪阵** 维护。 |
| **符修** | 辅脉；不抢主轴风头。 | **符阵灯**、传讯符、押运符、通宝台 **符禁**；可隶镇渊或匠城 **符院**。 |
| **剑修 / 护道** | 辅脉；**执法与押运** 刚需。 | **绳墨司** 执法队、驻外镖队、灵蚀域 **清剿小队** 与偃修混搭。 |
| **体修 / 土金灵根** | 辅脉；**矿道、暗河、遗迹** 下井。 | **铢府司** 矿卫、遗迹前哨；与 **偃师** 组队探窟。 |
| **丹修（偏火）** | 辅脉；**为炉口服务** 非以丹立宗。 | **砺火峰** 配炉、淬炼 **药引**、控温；少数驻 **万宝阁** 鉴定 **灵液**（与丹鼎盟有别）。 |
| **御兽 / 灵虫** | 辅脉；规模小。 | 矿道 **侦毒**、遗迹 **探隙**、传讯；不主打驭兽战。 |
| **商修 / 账房** | 辅脉；**万宝阁** 中占比高。 | 拍品估价、分阁对账、契书与 **货殖**；宗内 **督通宝** 一条线。 |

**叙事原则**：大宗门 **应有** 各色修士；正文写 **「某峰剑修」「万宝阁符掌柜」** 等 **不违和**。但 **升堂入阁、天骄册、宗门面** 仍以 **器/偃实绩** 为主轴，辅脉 **极少** 任峰主，除非 **战功或财阀** 特例。

### 1.2 万宝阁（宗门直属商贸组织）

**万宝阁** 不是外宗挂靠，而是 **天工偃月宗** 下设的 **对外商贸与拍品体系**：七域 **分阁**、幽机域 **总阁**、**万宝盛会** 与 **万宝试艺** 承办，**人事、账目、授意** 均受宗门 **督通宝 / 副宗主（通宝）** 节制；分阁 **阁主** 可为 **商修** 或 **器修** 出身，须 **匠籍** 或 **宗门契书** 绑定。

| 层级 | 说明 |
| :--- | :--- |
| **总阁** | 驻 **幽机域**（近 **匠城·通宝台**）；拍品 **背书**、域际 **调度**、与 **铢府司** 对大宗灵材。 |
| **分阁** | 各域大城；**试艺报名、发榜、引荐履诺** 多经分阁；利润 **按比例** 上解总阁。 |
| **与考工** | **万宝试艺** 外埠数据 **回传考工司**；头名 **内门名帖** 由宗门 **授权**、万宝阁 **执行** 接引礼仪。 |
| **人员** | **鉴宝师、估价、护卫、押运、符禁、账房** 混编；**剑修镖队** 与 **偃师** 押 **重器** 为常见组合。 |

**口径**：对外可称「万宝阁」；宗内文书常写 **「通宝一脉」** 或 **「万宝阁执事」**，与 **纯匠** 弟子区分。

### 1.4 仙盟灵网（神识传讯网络）

**归枢道君**（宗主 **晏归枢**）所创、仙盟共用的 **神识传讯网络**，全称 **「仙盟灵网」**，日常直呼 **「灵网」**（口语：**上灵网、入灵网**）。表意：**灵力为线、符印为结**，千里传讯、聚众议事，皆走网上。

| 概念 | 说明 |
| :--- | :--- |
| **千丝灵网** | **化神级大型法宝**本体，镇 **玄枢峰·枢藏**；仙盟灵网之 **枢纽**，诸枚 **灵网符** 皆自此分灵结出。 |
| **灵网符** | **唯一规格**之入网玉符，**不别内外**。本宗入籍配发；亦可以 **灵石向万宝阁等渠道价购**。**未滴血之白符** 可 **转赠、转售**；**已认主** 若欲易人，须 **通宝枢务** 办 **解绑销籍** 后，由受符者重新滴血，以免追责串线。每一枚自 **千丝灵网** 分灵结出时即烙 **符籍号**（全网 **独一**，终身与此物理符绑定，销籍后亦不 **重号发放**），挂失、办案、定向投递皆以此为凭。 |
| **私笺** | 点对点传讯。 |
| **名录** | 互加印痕为友，便于定向投递。 |
| **公议牍** | 题帖、跟帖，类坊间茶榜；**绳墨司**可依 **网迹、符痕** 追责违禁谤仙、泄密、贩禁等。 |

**口径**：网络直呼 **灵网** / **仙盟灵网**；**千丝灵网** 专指镇山 **法宝本体**，与峰名 **玄枢** 不混称。

## 三、四峰（内门传承）

**外门不设峰**，只隶 **匠城坊区**。内门 **四峰** 依地下灵脉而立，名称上 **避与宗名完全重复**，又保留 **偃月** 一脉为宗徽所系。

| 峰号 | 峰名 | 传承侧重 | 建筑/地脉意象 | 首座（姓名·境界·称谓） |
| :--- | :--- | :--- | :--- | :--- |
| **枢** | **玄枢峰** | **宗门枢机**、枢藏；宗主洞府所在。 | 主升降井、**通宝台** 毗邻；洞顶符灯如星链。 | **晏归枢** 兼领；日常协理枢藏：**穆徽音**（元婴初期，**徽音仙子**） |
| **傀** | **偃月峰** | **偃甲、灵傀、战阵傀**；前线驻点、傀潮任务主脉。 | 傀廊回环如月缺；试傀场占半座山腹。 | **凌弦月**（元婴中期，**弦月仙子**） |
| **火** | **砺火峰** | **炼器、锻胚、灵火调控**；开炉、淬灵课业。 | 地火井、淬池、防炸阵；金铁声与热浪终年不息。 | **温炽阳**（元婴中期，**炽阳真人**） |
| **禁** | **镇渊峰** | **机关阵盘、城防偃城、禁制拆解**；遗迹接口。 | 近 **古遗迹回廊**；弟子入遗迹常佩 **护识符**。 | **谢渊渟**（元婴中期，**渊渟真人**） |

**命名说明**：原拟「天工峰」易与宗名 **天工偃月** 叠字，现改为 **玄枢峰**（玄冥之枢）；「熔锋」改为 **砺火**（砺器于火，点明炉火一脉）。

## 四、四司（跨峰实务）

司署多在 **匠城** 与 **诸峰山麓** 设 **对外窗口**，职权清晰、少与峰重叠。

| 司名 | 别称意象 | 职能 | 地域/叙事钩子 |
| :--- | :--- | :--- | :--- |
| **绳墨司** | 匠作取直 | **门规、奖惩、矿道与遗迹禁入令**；试炼生死状、冲突裁断。 | 与 **暗河码头** 走私线天然对冲。 |
| **铢府司** | 铢两分明 | **灵金、机关木、阵纹石** 入库、分拨、**矿道配额**；对接 **灵金镇**、暗河矿道。 | 深窖分 **禁火区 / 防潮区**；失铢即失命。 |
| **考工司** | 古工正遗意 | **门内考工、小比、升阶**；**万宝试艺** 外埠发榜 **回传**；与 **匠籍殿** 联动升降档。 | 试场必有 **验伪阵、计时漏**；可复核。 |
| **符行司** | 符牒行天下 | **驻外分阁、灵蚀域驻点、九宫阵道联合任务** 的调遣与文书；**出宗腰牌、传送批条**。 | 与 **万宝阁** 分阁 **对账、抢功** 常起摩擦，易出戏。 |

**已并入他处**：原拟「传功司」——**功法兑换、师承登册** 归 **匠籍殿** 下属 **师承笔**（见下），避免「司」过多、与 **考工** 争权。

**辅脉弟子** 多 **挂靠** 某峰或某司（如 **绳墨执法剑修**、**万宝阁商修**），**匠籍** 上仍记 **辅脉标注**，晋升天骄 **以实绩论**，不唯器偃。

**四司督名（补）**：**考工司督** **公输仪**（金丹后期，**尺仪真人**）；**符行司督** **沈砚舟**（金丹后期，**砚舟真人**）。

### 7.1 履诺入门协领（通宝接引）

持履诺帖者入门，常由 **万宝阁·通宝一脉** 执事 **自通宝台引至匠籍殿**，验籍后亦可 **继续协领** 认路、安顿、**灵网** 入网（头名等待遇尤然）。通宝执事 **非四峰匠籍**，与内门峰务并行不悖。
