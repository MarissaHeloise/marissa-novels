# 势力档案：天工偃月（玄门九宗-器修）

**创建时间**：2026-02-28  
**最后更新**：2026-03-28  
**ID**：faction_天工偃月  
**状态**：Active

## 1. 基本信息 (Basic Information)

- **名称**：天工偃月（宗门文书常署 **天工偃月宗**）
- **别名/称号**：Heavenly Craft Moon
- **类型**：玄门九宗-器修 / 偃修（宗门内 **器修、偃修** 为干，兼收 **阵、符、剑护、体修矿卫、丹火辅炼、商修** 等辅脉，见 `faction_天工偃月宗_山门组织.md` §1.1）
- **所属界域**：修真界（灵界）
- **谱系**：玄门九宗之一；**炼器** 与 **偃甲灵傀** 并立，机关阵禁为辅。
- **势力阵营**：秩序·灵界（玄门正统，器修一脉，与蚀骨渊对立）
- **创建与发展背景**：据幽机域；**万宝阁** 为宗门 **直属商贸组织**（总阁、分阁、拍卖、万宝试艺承办），与 **铢府、符行、督通宝** 一条线，与林微炼器线交汇；与九宫阵道在秘境与要地常联手，与浊界蚀骨渊骨器傀兵为对立面，与凡界丹鼎盟为凡器与灵器分野。

## 2. 职能与特色 (Function & Characteristics)

- **特色**：**器修、偃修** 为宗门主干；**万宝阁** 为 **直属商贸组织**（非外宗），七域分阁、拍卖、试艺与 **货殖** 皆归宗门节制；与 **九宫阵道** 等协同时，宗内 **阵、符、剑护** 等辅脉齐备。

### 万宝试艺（外埠一体式活动）

各域 **万宝阁** 承办、**天工偃月宗** 授权并监督的 **同一场** 外事活动，兼具：

| 面向 | 内容 |
| :--- | :--- |
| **试艺** | 器修、偃师、机关师同台较艺，按场公布优劣。 |
| **外埠考工·发榜** | 相当于宗门在边陲设 **考工**，发榜昭示名次与录用意向。 |
| **选拔** | 为 **天工偃月宗** 吸纳外埠人才，与内门晋升通道挂钩。 |
| **奖励** | **灵石**、**法宝**（届别与分场不同，正文可具体化）。 |
| **头名** | **第一名** 可获 **直入天工偃月宗内门** 的正式邀请（名帖）。 |

**入宗路径（叙事常用顺序）**：于 **外埠万宝阁** 报名并参加 **万宝试艺** → **海选** → **正赛夺魁（头名）** → 当场或发榜后领取 **内门弟子资格/名帖** → **万宝阁** 方面 **引荐**，乘 **域际传送阵** **再赴幽机域** 山门 **履诺入宗**（本山在幽机域，外埠试艺不必开局即在幽机）。入宗后修习、领 **灵蚀域** 外派任务等，见 `plots/volume1_chapter_plan.md` 幕二（**卷一** 与白涂涂再会安排在 **灵蚀界寻人偶遇**，**非**林微例行绕访幻花域）。

**说明**：不设单独的「天工阁」「灵偃宗」；林微入宗、赚资源、扬名，均走 **天工偃月宗—万宝阁** 这一条线即可。散修口头上可称 **「天工大比」**（强调器修、偃师同台较艺），与 **万宝试艺** 指同一套外事活动，正文二名可并用（一俗一正式）。
- **主修功法**：天工诀、炼器心法、机关与傀儡炼制、偃甲与灵器锻造。
- **主要资源**：幽机域山门与工坊、万宝阁总阁及各域分阁、灵材与拍卖渠道、与九宫阵道的秘境封印协作、林微炼器线成长节点。
- **势力特点**：灵器与傀儡正统；与蚀骨渊一灵一浊对立；万宝盛会为宝物流通枢纽。
- **势力目标**：维持幽机域器修话语权；拓展万宝阁与各域拍卖；与九宫阵道巩固秘境与封印合作。

## 3. 活动区域与地域 (Activity & Geography)

- **活动区域**：灵界幽机域及七域大城；万宝阁在各域设点。
- **地域对应**：幽机域为主；各域要地、秘境封印需器修与阵修协作。

## 4. 主要人物与职位 (Key Figures & Positions)

- **主要职位**：宗主、偃师、炼器长老、万宝阁总阁主与各域主事、工坊与秘境协作负责人。
- **已知/核心人物**：**林微**（炼器线可经天工偃月或万宝阁成长）。**宗主晏归枢**（归枢道君）、**副宗主云疏桐**（疏桐仙子）、**副宗主商时渺**（时渺真人）、**万宝阁总阁裴照雪**（照雪仙子）等 **姓名·境界·称谓** 见 **`faction_天工偃月宗_山门组织.md` §八**。
- **山门组织（详）**：诸峰、诸司、堂阁、弟子阶次、**称谓惯例** 与 **NPC 一览**，见 **`faction_天工偃月宗_山门组织.md`**。

## 5. 势力关系 (Faction Relationships)

- **与九宫阵道**：阵修与器修，各域要地与秘境封印常联手。
- **与幻戏宗**：凡界彩门傀儡与灵界天工傀儡，可设渊源彩蛋。
- **与蚀骨渊**：浊界骨修与尸道也炼骨器傀兵，与天工偃月为对立面。

## 6. 重要据点与关联地点 (Key Locations)

- **据点**：幽机域天工偃月山门、万宝阁总阁及各域分阁；万宝盛会举办地。
- **关联地点**：幽机域、七域大城、各秘境与封印节点。
