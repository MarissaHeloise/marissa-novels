# 势力关系与潜在冲突设定 (Faction Relationships & Potential Conflicts)

**创建时间**：2026-02-28  
**最后更新**：2026-02-28  
**说明**：本文档系统梳理各势力间关系类型与潜在冲突诱因，与各 `faction_*.md` 中「势力关系」一节互补；用于剧情推进、阵营任务与冲突线设计。各势力档案内仍保留简要关系，详细矩阵与冲突设定以本文为准。

---

## 1. 关系类型定义 (Relationship Types)

| 类型 | 说明 | 典型表现 |
| :--- | :--- | :--- |
| **同盟** | 共同目标、长期盟约或从属 | 地府—十殿/勾魂使/城隍；大衍—地脉九司 |
| **紧密合作** | 日常协作、资源共享、互不背刺 | 镇妖司—金刚寺/九宫阵道（守边）；神农谷—苍梧灵族—镇妖司（苍梧） |
| **合作** | 有共同利益，可临时或长期联手 | 玄门九宗内部；百草堂—神农谷；渡界使—万货帮（商贸） |
| **中立** | 无直接利害，可随剧情偏合作或对立 | 天机阁—多数势力；合欢宗—七域调停 |
| **紧张** | 利益摩擦、理念不合，未到开战 | 镇妖司—东海龙宫；纵横楼—影卫；地师会—灵植司（灵脉） |
| **对立** | 常态对抗、执法/秩序 vs 违法/混乱 | 问心局—血影门；地府—幽冥教/噬魂殿；天枢剑宗—青丘（镇妖立场） |
| **死敌** | 根本利益与秩序冲突，不可调和 | 地府—噬魂殿（轮回 vs 吞魂）；玄门正统—浊世九渊（灵浊） |

## 2. 凡界内部关系与潜在冲突

### 2.1 大衍王朝与地脉九司

| 关系对 | 关系类型 | 潜在冲突 |
| :--- | :--- | :--- |
| 大衍—影卫 | 同盟（皇室直属） | 影卫与纵横楼门生同在朝堂活动，慕清歌上位前后可爆发清洗或制衡冲突 |
| 大衍—钦天监 | 同盟 | 钦天监观测与天机阁民间占卜竞争；若观测结果与皇室利益不符，可被施压 |
| 大衍—镇妖司/问心局 | 紧密合作 | 镇妖司总府在苍梧，凡界分司听调；问心局执法若触及权贵，可与纵横楼/影卫线交叉 |
| 大衍—纵横楼 | 紧张 | 爵门渗透六部，慕清歌线可与之博弈；血煞盟与纵横楼有脉络，镇压血煞盟时牵出纵横楼 |
| 大衍—市井九流其余 | 中立～紧张 | 百草堂/丹鼎盟/万货帮与朝廷有税赋与管制；玄机殿盗墓、霓裳坊暗杀若曝光可遭问心局与守陵人追查 |

### 2.2 市井九流内部

| 关系对 | 关系类型 | 潜在冲突 |
| :--- | :--- | :--- |
| 天机阁—钦天监 | 紧张 | 星象与秘境情报的官方 vs 民间，灰色合作与竞争并存 |
| 纵横楼—影卫 | 紧张 | 朝堂与地方情报与暗杀重叠，可合作可互相铲除 |
| 纵横楼—血煞盟（血影门凡界分支） | 合作（灰色） | 纵横楼为血煞盟提供权谋与情报；若朝廷镇压升级，纵横楼可能切割或反噬 |
| 百草堂—万毒窟 | 对立 | 一医一毒，凡界百草堂与浊界万毒窟无直接交战但立场对立；凡界若出现毒案可牵出万毒窟线索 |
| 玄机殿—守陵人 | 对立 | 盗墓 vs 护陵，玄机殿暗业若在皇陵或上古遗迹动手，守陵人必究 |
| 玄机殿—藏书楼 | 紧张 | 禁忌知识流通 vs 封印；可演化为「某秘本流出」导致两方对峙 |
| 地师会—灵植司/九宫阵道 | 紧张 | 凡界灵脉与凡灵交界灵脉归属争夺；九宫阵道若在凡界布阵可与地师会冲突 |
| 霓裳坊—幻戏宗 | 合作～紧张 | 同属演艺与「幻」系，可联手可争地盘；幻戏宗致幻劫掠若波及霓裳坊客户会翻脸 |

### 2.3 凡界—资源与通道

- **渡界使**：与铸币司、万货帮在凡灵商贸上有交集；若朝廷收紧天门关/龙门，渡界使与万货帮利益受损，可形成潜在冲突。
- **守陵人**：与地府在尸变亡魂上协作；与玄机殿、盗墓势力对立；凡灵交界上古遗迹被破坏时可与渡界使、世界碎片管理者线索交叉。

## 3. 灵界内部关系与潜在冲突

### 3.1 玄门九宗内部

| 关系对 | 关系类型 | 潜在冲突 |
| :--- | :--- | :--- |
| 玄门九宗彼此 | 合作 | 共同维护灵界秩序、七域争霸中可组联军；宗门大比与资源争夺可产生局部摩擦 |
| 天枢剑宗—青丘妖国 | 对立（门派层面） | 镇妖立场 vs 妖族；莫惊尘（剑修）与白涂涂（青丘）为不同线，门派层面可对峙，避免全面开战 |
| 神农谷—万毒窟 | 对立 | 药修 vs 毒修，苍梧灵界侧与浊界毒瘴魔林对峙；百草谷与万毒窟若在边陲争夺灵植或试药会冲突 |
| 金刚寺/九宫阵道—魔罗殿/无相窟 | 对立 | 守边与封印 vs 浊界渗透；凛渊域前哨与极北裂缝为常备冲突点 |
| 紫霄法脉—大衍王朝 | 紧密合作 | 功法同源，慕清歌线；若慕清歌传位或自杀，紫霄与凡界皇室关系仍可延续或重组 |

### 3.2 荒域九部与玄门

| 关系对 | 关系类型 | 潜在冲突 |
| :--- | :--- | :--- |
| 青丘—凌霄符殿 | 合作（联姻/交流） | 白涂涂与上官晚为 CP，两派有联姻；若玄门整体对妖族施压，凌霄符殿可能被夹在中间 |
| 东海龙宫—镇妖司 | 紧张 | 龙宫偏中立或反抗，镇妖司监管妖邪；龙门通道与龙族尊严可触发摩擦 |
| 东海龙宫—逆乱盟/碧落海族 | 紧张～合作 | 坠星海共域，有贸易有摩擦；海盗与龙宫冲突可升级 |
| 万兽山庄—蛮荒战部/荒火部落 | 紧张 | 荒狩域兽源与地盘争夺；有战有和，灵潮或资源枯竭时易爆发 |
| 苍梧灵族—神农谷/镇妖司 | 紧密合作 | 苍梧边陲与封印协作；与万毒窟对峙时三方同线 |
| 逆乱盟—玄门/镇妖司 | 对立 | 打破屏障 vs 维护秩序；坠星海与凡灵交界为冲突前沿 |
| 陨星遗民—渡界使/世界碎片管理者 | 中立～紧张 | 上古遗迹与通道，可有渊源或对立；秘境与接天塔线索可牵出冲突 |

### 3.3 合欢宗与灵界

| 关系对 | 关系类型 | 潜在冲突 |
| :--- | :--- | :--- |
| 合欢宗—焚心盟 | 对立 | 阴阳调和 vs 极端执念；理念相悖，合欢宗视焚心盟为「情欲之邪」 |
| 合欢宗—玄门九宗 | 中立～合作 | 玄门有戒心但不列为邪修；七域争霸中合欢宗作调停，若调停失败可被一方迁怒 |
| 合欢宗—青丘/妙音坊/凌霄符殿 | 合作 | 联姻与交流；利益一致时稳固，若某域对「双修」污名化可波及合欢宗 |

## 4. 浊界内部关系与潜在冲突

### 4.1 浊世九渊彼此

| 关系对 | 关系类型 | 潜在冲突 |
| :--- | :--- | :--- |
| 魔罗殿—蚀骨渊 | 紧密合作 | 军工协作，傀兵与骨甲；利益一致时稳固，若蚀骨渊坐大或垄断尸骨源可生隙 |
| 幽冥教—噬魂殿 | 合作～紧张 | 同涉魂与亡魂，幽冥教拘役、噬魂殿吞噬；争夺魂源与灵浊交界魂口时可冲突 |
| 幽冥教—蚀骨渊 | 合作～紧张 | 尸傀与尸骨炼器；亡魂与尸骨资源分配可争 |
| 血影门—逆乱盟（血煞盟） | 合作（脉络） | 血煞盟为凡界分支，争夺坠星秘境；若逆乱盟与灵界全面开战，血影门可能被卷入或切割 |
| 无相窟—九宫阵道/金刚寺 | 对立 | 渗透 vs 守边；无相窟谍报若暴露可遭灵界清算 |
| 焚心盟—妙音坊/合欢宗 | 对立 | 极端执念 vs 音修摄魂/双修调和；理念冲突可化为追杀或围剿 |
| 渊墟殿—灵气守护者/浊气掌控者 | 对立 | 崇拜深渊、浊气失控 vs 规则层平衡；渊墟殿若冲击灵浊交界会与两翼冲突 |
| 万毒窟—神农谷/百草堂 | 对立 | 毒修 vs 药修；苍梧毒瘴魔林与百草谷对峙，试毒与解毒可成剧情线 |

### 4.2 浊界与灵界/凡界

- **魔罗殿/血影门/幽冥教/噬魂殿** 与 **玄门/问心局/镇妖司/地府** 的冲突为常态；潜在冲突诱因：灵浊裂缝扩大、苍梧封印被破坏、凡界血煞盟叛乱升级、魂源争夺战、灵潮病与浊气泄露。
- **无相窟** 渗透凡界与灵界，潜在冲突：谍网被破、要员被刺、秘境情报泄露导致某一方先发制人。

## 5. 与阴界·地府的关系与潜在冲突

### 5.1 地府与各方

| 关系对 | 关系类型 | 潜在冲突 |
| :--- | :--- | :--- |
| 地府—问心局/镇妖司/守陵人 | 紧密合作 | 亡魂、妖魂、尸变、厉鬼处理；若某方越权插手轮回或隐瞒亡魂，可生摩擦 |
| 地府—幽冥教 | 死敌 | 拘役亡魂 vs 接引轮回；地府会追讨被拘魂灵，幽冥教需魂源，不可调和 |
| 地府—噬魂殿 | 死敌 | 吞噬神魂 vs 轮回；噬魂殿若大规模吞魂会遭地府与阳间势力联合围剿 |
| 地府—蚀骨渊 | 对立 | 尸道与亡魂秩序；蚀骨渊炼尸若用新亡魂可触地府逆鳞 |
| 地府—合欢宗/问心局 | 合作 | 合欢宗戒律禁采补，问心局执法；若合欢宗弟子在凡界违规（如采补凡人），问心局介入、地府记业力 |
| 地府—灵气守护者/浊气掌控者 | 间接相关 | 世界大失衡时亡魂激增或轮回异常，与规则层在「世界平衡」层面衔接；无直接从属 |

### 5.2 城隍、勾魂使与阳间

- **城隍** 与当地**大衍官府、天师府、修士**：协作超度与阴事；潜在冲突：阳间势力强留亡魂、或城隍庙被毁导致鬼门失控。
- **勾魂使** 与**灵界/浊界大能**：若大能阻挠接引或私扣亡魂，可爆发冲突；常态为地府优先以规则与谈判解决。

## 6. 与规则层的关系与潜在冲突

### 6.1 规则层势力彼此

| 关系对 | 关系类型 | 潜在冲突 |
| :--- | :--- | :--- |
| 天道意识—灵气守护者/浊气掌控者/世界碎片管理者 | 规则一致 | 不直接指挥，使命与天道一致；若某方曲解使命或失控，天道可通过林微/镜中人间接纠正 |
| 灵气守护者—浊气掌控者 | 紧密合作 | 灵浊平衡两翼；若灵界或浊界一方撕毁平衡协议，两翼会施压 |
| 世界碎片管理者—渡界使 | 监管～合作 | 渡界使利用通道，管理者维护壁垒；若渡界使滥用通道导致壁垒不稳，管理者会介入 |
| 世界碎片管理者—逆乱盟 | 对立 | 打破屏障 vs 维护壁垒；逆乱盟若冲击关键节点会与管理者或其代理人冲突 |

### 6.2 规则层与世俗势力

- **天道意识** 仅通过**林微与灵衍镜**推动，不与玄门/九渊直接对话；潜在冲突：若林微与某势力为敌，该势力可能视「天命」为敌。
- **灵气守护者/浊气掌控者** 与 **噬魂殿/幽冥教**：魂能失控、大规模吞魂或拘役会导致阳间秩序崩坏，间接影响灵脉与生灵，两翼可能支持地府或阳间势力讨伐。
- **世界碎片管理者** 与 **陨星遗民、接天塔遗址、上古遗迹**：渊源或对立；林微收集碎片与管理者目标重合，可与剧情主线结合。

## 7. 主要冲突轴汇总 (Major Conflict Axes)

| 冲突轴 | 代表势力对 | 诱因示例 | 可剧情化方向 |
| :--- | :--- | :--- | :--- |
| **灵浊正统** | 玄门九宗 vs 浊世九渊 | 苍梧封印、极北裂缝、灵潮病、浊气泄露 | 守边战、渗透与反渗透、灵浊和解线 |
| **轮回与魂** | 地府 vs 幽冥教/噬魂殿 | 魂源争夺、大规模拘役/吞魂、厉鬼暴动 | 地府讨伐、问心局/镇妖司协防、魂飞魄散与救赎 |
| **凡界秩序** | 大衍/问心局/影卫 vs 纵横楼/血煞盟 | 慕清歌上位、血煞盟叛乱、云州与坠星秘境 | 朝堂博弈、镇压与切割、林微/慕清歌线 |
| **镇妖与妖** | 镇妖司/天枢剑宗 vs 东海龙宫/青丘 | 龙门归属、妖族为祸指控、七域争霸站队 | 叶红绡/敖漓线、莫惊尘/白涂涂门派立场 |
| **秘境与通道** | 逆乱盟/血煞盟 vs 镇妖司/九宫阵道/世界碎片管理者 | 坠星秘境、屏障裂隙、天门关管制 | 林微秘境线、渡界使与管理者监管 |
| **资源与地盘** | 万兽山庄 vs 蛮荒战部/荒火；地师会 vs 灵植司/九宫阵道 | 兽源、灵脉、矿脉 | 边陲争夺、凡灵交界纠纷 |
| **禁忌与秩序** | 藏书楼/守陵人 vs 玄机殿/幽冥教/蚀骨渊 | 盗墓、禁忌知识流出、尸变与尸傀 | 守陵人护陵、藏书楼封禁与玄机殿黑市 |
| **规则层平衡** | 灵气守护者/浊气掌控者 vs 渊墟殿/焚心盟（失控） | 灵浊失衡、深渊崇拜、执念泛滥 | 林微补全本源、唤醒上古神邸、平衡灵浊 |
