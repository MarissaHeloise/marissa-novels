# 坠星秘境 · 试炼势力与背景板

**用途**：为第一卷坠星秘境篇（第41–62章）提供**散修势力、江湖武者势力、中小势力**背景板，用于添加事件、冲突与路人戏，增强故事性。  
**关联**：`location_坠星秘境.md`、`plots/volume1_chapter_plan.md`、`world/settings/power_systems.md`（凡界武者/练气压制）

---

## 一、散修势力（无宗门归属或小团体）

| 名称 | 定位 | 特点 | 事件/冲突钩子 |
| :--- | :--- | :--- | :--- |
| **星尘散人** | 独行或 2–3 人小队，常年在凡界与秘境边缘讨生活 | 熟悉秘境外围路线与星铁产出点；不惹大势力，见机行事；练气中后期为主 | 与林微争同一处星铁/灵草；被血煞盟驱赶时与林微同路暂合作；提供「某条路有机关」的情报换保护 |
| **拾荒客** | 散修组成的松散联盟，以捡漏、挖遗迹残片为生 | 人多但战力参差，内部有头目与跟班；与要门·万货帮、火门·丹鼎盟有私下交易 | 在某遗迹入口与林家或林微队伍抢道；有人私藏潜龙令引发内讧，波及路过者；可做情报源（哪里出了好东西） |
| **东莱散修会** | 东莱州本地散修抱团，非正式组织 | 多为练气前期～中期，有简单约定「遇险互相照应」；对青阳城、坠星郡情况熟 | 盛会前与林微同船/同路；秘境中某支小队遇妖兽求救，林微可选救或不救；与血煞盟冲突时被当炮灰，林微可出手或冷眼 |
| **遗民后裔** | 自称与陨星遗民有血脉或传承渊源的零散修士 | 对秘境古阵、星陨传说格外执着；人数少，偶有古怪仪式或禁忌知识 | 在传承地外围与林微方向一致，可短暂同行或竞争；提供「偃师遗迹某条岔路」的真假情报，制造选择与代价 |

## 二、江湖武者势力（武道为主，秘境压制下占优）

| 名称 | 定位 | 特点 | 事件/冲突钩子 |
| :--- | :--- | :--- | :--- |
| **青阳镖局** | 东莱州青阳城及周边走镖，兼接护送到坠星湖的生意 | 总镖头化劲、镖师内劲为主；与要门·万货帮、云州商会有往来；进秘境为「护主」或「顺带寻机缘」 | 护送某批试炼者入秘境途中遇伏（血煞盟或散修）；秘境内与林微同遇机关，武者破机关、修士辨阵眼可分工；某镖师贪心独走遇难，引发道德选择 |
| **断江堂** | 江湖门派，以刀法刚猛著称，活动于东莱—云梦一带 | 堂主或长老为宗师/化劲；弟子进秘境历练、争潜龙令以扬名；与天师府有旧怨或旧交（可二选一） | 在争夺某处灵脉节点时与林微队伍冲突后罢手；某弟子被血煞盟所害，断江堂与林微共同对敌；可引出天师府玄青的过往 |
| **坠星武馆** | 坠星郡本地武馆，每届盛会都派弟子入秘境 | 外劲～内劲为主，重在「见世面」；与秘境原住民、坠星镇商户关系熟 | 入口处与林微交换「今年哪条路好走」；秘境中某弟子迷路被林微所救，日后可做坠星镇线人；武馆师傅在湖畔与周姑姑/林家管事有旧识，人情小戏 |
| **云梦水寨** | 云梦州绿林，以水陆两路讨生活，非纯匪 | 寨主化劲，手下内劲；进秘境多为寻宝或受雇于某方；与要门、爵门有灰色往来 | 与血煞盟临时合作夺宝后翻脸；在渡河/过机关区时与林微队伍抢先后，可打可谈；某人认出资质好的秋蝉/慕清歌，起招揽或歹心，制造小危机 |

## 三、中小势力（小宗门、地方世家、商会护卫）

| 名称 | 定位 | 特点 | 事件/冲突钩子 |
| :--- | :--- | :--- | :--- |
| **碧波庄** | 东莱州近海小世家，以水属功法与渔盐生意立足 | 练气为主，偶有筑基长辈不入秘境；与林家无仇无恩，可交易可竞争 | 在某灵泉/灵脉点与林微先后抵达，协商分配或短暂冲突；庄中某人识得慕清歌身份，可制造「保密/灭口/卖人情」选择 |
| **丹霞观** | 凡界小宗门，以炼丹与采药见长，与疲门·百草堂有渊源 | 弟子练气前期～中期，进秘境多为采灵药、寻丹方；不擅厮杀，常依附大队伍 | 遇妖兽围攻时向林微队伍求救，事后以丹药或情报答谢；观中长老与百草堂某人的关系可在对话中带出，为卷二/卷三百草谷线埋笔 |
| **云州商会·护卫队** | 要门·万货帮分支，护送商会子弟或货物入秘境 | 练气+武者混编，以保护「少爷/小姐」或押运物资为主；与林家、血煞盟在云州有利益纠葛 | 秘境中与血煞盟夺宝冲突，林微可帮一把换人情；某护卫认出林微是「青阳城林家少家主」，口风可紧可松，影响后续云州线 |
| **铁骨门** | 凡界小派，主修外功与筋骨，偏体修路子 | 弟子外劲～内劲多，在秘境压制下肉身占优；与断江堂有旧怨（争地盘或争名） | 与断江堂在秘境中狭路相逢，林微可劝和、选边或趁乱通过；某弟子欠林微人情（如指路、分药），日后可还 |
| **钦天监外派** | 地脉九司·钦天监派出的观察使与随从，非主力作战 | 记录秘境异象、潜龙令分布、势力动向；练气为主，有简单护身手段 | 向林微打听「某处是否见过异常星象/古阵」；可透露「血煞盟本次人手与路线」的模糊情报换合作；为问心局/地脉九司后续出场做铺垫 |

## 四、使用建议（与章节对应）

- **第41–43章（秘境初探、叶红绡、血煞盟袭杀）**：可插入**星尘散人**争路/暂合作、**拾荒客**抢道或内讧波及、**青阳镖局**或**断江堂**与血煞盟遭遇。
- **第44–45章（白涂涂、涂涂认主）**：**坠星武馆**弟子迷路、**东莱散修会**某支小队遇险，白涂涂可带路或捣乱制造笑料与选择。
- **第46–49章（慕清歌遇险、四人汇合、共渡难关）**：**云梦水寨**与血煞盟翻脸、**碧波庄**识破慕清歌身份、**云州商会护卫队**与血煞盟冲突可让林微选边；**丹霞观**求救与答谢可放在汇合前后。
- **第50–52章（传承在望、偃师遗迹、傀儡之缘）**：**遗民后裔**同向而行或假情报、**钦天监外派**打听古阵异象、**铁骨门**与**断江堂**在遗迹外围冲突。
- **第61–62章（秘境将闭、出秘境）**：**拾荒客**或**东莱散修会**有人抢在关闭前夺宝引发混乱；**青阳镖局**/ **坠星武馆**与林微在出口处再遇，可收人情或口风。

写作时从中任选 1–2 股势力 per 章节即可，不必全上；冲突以**推动主线（血煞盟、叶红绡、慕清歌、白涂涂）与林微选择**为主，背景板不喧宾夺主。
