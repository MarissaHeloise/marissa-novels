# 区域档案：阴界（冥土/幽都）

**创建时间**：2026-03-01
**最后更新**：2026-03-01
**ID**：location_阴界
**状态**：Active

## 1. 基本信息 (Basic Information)

- **名称**：阴界
- **别名/称号**：冥土、幽都、地府所在
- **类型**：规则层界域（与凡/灵/浊三界并列）
- **所属世界**：灵衍世界
- **所属界域**：阴界
- **地理位置**：与凡界、灵界、浊界并列，规则层上的另一面，不在三界空间内
- **面积**：无阳间尺度；魂灵流转空间

## 2. 地理与势力 (Geography & Faction)

- **地形**：冥土风貌，阴气弥漫，魂能充沛；魂灵在此可补充魂能。
- **地府**：阴界唯一秩序主体，司三界灵魂接引、审判、洗业与轮回。详见 `underworld_soul_system.md`。
- **进入方式**：死者经黄泉路或勾魂使接引；生者极少数经鬼门、阴阳井、奈河渡（秘法或地府特许）。阳间侧：凡界城隍庙、乱葬岗、古战场等有鬼门或黄泉岔口。

## 3. 魂灵流程与地点 (Soul Flow)

鬼门关 → 黄泉路 → 酆都（十殿）→ 望乡台 → 忘川/奈何桥（孟婆汤）→ 轮回井/转生池。各地点见独立档案。

## 4. 冥土分区、道路节点与地标 (Underworld Layout, Routes & Landmarks)

阴界无阳间尺度，魂灵依流程与地府规则移动；以下为便于剧情取景的节点与地标。

### 4.1 冥土分区与节点

| 分区/节点 | 说明 |
| :--- | :--- |
| **鬼门关外** | 魂灵初入阴界，阴兵把守；生人勿近 |
| **黄泉路** | 勾魂使带魂经此路入阴；岔口可通阳间城隍庙、乱葬岗、古战场等 |
| **酆都外城** | 魂灵排队候审、暂歇；阴差与引路吏活动区 |
| **酆都内城·十殿** | 一殿秦广（初审/孽镜）→ 二至九殿依业受刑或枉死城 → 十殿转轮（轮回发放） |
| **望乡台** | 在五殿阎罗殿外；魂灵可最后回望阳间故土与亲人 |
| **忘川—奈何桥** | 忘川洗业障与执念；奈何桥跨忘川，孟婆施汤，饮后忘尽前尘 |
| **轮回井/转生池** | 刑满或直判之魂由此投入轮回，转世至凡/灵/浊 |

### 4.2 道路与通道

- **魂灵主路**：鬼门关 → 黄泉路 → 酆都 → 十殿（依业分流）→ 望乡台（可选）→ 忘川/奈何桥 → 轮回井。
- **阳间侧节点**：凡界城隍庙、乱葬岗、古战场等阴气汇聚处有鬼门或黄泉岔口；灵界/浊界由对应阴气节点或勾魂使接引。生者入阴极少数经鬼门、阴阳井、奈河渡（秘法或地府特许）。

### 4.3 奇观与地标

- **孽镜台**（一殿秦广）：照见生前业障，初审分流。
- **望乡台**：最后回望阳间，常有魂灵在此久驻不肯离去。
- **忘川**：阴界大河，坠入者沉沦历劫；岸畔有“忘川石”传说。
- **奈何桥**：孟婆在此施汤；桥头有“三生石”等传说地标。
- **枉死城**（六殿卞城）：横死、自杀、冤死未超度之魂暂居；沈梦蝶类冤魂可入。
- **转轮殿**：十殿最后一殿，依业力与因果分配轮回井池口（凡/灵/浊、胎卵湿化等）。
