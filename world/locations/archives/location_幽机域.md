# 区域档案：幽机域（玄冥幽机）

**创建时间**：2026-03-01
**最后更新**：2026-03-01
**ID**：location_幽机域
**状态**：Active

## 1. 基本信息 (Basic Information)

- **名称**：幽机域
- **别名/称号**：玄冥幽机、地机关境、偃师域
- **类型**：灵界七域之一
- **所属世界**：灵衍世界
- **所属界域**：灵界
- **地理位置**：灵界北方与地底，地表荒凉，主体为地下机关城与暗河网
- **面积**：域内约数百万平方公里

## 2. 地理环境 (Geographical Environment)

- **地形地貌**：地下溶洞、机关城、升降机与符阵通道、古遗迹与傀儡工坊；地表多风沙，灵脉深藏地下。
- **气候特征**：地底阴冷潮湿；地表多风沙。
- **自然资源**：灵金、古机关核心、傀儡残骸、土系与金系灵脉、上古偃师传承。
- **生态系统**：机关城（地底层）古机关网络、暗河与灵金矿道、古遗迹回廊（残阵与陷阱密布）。

## 3. 势力分布 (Faction Distribution)

- **天工偃月**：器修圣地，为本域魁首；与**林微**炼器线交汇。
- **九宫阵道地脉**：以山川与地脉为阵，在此布阵。
- **市井九流**：册门·玄机殿在此寻古籍与复刻秘籍；火门·丹鼎盟有地火分舵。
- **其他**：机关世家、傀儡军、守陵人常入此地探索古陵。

## 4. 历史背景 (Historical Background)

幽机域为上古偃师文明遗迹所在，世界破碎后地壳变动，大量机关城沉入地底，后人据此重建“玄冥”风格的地下文明。灵潮会激活古机关与傀儡，既带来机缘也带来凶险。**林微所得偃师传承与坠星秘境中的机关，多与此域古法同源。**

## 5. 特殊资源 (Special Resources)

- **灵金、机关木、阵纹石**；炼器、傀儡、阵盘与城防机关产业链完整。
- **幽机域城**：器市为核心，符箓、阵盘、傀儡零件与灵金交易繁盛。
- **天工偃月山门**：外门作坊群形成“匠城”。

## 6. 交通情况 (Transportation)

- **地下交通**：轨道推车/暗河舟发达；陷阱、失控傀儡、矿道塌陷为常见风险。

## 7. 重要地点 (Important Locations)

- **幽机域城**：器市、符箓、阵盘、傀儡零件与灵金交易。
- **天工偃月山门**：器修圣地，匠城围绕。
- **机关城（地底层）**：古机关网络，层层门禁，常见失控傀儡。
- **暗河与灵金矿道、古遗迹回廊**：上古偃师遗构。

## 8. 境内据点、地形与资源 (Settlements, Geography & Resources)

### 8.1 城池、镇坊与村落

| 类型 | 名称 | 说明 |
| :--- | :--- | :--- |
| **域城/大据点** | 幽机域城、天工偃月山门（匠城） | 域城器市为核心，符箓/阵盘/傀儡零件/灵金交易；匠城为天工偃月外门作坊群 |
| **镇/坊** | 灵金镇、暗河码头镇、遗迹前哨镇 | 灵金镇依矿道而建；暗河码头镇为地下舟运枢纽；遗迹前哨镇为古遗迹探险登记与补给 |
| **村落/营地** | 矿工聚居区、偃师世家庄园、守陵人营地 | 矿道外围与世家庄园散布；守陵人常入古遗迹，有临时营地 |

### 8.2 山川、地下与道路

- **地表**：风沙丘陵、灵脉外泄口（少数）、升降机井与符阵通道入口。
- **地下**：机关城（地底层）、暗河网、灵金矿道、古遗迹回廊（残阵与陷阱）。
- **道路**：域城—匠城官道、域城—灵金镇矿道、暗河舟线、遗迹前哨—古遗迹栈道（轨道推车/暗河舟）。

### 8.3 奇观与资源点

- **奇观**：机关城核心“玄冥之心”、古偃师巨像残骸、暗河地下瀑布、灵金矿脉荧光带。
- **资源点**：灵金、机关木、阵纹石、古机关核心、傀儡残骸、上古偃师传承遗迹；陷阱与失控傀儡为常备风险。

## 9. 居民情况 (Residents)

- **主要种族**：人族，以偃师与机关世家为主。
- **人文风俗**：尚“匠德”与师承谱系；符阵灯笼、机关城、升降梯与傀儡代步；以作品与功绩论高低，坊市规矩森严。

## 10. 区域特色 (Regional Characteristics)

- **林微炼器线**核心舞台之一；与坠星秘境偃师传承同源。
- **万宝阁**（天工偃月在凡界商贸分支）与本域互通。
