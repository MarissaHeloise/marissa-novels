# 区域档案：龙门（凡界—灵界海底通道）

**创建时间**：2026-03-01
**最后更新**：2026-03-01
**ID**：location_龙门
**状态**：Active

## 1. 基本信息 (Basic Information)

- **名称**：龙门
- **别名/称号**：东海海底通道、凡灵龙门
- **类型**：跨界通道
- **所属世界**：灵衍世界
- **所属界域**：凡界（东莱州·东海郡）—灵界（东海龙宫）；通道在东海海底
- **地理位置**：东莱州东海郡外东海海底；凡界侧为渡口与渔市，海底通灵界东海龙宫
- **面积**：通道节点；凡界侧渡口与渔市

## 2. 地理环境 (Geographical Environment)

- **地形地貌**：东海海底通道，凡界侧为东海渡口、渔市；灵界侧通往沧溟域东海龙宫（深海水晶城）。
- **气候特征**：凡界侧为东海沿岸；海底段为灵界规则，与龙门通道相连。
- **自然资源**：凡界侧渔盐、渡船；通道为渡界使与修士往来凡灵的重要节点之一。

## 3. 势力分布 (Faction Distribution)

- **凡界侧**：东莱州东海郡管辖，渡口与渔市；地脉九司渡界使掌握通道秘密，经此往来凡灵。
- **灵界侧**：东海龙宫（敖漓所在）；龙门通凡界东海，海雾、磁潮、海兽潮为风险。
- **交通**：凡界→灵界主要通道之一（与天门关、接天塔并列）；world_map §4.1：凡界→灵界 ② 龙门（东莱州东海—东海龙宫）。

## 4. 与天门关、接天塔的对比 (Comparison)

- **天门关**：朔方州，需通行令，通往灵界烛阴国。
- **龙门**：东莱州东海海底，通往灵界东海龙宫。
- **接天塔遗址**：中州京兆郡，旧通道已残，现为修真者驻凡办事处。

- **关联势力**：东海龙宫、渡界使、东莱州
