# 区域档案：黄泉路（阳间—阴界魂灵通道）

**创建时间**：2026-03-01
**最后更新**：2026-03-01
**ID**：location_黄泉路
**状态**：Active

## 1. 基本信息 (Basic Information)

- **名称**：黄泉路
- **别名/称号**：魂灵通道、阴阳路
- **类型**：通道/路径
- **所属世界**：灵衍世界
- **所属界域**：阴界（连接阳间与阴界）
- **地理位置**：连接阳间与阴界的魂灵通道；勾魂使带魂经此路入阴，终点为鬼门关
- **面积**：通道形态，无固定“长度”；魂灵感知为一段路途

## 2. 地理环境 (Geographical Environment)

- **地形地貌**：魂灵通道，两侧为阴气与魂能交织；亡魂经此路由阳间“沉”入阴界，常由勾魂使（黑白无常）押解或引导。
- **气候特征**：阴气渐浓，越往阴界深处越接近冥土规则。
- **职能**：亡魂受阴面法则牵引或由勾魂使接引后，经黄泉路抵达鬼门关，再入酆都。

## 3. 势力分布 (Faction Distribution)

- **勾魂使**：黑白无常等赴阳间接引亡魂，押解至阴界，经黄泉路入鬼门关。
- **阳间侧**：凡界各地有**黄泉岔口**，与城隍庙、乱葬岗、古战场等阴气节点相连；灵界/浊界亡魂由对应区域节点接引后，同样经黄泉路入阴。

## 4. 与阳间节点 (Yang-Side Nodes)

- **凡界**：城隍庙、乱葬岗、古战场等阴气汇聚处，常有黄泉岔口，勾魂使由此出入。
- **灵界/浊界**：无“地府分部”，勾魂使可跨界出差，接引后魂灵经阴面通道（黄泉路）入阴界。
