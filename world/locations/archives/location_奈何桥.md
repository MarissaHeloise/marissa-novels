# 区域档案：奈何桥（跨忘川之桥）

**创建时间**：2026-03-01
**最后更新**：2026-03-01
**ID**：location_奈何桥
**状态**：Active

## 1. 基本信息 (Basic Information)

- **名称**：奈何桥
- **别名/称号**：奈河桥、忘尘之桥
- **类型**：桥梁/关卡
- **所属世界**：灵衍世界
- **所属界域**：阴界
- **地理位置**：跨忘川的必经之桥，位于忘川之上；魂灵经审判、刑满或直判后，赴轮回前必经此桥
- **面积**：单桥节点，阴界尺度

## 2. 地理环境 (Geographical Environment)

- **地形地貌**：横跨忘川，桥下为阴界大河；魂灵过桥须饮孟婆汤，饮后忘尽前尘方可入轮回。
- **气候特征**：阴气与忘川水汽交织；为“忘尘”与“轮回”的最后一关。
- **职能**：跨忘川的必经之桥；**孟婆**在此施汤，掌“忘尘”一环。与转轮殿、轮回井/转生池为一套流程。

## 3. 势力分布 (Faction Distribution)

- **孟婆**：守奈何桥，施孟婆汤；修为深不可测，多视为“规则化身”或上古阴神。详见 `underworld_soul_system.md` §4.4。
- **地府**：奈何桥为轮回发放流程之一环，与十殿转轮殿、轮回井衔接。

## 4. 魂灵流程 (Soul Flow)

魂灵经十殿审判/受刑后，至转轮殿定来世去向，再经奈何桥：饮孟婆汤 → 忘尽前尘 → 入轮回井/转生池，转世至凡/灵/浊。
