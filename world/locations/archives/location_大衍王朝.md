# 区域档案：大衍王朝（中州·京畿）

**创建时间**：2026-02-26
**最后更新**：2026-03-01
**ID**：location_大衍王朝
**状态**：Active

## 1. 基本信息 (Basic Information)

- **名称**：大衍王朝
- **别名/称号**：凡界第一王朝、中州·京畿
- **类型**：皇朝
- **所属世界**：灵衍世界
- **所属界域**：凡界
- **地理位置**：凡界中部「中州（京畿）」区域
- **面积**：约500万平方公里（中州为核心，幅员辐射四州）

## 2. 地理环境 (Geographical Environment)

- **地形地貌**：以平原与缓丘为主，银汉河干流与多条支流自西北向东南流经；中州腹地自古为王朝兴替之所。
- **气候特征**：四季分明，雨热适中；适宜农耕与城镇密集分布。
- **自然资源**：粮棉、桑麻、基建石材为主，兼有低阶灵矿与药材；更多依赖他州与灵界贸易补充高阶资源。
- **生态系统**：河谷农耕带、丘陵林带与城镇圈层交织；适合凡人聚居，灵脉多为稀薄或被官方掌控的节点。

## 3. 势力分布 (Faction Distribution)

- **皇室与朝堂**：大衍皇室、宗室与文武百官；天京皇城与内城是权力核心。
- **地脉九司（中州总署）**：
  - **钦天监**：观测星象、推演气运。
  - **镇妖司凡界司**：负责凡界侧妖邪与凡灵交界事件，中州为调度中枢（总府常驻苍梧，见相应档案）。
  - **问心局、灵植司、铸币司、藏书楼、影卫等**：皆在天京设总署或主楼。
- **市井九流总舵/大分舵**：
  - **惊门·天机阁**：天京/天中城设总阁，掌凡界情报与机缘风向。
  - **爵门·纵横楼**：深植朝堂与官场的人脉网络。
  - **飘门·霓裳坊、疲门·百草堂**：在中州东市、西市均有体面门面。
- **地方军政**：各郡守、县令与边军内迁的老营部族。

## 4. 历史背景 (Historical Background)

**时间线对应**：详见 `world/settings/timeline.md`（近古·大衍立国、当世·景帝朝）。

大衍王朝在**上古**世界碎片稳定、**中古**凡界王朝多次更迭之后，于**近古**在凡界立国，是诸多王朝更迭后的“合流之朝”。  
中州原为旧朝遗址与诸侯割据之地，后来大衍以中州为根基，陆续纳入东莱、云梦、陇西、朔方诸州，成为凡界名义上的共主。**当世**为景帝在位，剧情时间锚点为**大衍景帝三十五年春**（坠星湖异象指引秘境开启、特别异象重宝出世，天命指引秘境可提升林微资质）。

- **前朝遗绪**：天中城一带保留了旧皇城格局，现为中立自由贸易区，各方势力在此博弈。
- **修真与凡权**：由于凡界灵气惰性，高阶修真者难以久居，中州的真实权力多掌握在皇室、地脉九司与市井九流的暗面之手。
- **对外关系**：通过天门关与灵界烛阴国保持有限通商与使节往来；对浊界则多依赖苍梧封印一线的情报。

## 5. 特殊资源 (Special Resources)

- **粮仓与铸币**：中州掌控大部分皇粮、军粮与官方铸币权（铸币司总坊在天京）。
- **气运节点**：天京、天中城、接天塔遗址一带有“气运交汇”之说，钦天监以此布命盘大阵。
- **隐秘灵脉**：少数由地脉九司与皇室掌控的灵脉节点，用于供给官方修士与镇压祟气。

## 6. 交通情况 (Transportation)

- **驿道**：京畿道为枢纽，向东连东莱州青阳城、向南通云梦州云州城、向西连陇西州西凉、向北通朔方州天门关。
- **水路**：银汉河干流贯穿天中、河内，是中州漕运命脉。
- **传送阵**：天京、天中城及各郡治均设有官方传送阵，需凭文书或令牌动用。

## 7. 五州与郡县 (Five Prefectures & Counties)

大衍实行**州—郡—县—镇/乡—村**五级政区。京畿所在为**中州**，余为四方边州；州设刺史，郡设郡守，县设县令，镇设镇守或巡检，村设里正。

### 7.2 中州（京畿）

- **京兆郡**：都城**天京**，皇宫、六部、钦天监、镇妖司凡界总部、影卫驻地；内城、东市、西市；天门关在此郡北缘。
- **天中郡**：**天中城**为旧都遗址，现为最大中立自由贸易区；河内郡为粮仓与银汉河漕运枢纽。
- **交通**：银汉河经京兆、天中、河内；驿道通东莱州青阳、云梦州云州、朔方州天门关、陇西州西凉。
- **经济**：科举与官僚中心、铸币司总坊、藏书楼总楼；市井九流天机阁、纵横楼、霓裳坊、百草堂等汇聚天京与天中城。
- **势力**：皇室、地脉九司（钦天监/镇妖司/问心局/灵植司/铸币司/藏书楼/影卫）、市井九流多门总舵或分舵。

### 7.3 东莱州

- **青阳郡**：**青阳城**为郡治，林府、林家炼器坊、青阳城守卫；城东**坠星镇**临坠星湖，秘境每十年开启。
- **东海郡**：海盐、临波等县，近扶桑海；**龙门**通道在东海海底（灵界侧），凡界为渡口与渔市。
- **坠星郡**：以坠星湖及周边县镇为主，坠星秘境入口、灵韵异动多发，问心局与镇妖司有派驻。
- **交通**：驿道连中州与东海渡口；银汉河支流经青阳郡。
- **资源**：低阶灵矿、药材、渔盐；青阳城炼器与林家闻名。
- **势力**：林家、青阳城守卫、市井九流（天机阁、百草堂分舵），血煞盟在云州但东莱州有眼线。

### 7.4 云梦州

- **云梦郡**：**云州城**为郡治，云州商会、万货帮（要门）控制经济命脉；江陵等县为水陆码头。
- **银江郡**：沿银汉河干流，漕运与商贸发达。
- **南疆郡**：**迷雾林**在郡南，常人入内必死，为凡灵交界险地；苗寨与边民村落散布。
- **交通**：银汉河干流纵贯；驿道通中州、东莱州；茶马道通南疆。
- **资源**：茶叶、药材、木材、码头商贸；云州商会垄断多条商路。
- **势力**：云州商会、万货帮、市井九流要门/册门/风门等；血煞盟与纵横楼有脉络。

### 7.5 陇西州

- **陇西郡、沙洲郡**：西凉城、敦煌县、玉门县；毗邻**陇沙荒漠**，流沙下埋上古古国。
- **交通**：边市、驿道、戍堡线；与中州驿道相连。
- **资源**：玉石、沙金、荒漠边缘低产粮；边贸与戍军为主。
- **势力**：边军、守陵人活动于古国遗址外围；市井九流册门（盗墓销赃）偶现。

### 7.6 朔方州

- **北塞郡、烛阴郡**：**天门关**为凡界—灵界通道，需通行令；关内城、军镇、马市。
- **交通**：驿道直通中州天京；天门关外为灵界凛渊域（烛阴国）。
- **资源**：马匹、皮货、边军粮饷；渡界使与镇妖司前哨人员经此往来。
- **势力**：边军、镇妖司关口巡检、渡界使；地脉九司与灵界交接要地。

## 8. 边疆禁区与水域 (Border Forbidden Zones & Waters)

### 8.1 边疆禁区 (Border Forbidden Zones)

| 场景 | 所属州/郡 | 性质 |
| :--- | :--- | :--- |
| **陇沙荒漠** | 陇西州/边疆 | 大陆以西，流沙下埋上古古国；凡人慎入 |
| **碧霄云海** | 上空特殊空域 | 空贼与飞行妖兽出没；凡人慎入 |
| **幽冥裂谷** | 地下极深 | 直通浊界边缘，魔气偶有渗出；凡人慎入 |
| **皇陵、上古遗迹** | 中州/东莱州等 | 守陵人守护，防古尸异变 |

- **陇沙荒漠 (Longsha Desert)**：大陆以西，流沙下埋上古古国。
- **碧霄云海 (Bixiao Cloud Sea)**：上空特殊空域，空贼与飞行妖兽出没。
- **幽冥裂谷 (Nether Rift)**：地下极深，直通浊界边缘，魔气偶有渗出。

### 8.2 水域 (Waters)

| 水域 | 说明 |
| :--- | :--- |
| **银汉河** | 横贯大陆主干河，经大衍南部汇入东海方向；经京兆、天中、河内、云梦、银江；漕运主道，渡口镇、云州码头镇为要埠；支流经东莱州青阳郡，连坠星湖与东海方向。 |
| **扶桑海** | 东方海域，多风暴；龙门通道在东海海底（灵界侧），凡界侧为渔港与渡船。 |

### 8.3 凡界→灵界/浊界 通道 (Barrier Crossings)

- **天门关**：朔方州烛阴郡，天京北向驿道终点，通往灵界烛阴国，需通行令。
- **龙门**：东莱州东海郡外东海海底通道，通往灵界东海龙宫。
- **迷雾林**：云梦州南疆郡南，常人入内必死，凡灵交界险地。
- **接天塔遗址**：中州京兆郡，旧通道已残，现为修真者驻凡办事处。

## 9. 重要地点 (Important Locations)

- **天京**：大衍都城，皇宫、六部、地脉九司中枢所在。详见 `location_天京.md`。
- **天中城**：中立自由城，旧皇城遗址，市井九流与商会势力盘踞。
- **接天塔遗址**：古通道残迹，现为修真者驻凡办事处。
- **天门镇**：天门关下小镇，朔方州与中州交界的咽喉。天门关详见 `location_天门关.md`。
- **青阳城、坠星湖、云州城**：各有独立地点档案；坠星秘境见 `location_坠星秘境.md`，迷雾林见 `location_迷雾林.md`。

## 10. 居民情况 (Residents)

- **主要种族**：人类
- **人口数量**：约5000万（含中州城镇与广袤乡村）；全境约2亿+（大衍王朝总人口）
- **生活方式**：农耕、商贸、手工、少量修炼。
- **文化特色**：科举与官场文化浓厚，表面上尊文治，暗地里也对“强者”极为敬畏；市井与朝堂皆有九流杂处的影子。

## 11. 区域特色 (Regional Characteristics)

- **政治中心**：凡界名义上的最高权力中枢。
- **多势力交汇**：皇室、地脉九司、市井九流、外来修真者在此“共舞”。
- **对外枢纽**：通过朔方州天门关与灵界沟通，是凡界了解灵界的最前线之一。

## 12. 备注 (Notes)

大衍王朝中州·京畿既是林微踏入灵衍世界后“仰望”的中心，也是她后续不得不正面面对的权力漩涡。五州、边疆、水域及郡县制详见 `world/settings/world_map.md` §2.1。
