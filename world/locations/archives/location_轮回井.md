# 区域档案：轮回井（转生池）

**创建时间**：2026-03-01
**最后更新**：2026-03-01
**ID**：location_轮回井
**状态**：Active

## 1. 基本信息 (Basic Information)

- **名称**：轮回井
- **别名/称号**：转生池、轮回池口
- **类型**：设施/节点
- **所属世界**：灵衍世界
- **所属界域**：阴界
- **地理位置**：十殿转轮殿之后，魂灵饮过孟婆汤后由此投入轮回；与奈何桥、转轮殿为一套流程
- **面积**：池口/井口节点，阴界尺度；池口对应凡/灵/浊及胎卵湿化等去向

## 2. 地理环境 (Geographical Environment)

- **地形地貌**：轮回井或转生池，池口依业力与因果分配至不同去向（凡界、灵界、浊界及胎卵湿化等）；魂灵饮孟婆汤忘尽前尘后，由此投入轮回。
- **气候特征**：阴界规则与轮回法则交汇处；魂灵经此“转世”至阳间。
- **职能**：通过审判、饮过孟婆汤的魂灵由此投入轮回，转世至凡/灵/浊；**转轮王**掌“最后一签”，依业力与因果分配池口。

## 3. 魂灵流程 (Soul Flow)

十殿转轮殿定来世去向 → 奈何桥饮孟婆汤 → **轮回井/转生池**投胎。来世投胎至凡界、灵界还是浊界，由地府依业力、因果与轮回井的“池口”规则决定，并非凡界死者只能投凡界。

## 4. 与三界的关系 (Relation to Three Realms)

- 凡界、灵界、浊界众生死后灵魂皆可入地府轮回（除非魂飞魄散）；轮回井为共同出口，池口对应三界及胎卵湿化等。
- **身死道消**仅指肉身与修为；灵魂归宿以此井（及前序流程）为准。
