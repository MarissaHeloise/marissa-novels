# 地点档案 (Location Archives)

## 命名规则 (Naming Convention)

**文件名**：`location_` + **地点名**（与档案内「名称」一致）

- 格式：`location_<地点名>.md`
- 示例：`location_青阳城.md`、`location_坠星湖.md`、`location_灵界七域总览.md`
- 档案内 **ID** 与文件名保持一致：`**ID**：location_<地点名>`

**以后新建地点档案均按此规则命名。**
