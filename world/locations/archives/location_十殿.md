# 区域档案：十殿（酆都十殿）

**最后更新**：2026-03-01

## 1. 基本信息 (Basic Information)

- **名称**：十殿
- **别名/称号**：酆都十殿、地府十殿
- **类型**：机构/建筑群（十座大殿）
- **所属世界**：灵衍世界
- **所属界域**：阴界·酆都
- **地理位置**：酆都主城，十座大殿依序排列；魂灵入阴后依业力分派至各殿受审或受刑
- **面积**：阴界尺度，十殿合计为酆都核心

## 2. 十殿职责总表 (Ten Courts Overview)

魂灵入阴界后，依生前业力与死因分派至不同殿；十殿顺序流转，并非每魂都走遍十殿。详细职责与刑罚见 `underworld_soul_system.md` §4.3。

| 殿序 | 殿名 | 职责概要 | 备注 |
| :--- | :--- | :--- | :--- |
| **一殿** | 秦广殿 | 初审与分流；孽镜照前业；无罪或功德高者可直接送转轮殿或还阳；有罪者发二至九殿 | 孽镜台在此；核验魂飞魄散者是否仍存 |
| **二殿** | 楚江殿 | 寒冰地狱；惩偷盗、拐卖、伤人、谋财害命等 | 伤生害命类业报 |
| **三殿** | 宋帝殿 | 黑绳地狱；惩忤逆、挑拨、诬陷等 | 绳缚刀锯，刑满流转 |
| **四殿** | 五官殿 | 血池/血脓地狱；惩滥杀、虐杀、见死不救等 | 与“血”相关业障 |
| **五殿** | 阎罗殿 | 望乡台所在；阎罗王亲审大案（帝王、大修、跨界重犯）；定刑期与来世去向 | 慕清歌等经此殿；望乡台在殿外 |
| **六殿** | 卞城殿 | 枉死城；收容横死、自杀、冤死未超度之魂，暂居待因果了结或超度后再判 | 沈梦蝶类冤魂可入 |
| **七殿** | 泰山殿 | 热恼/蒸笼地狱；惩口业、毁誉、邪淫等 | 刑满流转 |
| **八殿** | 都市殿 | 大热恼地狱；惩不孝、不义、虐亲、欺师等 | 重者与九殿衔接 |
| **九殿** | 平等殿 | 阿鼻/无间；极恶之魂长刑或永世不得轮回 | 噬魂殿、幽冥教罪大恶极者多判入此殿 |
| **十殿** | 转轮殿 | 轮回发放；各殿刑满或直判之魂汇总，依业力与因果分配轮回井池口（凡/灵/浊、胎卵湿化等），饮孟婆汤后投入轮回 | 转轮王掌“最后一签”；与奈何桥、转生池为一套流程 |

## 3. 魂灵流程 (Soul Flow)

一殿初审 → 有罪者二至九殿受刑（或入枉死城）→ 刑满/直判 → 十殿转轮殿定来世 → 奈何桥饮孟婆汤 → 轮回井/转生池投胎。

## 4. 重要地点 (Important Locations)

- **孽镜台**：一殿秦广殿。
- **望乡台**：五殿阎罗殿外。
- **枉死城**：六殿卞城殿所辖。
- **轮回井/转生池**：十殿之后，见 `location_轮回井.md`。
