## 第五阶段：坠星湖相遇与慕清歌互动（第 32–40 章）

| 章 | 标题 | 内容要点 |
|:---|:---|:---|
| **32** | 一枝独秀 | 某场小比/答题/展示中林微表现突出但与权贵格格不入；慕清歌暗中留意、未表露身份。**破功伏笔**：慕清歌对身边人一瞥或轻咳即噤声退下，林微若注意到可感违和，被温婉笑容盖过。 |
| **33** | 暗中相助 | 林微遇小麻烦（刁难、克扣、陷害）；慕清歌通过侍女/心腹暗中化解；林微有所察觉但不知是谁，对「贵人」好奇。路人提及「像是宫里的人」，为 Ch34 揭晓铺垫。 |
| **34** | 长公主之邀 | 慕清歌以长公主身份正式与林微见面（茶会/游园或私下面谈）；**揭晓**相助者即慕清歌。慕清歌温婉全开：「在宫里没有能说真心话的人」「只有你不把我当长公主看」；把相助说成「顺手」「不忍你被欺负」。 |
| **35** | 秘境名额 | 秘境名额/分组中慕清歌暗中帮林微获有利位置或队友；话术让林微觉得「自己运气好」或「长公主只是公正」，绑住感激与依赖。 |
| **36** | 并肩而行 | 某次共同行动（巡湖、祭典、应对突发）；慕清歌不经意肢体距离（袖口相触、并肩看湖）、「若在秘境中走散，我会找你」等，暧昧与占有欲伏笔。 |
| **37** | 试探与分寸 | 慕清歌绵里藏针试探林微来历、志向、对皇家态度；林微谨慎应答。慕清歌欣赏不攀附、有主见，内心更想「留在身边」。 |
| **38** | 信物之始 | 慕清歌赠林微玉佩/香囊/令牌（「秘境遇险可求援」「只给你一人」）；信物可带皇家/紫霄印记，为卷二埋笔。 |
| **39** | 入秘境前夜 | 最后一夜告别或约定「秘境中互相照应」；慕清歌入秘境动机轻描淡写（「历练」「想看看外面的世界」）；血煞盟等夺宝传闻。 |
| **40** | 秘境之门 | 秘境开启，众人进入；林微与慕清歌入口分散或同组；天命指引偃师传承、万象归元诀；慕清歌有影卫/心腹，进入后分散行动。 |

## 第六阶段：秘境·四线交汇（第 41–50 章）

| 章 | 标题 | 内容要点 |
|:---|:---|:---|
| **41** | 秘境初探 | 修为压制练气大圆满；机关/妖兽险些致命，万华解析破局；血煞盟夺宝线初现。林微劲装。 |
| **42** | 叶红绡登场 | 镇妖司·家族复仇线；与林微共同对敌，高冷剑修，对林微初步认可；家族被屠之仇与镇妖司身份暂未全露。 |
| **43** | 血煞盟袭杀 | 血煞盟袭杀/夺宝；林微与叶红绡联手，建立信任；叶红绡透露寻仇/历练。 |
| **44** | 青丘小狐 | 白涂涂（练气五层）登场；青丘小狐落难或贪玩被捉，收服/结缘；幻术、狐族遁术。 |
| **45** | 涂涂认主 | 白涂涂认主或结缘，约定「出秘境后跟你混」；血煞盟线推进。 |
| **46** | 慕清歌遇险 | 慕清歌被血煞盟/妖兽/机关所困，重伤或护身玉碎；林微感应信物或听闻消息、决意相救。慕清歌落难/战损外貌。 |
| **47** | 独处相救 | 林微赶到独处相救；慕清歌示弱「只有你在我才安心」；暧昧升温；偶露破功（对影卫「退下」等）。 |
| **48** | 四人汇合 | 林微、慕清歌、叶红绡、白涂涂汇合；血煞盟夺宝线高潮，多方混战；四人战斗分工明确。 |
| **49** | 共渡难关 | 团队配合巩固，击退血煞盟或暂退；夺宝线阶段收束，传承线接上。 |
| **50** | 传承在望 | 万象镜指向偃师传承与万象归元诀；林微带队或独行前往传承地。 |

## 第七阶段：传承·突破·暂别（第 51–75 章）

| 章 | 标题 | 内容要点 |
|:---|:---|:---|
| **51** | 偃师遗迹 | 进入偃师传承遗迹；机关、考验与选择；万象镜天命牵引通过初关。 |
| **52** | 情毒·交欢数日 | **情毒事件**：妖兽/妖植情毒，独处密室/洞窟，从抗拒到情不自禁、交欢数日。**床戏双版本**（PO18/过审），唯美又很黄。 |
| **53** | 事后·尴尬暧昧 | 情毒解除后双方尴尬；林微一直躲慕清歌；慕清歌试探、跟随、保护；获得万象归元诀或偃师认可。拉扯感满、暧昧极致。 |
| **54** | 传承加身 | 正式接受偃师传承；林微向练气五层迈进（三层或四层），慕清歌秘境奇遇后练气四层；继续躲与暧昧。 |
| **55** | 灵根之变 | 资质提升线索；林微练气四层→五层过渡；慕清歌影卫跟随、遇险现身「你就这么不想见我？」 |
| **56** | 练气五层 | 林微突破练气五层，慕清歌稳固练气四层；出秘境修为节点。 |
| **57** | 血煞盟收束 | 血煞盟余孽击退/遁走，夺宝线收束。 |
| **58** | 慕清歌信物 | 留信物「天京再见」；林微谎称去云州时慕清歌点破、林府废墟信物、「无家可归跟我回天京」。 |
| **59** | 叶红绡之约 | 约定日后联系/同行；家族被屠与镇妖司身份已显露，敖漓线铺垫。 |
| **60** | 白涂涂随行 | 白涂涂决定随林微出秘境，青丘灵宠线卷一收束。 |
| **61** | 秘境将闭 | 众人寻路离开；出口处小冲突。 |
| **62** | 出秘境 | 回到坠星湖；林微练气五层、慕清歌练气四层，偃师传承与万象归元诀在身。 |
| **63** | 慕清歌回京 | 慕清歌启程回天京，车驾威仪；与林微告别；命运联系伏笔；镜中宫阙/人影可呼应。 |
| **64** | 镜中人微光 | 镜中人与林微微弱联系；「命运」「羁绊」类提示。 |
| **65** | 归途启程 | 与白涂涂（及可选叶红绡）踏上归途；无林家可回，秋蝉执念。 |
| **66** | 归途见闻 | 凡界武者/修者/九流回顾；心态从求存到初成。 |

| **68** | 求见秋蝉·慕清歌推脱 | 林微求慕清歌让自己见秋蝉；慕清歌假装非己所为、推脱，骗林微入京（「到天京才能见」）。 |
| **69** | 入京·秋蝉真相 | 林微入京；秋蝉已成皇室傀儡（秘药控制、练气五层）；林微如遭雷击，察觉慕清歌与秋蝉被夺的关联。 |
| **70** | 识破·欲逃 | 林微识破慕清歌计谋，欲偷偷离开；慕清歌挽留/阻拦，拉扯到高潮。 |

| **72** | 软禁寝宫 | 慕清歌将林微软禁在长公主府寝宫；林微练气一层、无力反抗；囚禁与对峙。 |
| **73** | 质问·「因为太爱」 | 林微质问采补/软禁/秋蝉/骗入京；慕清歌回应「都是因为太爱她」；卷一慕清歌线收束于扭曲之爱与囚禁。 |
| **74** | 寝宫·床戏（一） | **全章床戏**：软禁期间寝宫交欢（强制/半强制或药效/虚弱）；双版本；占有、执念与身体纠葛；无收束/盘点。 |
| **75** | 寝宫·床戏（二） | **全章床戏**：延续或另一情境交欢；双版本；卷终于身体与情感极致纠缠；无收束/盘点。 |
