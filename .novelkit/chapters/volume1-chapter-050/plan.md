## 剧情提要（本章目标：按正文口径同步要点）

本章已按正文定稿，不再采用“驻点两月日常切片”的写法，而以**十年驻守概述**推动时间与成长：林微在沙洲郡清剿灵傀与邪修，天命点攒满**十万**后兑换**混沌灵根**；灵蚀界灵机与浊气暴涨引发天材地宝现世，林微借系统指引夺灵药灵材，修为顺利至**筑基中期**；并将**栖云偃庐**改造炼化为**栖云飞舟**（灵石驱动法阵，更迅捷省灵石）。

人际线落点：林微结识驻守修士中修为最出众者**莫惊尘（筑基大圆满，天枢剑宗天骄）**，在作坊中以“**逐风升级**”为媒相识，并以对酌谈天逐渐成为知己。

后段战局转折：十年期满二人同赴灵蚀界主战场**沙洲郡**，营地庆功时浊界暗布的**大型邪阵**启动，浊气喷涌、低阶修士傀化混乱，二人背靠背杀出重围，撤离陇西地界后因浊侵难清，转赴**百草谷**求医；章末以林微念及**慕清歌**作余韵。

## 情节纲目（与正文对齐）

1. **十年驻守概述**：清剿灵傀/邪修 → 天命点攒满十万。  
2. **混沌灵根兑换与突破**：灵机催化机缘 → 系统指引夺灵药灵材 → 筑基中期。  
3. **器修里程碑**：栖云偃庐改造炼化为栖云飞舟（灵石驱动阵法）。  
4. **作坊初识**：莫惊尘托“逐风”升级 → 定价交易 → 互换灵网联络。  
5. **对酌成知己**：莫惊尘酿酒相赠 → 往来对饮、论道切磋。  
6. **主战场转折**：沙洲郡庆功设宴 → 地下邪阵启动、浊气喷涌、低阶傀化 → 二人背靠背杀出重围。  
7. **收束去向**：撤离陇西地界 → 浊侵难清 → 奔赴百草谷求医；章末念及慕清歌。

## 人物与场景（按正文）

- **林微**：主视角，十年驻守后攒满天命点换根，至筑基中期；改造栖云飞舟。  
- **莫惊尘**：筑基大圆满，天枢剑宗天骄；佩剑逐风；洒脱爽朗、剑意逍遥。  
- **管事/同僚**：作坊与关卡群像一笔即可，用于衬出“她来得随性、分量却不轻”。  
- **场景**：沙洲郡驻守与秘境险峰、作坊相识与对酌、沙洲郡主战场营地与邪阵爆发、撤离奔赴百草谷的路上。
