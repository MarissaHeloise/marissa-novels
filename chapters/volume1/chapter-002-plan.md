# 第二章 凡人初修·计划

**第一卷·凡尘镜影**

## 章节主题
林微通过一年的私塾和武馆课程，达到武者一层水平，了解修仙世界基本设定，尝试引气入体失败。

## 核心内容
1. **私塾学习**：学字、历史、修仙基本设定
2. **武馆训练**：熬练筋骨、体力消耗
3. **系统互动**：尝试引气入体失败
4. **世界设定**：凡人界修炼限制、灵气稀薄

## 详细规划

### 场景1：私塾课堂（上午）
- **时间**：穿越后一年，清晨
- **地点**：林府私塾
- **人物**：林微、私塾先生（周先生）、其他学童
- **内容**：
  - 学字：《千字文》《三字经》
  - 历史：《大衍王朝史》，了解大衍王朝的建立、历代皇帝、疆域范围
  - 凡人域设定：《凡人域志》，了解凡人域的地理分布、主要城市、势力分布
  - 修仙基本设定：《修仙入门》，了解灵根、修为等级、修炼方法
- **细节**：
  - 林微年纪小，坐姿端正，认真听讲
  - 周先生讲解详细，举例生动
  - 其他学童有的认真，有的调皮
  - 林微通过系统万象识别看到周先生的信息：【周先生，天赋：凡人，境界：凡人】

### 场景2：武馆训练（下午）
- **时间**：下午
- **地点**：林府武馆
- **人物**：林微、武馆教头（王教头）、其他弟子
- **内容**：
  - 基础训练：扎马步、踢腿、出拳
  - 熬练筋骨：负重跑、俯卧撑、仰卧起坐
  - 招式练习：基础拳法、剑法
- **细节**：
  - 林微体力消耗大，汗流浃背
  - 王教头严格要求，耐心指导
  - 其他弟子有的嘲笑林微年纪小，有的鼓励她
  - 林微坚持训练，不轻易放弃
  - 系统提示：【检测到宿主正在进行基础训练，体质+1】

### 场景3：晚间休息（晚上）
- **时间**：晚上
- **地点**：林微卧房
- **人物**：林微、秋蝉
- **内容**：
  - 秋蝉为林微擦拭身体，关心她的训练情况
  - 林微向秋蝉讲述一天的学习和训练
  - 尝试利用系统给的功法引气入体
  - 引气入体失败，系统提示：【引气入体失败，建议加强基础训练】
- **细节**：
  - 林微疲惫但坚持
  - 秋蝉心疼但支持
  - 系统界面显示：【系统任务：觉醒】
  - 【任务描述：引气入体，成为练气一层修士】
  - 【任务奖励：灵根资质提升】

### 场景4：与父母交流（次日）
- **时间**：次日早晨
- **地点**：林府客厅
- **人物**：林微、林远山、苏婉
- **内容**：
  - 林微向父母汇报学习和训练情况
  - 父母关心她的身体状况
  - 林父讲述凡人界修炼的限制
  - 林母鼓励她坚持努力
- **细节**：
  - 林父："凡人界灵气稀薄，修炼进度缓慢，你要做好长期坚持的准备"
  - 林母："微儿，不管结果如何，你都是娘的骄傲"
  - 林微："爹爹娘亲放心，我一定会坚持下去的"

## 世界设定引入
1. **凡人界限制**：
   - 最多只能修炼到炼气大圆满
   - 灵气稀薄，修炼进度缓慢
   - 除非有特殊机缘，否则天灵根才有可能在50岁前到达练气大圆满
   - 练气大圆满修士的寿命只有150岁

2. **修仙基本设定**：
   - 灵根：金、木、水、火、土五种基本灵根，还有变异灵根和伪灵根
   - 修为等级：凡人→武者（1-9层）→练气（1-10层）→筑基→金丹→元婴→化神
   - 修炼方法：引气入体、炼气化神、炼神返虚

3. **大衍王朝**：
   - 建立于500年前，现任皇帝为大衍景帝
   - 疆域包括青阳城在内的36郡
   - 与周边国家保持和平，但时有边界冲突

4. **凡人域**：
   - 凡人域是灵衍世界的一部分，主要居住凡人
   - 有少量低阶修士
   - 与灵界、浊界并称为三界

## 章节风格
- **语言风格**：古典与现代结合，保持修仙世界的古朴感
- **节奏控制**：中速，注重细节描写和人物成长
- **情感表达**：细腻真实，突出林微的努力和坚持
- **场景描写**：注重环境氛围的营造，增强代入感

## 章节结构
1. **开头**：私塾课堂场景，介绍学习内容
2. **发展**：武馆训练场景，展示训练的辛苦
3. **高潮**：尝试引气入体失败，系统提示
4. **结尾**：与父母交流，坚定修炼决心
