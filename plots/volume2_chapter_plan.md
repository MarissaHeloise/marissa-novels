## 二、第二卷第1章 = 全球第54章（详规）

**建议章题（三选一或组合）**：**玄枢复命·偃月争锋** / **浩劫令至·亲传名册** / **归宗·大比·拜师**

**正文路径（择一）**：`chapters/volume1/chapter-054.md` 或 `chapters/volume2/chapter-001.md`

**字数目标**：宪法常规区间 **≥2500 字**（去空白计宜 **2500–3500**）；若复命与大比都想写细，可拆为两章（54 复命+动员，55 大比+亲传），则本文件「第1章」拆成 54–55，下表顺延。

### 时间锚点

- **紧接第53章**：林微抵**幽机域**本山（或经由中转传送点），距「离宗十五载」文书与沙洲一线溃败**同一叙事连续段**，不宜再跳「又去百草谷求医」（该线已在第50–51章完成）。

### 本章必须落地的 ≥4 件事

1. **浊气浩劫压顶（卷氛围）**  
   - 用**公议牍、榜文、执事口头、工坊停工令、传送阵戒严**等至少两种载体，写出「前线不是小败，而是邪阵失控级灾难」已传到本山。  
   - 林微带回的情报要有**可被追问的细节**（阵式异变、傀化速率、缺口方位择一二即可），避免一句「很糟」带过。

2. **复命链完整**  
   - 走通：**验牒/腰牌 → 执事堂或功勋司 → 更高层旁听**（长老、峰主代表或副峰主择一露面即可）。  
   - 落地结果：**功过并书**（十五年外驻、筑基中期、混沌灵根不宜当众嚷嚷但高层可心知肚明）、**短期安置**（洞府/工坊权限是否暂封或升级）、**下一步强制义务**（与浩劫动员挂钩）。

3. **内门大比（规则自洽）**  
   - **动机**：浩劫前夕宗门要**选拔可带队/可守要隘的骨干**，或传统「季比/年比」恰逢动员升级；记名与内门匠籍同台，优胜者得**亲传提名**或**拜师资格**。  
   - **写法**：至少一轮**器/偃**可见的较艺（机关拆解、偃阵抗压、限时炼修、灵傀对抗等择其二），有**具体对手或题目**，不是擂台一笔。  
   - **林微**：胜式对齐其人设——**偃术托底、炼器思路破局**，五灵根短板用准备与算计补。

4. **晋升为亲传弟子**  
   - **路径**：大比名次或峰主当场点将 → **匠籍殿·师承笔**备案 → **弦月仙子**（凌弦月）行拜师礼或简礼（规模可因浩劫从简，但**名分必须落地**）。  
   - **代价与钩子**：亲传身份附带**秘境优先权**与**派系任务**，章末接到与**浊线/前线**相关的首道硬指令，或见到**照雪仙子**、**柳听潮**等旧线人物递新线头。

### 硬约束

- **莫惊尘**：已回天枢剑宗，本章以**传讯、侧闻、或末尾互致一言**点一下即可，不必占主篇幅。  
- **慕清歌**：不写同框；若出现，仅限**灵网一闪、未读、或半句留言**作情感余温。  
- **百草谷**：不作本章主场景；若提及，应是林微**一闪念的牵挂**或他人闲话，勿重复第54章旧纲「再赴谷求医」。

### 章末钩子（择一或组合）

- 浩劫**征发令**下到林微新领的**亲传任务**上；或  
- 公议牍出现**与沙洲同式邪阵**在另一域冒头的急报；或  
- 灵网上慕清歌一条**欲言又止**的讯息与林微**暂时无法回复**的错位。

## 六、第57–60章终局详规（全书收束）

### 第57章：藤妖暴走·金丹成魔

- **触发**：战场浊气（或异种阵纹反噬）诱发慕清歌体内**藤妖/半妖**彻底失控；神志被浊淹没。  
- **行径**：化非人藤躯，**吸食修士气血**以战养战；**滥杀无辜**（需写出具体惨相一组，勿虚化）。  
- **境界**：在暴走中强行**凝结金丹**（可用浊丹/伪丹命名，表现为金丹级战力与劫数异象）。  
- **群体反应**：联军从「救护」转为「除害」；舆论与军令双重压力。  
- **林微**：章末落地**被迫参与围攻**（师承、盟誓、护未撤离者的择一必须写清）。  
- **章末钩**：合围阵成 / 诛魔令下 / 林微掌心出汗仍掣出本命偃（择一）。

### 第58章：众矢之的·怀中尘灭

- **围攻战**：多宗手段尽出；慕清歌仍凶，终究**重伤将殒**。  
- **林微**：在关键时刻给予**无法回避的一击**（或破丹、断根、贯心——正文自定，须不可逆）。  
- **回光**：濒死前**神志清明**，在林微臂弯中**遗言**：想回到**初识**；若重来必**尊重林微、不强留**；事已无可挽回；祝她**自由、快乐、实现自己的愿望**。  
- **死亡**：身躯**灰飞**；唯留**一枚种子**（意象物，不必在后文回收）。  
- **林微**：**失去后才识本心**——心中有慕清歌，**永别无期**。  
- **章末钩**：空白的一声嗡鸣 / 灵网未发送的半句话 / 指缝流沙感（择一，压情绪）。

### 第59章：无情证道·化神荡浊

- **转向**：慕清歌死后，林微**自封情丝**，走**无情道**（心法名可自拟，与精神「空心执行」一致即可）。  
- **叙述**：**大时间跨越**——叠映短场景直至**化神**。  
- **世界线**：亲率或直接主导**荡平浊界**核心势力，**灵浊浩劫**宣告平息（一场决战为轴）。  
- **基调**：功业至矣而**人如空壳**；忌写成嗜杀狂喜。  
- **章末钩**：万象/system UI **异常闪烁** / 镜中无自身倒影（引天道出场）。

### 第60章：天道送归·黄粱人间（全书完）

- **天道**：于**万象系统**内觉醒，告知穿越**实为救世之棋**；今**平衡已立**，因果闭合。  
- **送归**：**瞬时剥离**修为、身份、此界羁绊；林微**不及反应**已被打出识海。  
- **现世**：仍坐**公司工位**，**加班**夜；屏幕冷光、咖啡凉、键盘声——**一切修真经历如梦**。  
- **收束**：权柄、修为、情爱皆**黄粱**；可余半息恍惚（ optional ）。  
- **全书完**：勿再接「二次穿越」尾巴，除非另开新书。
