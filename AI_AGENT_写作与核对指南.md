### 1.2 资源查找顺序建议

- **写/改某一地点**：`world_map.md` 定州/郡/域 → 若有独立档案则查 `location_<名>.md`。
- **写/改某一势力或阵营**：`factions.md` 定界域与称谓 → `faction_<名>.md` 与 `faction_relationships_and_conflicts.md` 定关系与冲突。
- **写/改修为、战力、寿命**：`power_systems.md`；凡界/灵界/浊界上限见 `worldview.md`。
- **写/改时间或年代**：`timeline.md`；剧情跨度见 `worldview.md` §9。
- **写/改通道与跨界**：`worldview.md` 屏障通道 + `world_map.md` 重要节点表 + `spirit_realms_seven_domains.md` 对应域（如凛渊域·烛阴国）。
- **写/改功法、突破机缘**：`power_systems.md` 定境界 → `功法套路全览.md` 选路线与功法；`境界机缘备选·凡灵妖.md` 或 `境界机缘备选·浊邪阴.md` 选道基/金丹/机缘；通道与地点名须与 `world_map.md` 一致（天门关为凡界—灵界，灵浊关口为神农封印/极北裂缝）。
- **写/改法宝、装备**：`法宝套路全览.md` 按路线与境界选法宝；等级与装备栏见该文件总表。
- **写/改修士类型、体质**：`修士类型全分类.md` 定类型与势力；`灵根与体质全分类.md` 定灵根/体质。

## 二、写作规则（必守）

**引入新人物时**，必须按以下顺序执行，不得跳过：

1. **先生成人物档案**：在 `world/characters/` 下新建 `character_<中文名>.md`，按既有人物档案格式填写基本信息、定位、性格、背景、人际关系、与世界观/势力/地理的对应关系等。
2. **核对设定**：确保该人物所属势力、地域、境界、功法、称谓等与 `worldview.md`、`factions.md`、`power_systems.md`、`world_map.md` 等权威文档一致，无错误、不违和。
3. **更新项目人物信息**：将该人物加入 `world/characters/key_characters.md`（若为主要角色）或 `world/characters/secondary_characters.md`（若为次要角色/NPC）的索引或列表，并同步更新与之相关的其他人物档案中的「人际关系」、势力档案中的「已知/核心人物」等引用。

未完成以上三步前，不得在正文中正式引入该新人物（可先以代号或伏笔出现，待档案与索引就绪后再正式出场）。

## 四、写作与核对清单

### 4.1 地点与地理

- [ ] **州/郡/域归属**：提到的城镇、湖泊、关口是否与 `world_map.md` 一致？  
  - 坠星湖 → **东莱州·坠星郡**，青阳城东；**不是**中州。  
  - 天门关 → **朔方州·烛阴郡**；天京仅为驿道南端，**天门关不在京兆郡**。  
  - 接天塔遗址 → 中州·京兆郡；已残，非主通道。  
  - 龙门 → 东莱州东海，海底通灵界东海龙宫；迷雾林 → 云梦州南疆郡。
- [ ] **界域**：场景在凡界/灵界/浊界/阴界？凡人无阵法庇护不能在灵界灵潮中久留；灵界修士入凡界被压制至练气大圆满。
- [ ] **通道**：跨界时走的哪个通道？凡界→灵界：天门关（朔方州—烛阴国）、龙门（东海—龙宫）、接天塔（已残）、或突破筑基后空间节点。灵界→浊界：苍梧神农封印、凛渊域极北裂缝或元婴级强渡。

### 4.2 人物与势力

- [ ] **新人物必先建档案**：引入新人物前须完成 §二 写作规则——先生成 `character_<中文名>.md`、核对设定无违和、更新 key_characters 或 secondary_characters 及相关人物/势力档案中的引用。
- [ ] **角色所属势力/地域**：与 `worldview.md` 主角表、`factions.md`、`faction_*.md` 一致。  
  - 林微：凡界东莱州青阳城起点；慕清歌：天京/大衍皇室；叶红绡/顾长歌：凛渊域镇妖司前哨；莫离：赤烬域天枢剑宗；敖漓：沧溟域东海龙宫；白涂涂：幻花域青丘；苏青黛：苍梧百草谷；夜琉璃：浊界魔罗殿；苏慕晚：九霄域凌霄符殿。
- [ ] **势力称谓**：玄门九宗用「法修/剑修/符修…」与 `factions.md`、`power_systems.md` 一致；紫霄法脉为**法修**。
- [ ] **势力关系**：两方合作或敌对时，对照 `faction_relationships_and_conflicts.md`，避免出现设定中的死敌联手、或同盟互杀且无剧情解释。

### 4.3 力量规则与修为

- [ ] **境界与寿命**：以 `power_systems.md` 为准；灵界元婴为上限，化神为世界顶点；凡界练气大圆满为上限。
- [ ] **压制**：凡界内筑基及以上被压制至练气大圆满；坠星秘境内所有人压制至练气大圆满；浊修入灵界/凡界同理受压制。
- [ ] **死亡归宿**：三界修者身死均为「修为化灵/浊气回馈对应天地，**灵魂入地府轮回**」（除非魂飞魄散）；见 `worldview.md`、`underworld_soul_system.md`。
- [ ] **武道**：凡界有外劲/内劲/化境/宗师，与练气修士在凡界可抗衡；见 `power_systems.md` §1。

### 4.4 时间线

- [ ] **当前时间**：剧情锚点为**大衍景帝三十五年春**（坠星湖异象指引秘境开启、特别异象重宝出世，灵衍天命指引秘境内可提升林微资质）。  
- [ ] **跨度**：主线自林微觉醒至终局约**五百年**（见 `worldview.md` §9 各卷分配）。  
- [ ] **历史事件**：上古/中古/近古/当世归属见 `timeline.md`；大衍立国属近古，七域格局、苍梧封印等属中古或近古。

### 4.5 命名与称谓

- [ ] **人名一致性**：同一人物全名/昵称/他称在全文统一；新角色取名可参考 `world/naming/`。  
- [ ] **地点名**：与 `world_map.md`、`location_*.md` 一致；新地点需先定界域与州/郡/域再写入正文。  
- [ ] **禁忌词**：严禁电压、监控、数据、逻辑核心、像素、代码、权限等级、洗浴中心、能量棒、赛博等；用灵压、神识、道则/本源、命理、法印、灵脉等替代（`constitution.md` §3、§8.4）。

### 4.6 功法、法宝与修士设定（可选）

- [ ] **功法与路线/境界**：写某角色功法时，与 `功法套路全览.md` 中该路线、境界一致；跨境功法可跨多境。
- [ ] **法宝与等级/势力**：写法宝时与 `法宝套路全览.md` 等级、装备栏、适用境界一致；本命器、成长性以该文件标注为准。
- [ ] **境界机缘与通道**：写突破机缘时，若涉及通道或地点，与 `world_map.md` 一致：**天门关**仅为凡界—灵界（朔方州—烛阴国）；灵界—浊界为**神农封印**（苍梧）、**极北裂缝**（凛渊域）；不可出现「浊界天门关」。
- [ ] **修士类型与势力**：写角色主修类型时，与 `修士类型全分类.md`、`factions.md` 对应（如紫霄法脉为法修、天枢剑宗为剑修）。

## 五、常见矛盾与禁例（避免再现）

以下问题已在一致性检查中修正或明确，写作时**不要再次出现**：

| 类型 | 错误示例 | 正确做法 |
| :--- | :--- | :--- |
| 地理 | 天门关在天京 / 京兆郡 | 天门关在**朔方州·烛阴郡**；天京仅写「驿道北通天门关」 |
| 地理 | 坠星湖在中州 | 坠星湖在**东莱州·坠星郡**，青阳城东 |
| 设定 | 浊界是「灵界的一部分」或「独立位面、物理隔离」 | 以 `worldview.md` 为准：凡、灵、浊**同处一方天地**，以**结界与规则**及**灵气/浊气浓度**区隔；浊界**非独立位面**，与灵界有**交接地带与争夺区域**（如苍梧、极北裂缝） |
| 称谓 | 紫霄法脉为「道修」 | 统一为**法修** |
| 通道 | 天门关连接对象未写明 | 凡界天门关通往灵界凛渊域**烛阴国**（人族政权）；七域文档中凛渊域位置已写明烛阴国 |
| 时间 | 时间线未体现剧情跨度 | 当世与元数据中已注明主线约 500 年，与 worldview §9 一致 |
| 通道 | 「浊界天门关」等混淆通道归属 | 天门关为**凡界—灵界**（朔方州—烛阴国）；灵界—浊界为**神农封印**、**极北裂缝**；机缘表中已修正为「灵浊关口」 |

**新增设定时**：若新增通道、关口、地点、势力，请在 `world_map.md`（或对应域/州）、`worldview.md`（若为屏障通道）、`factions.md`（若为势力）中同步更新，并考虑在 `世界观与时间线一致性检查与修订说明.md` 中补一笔，避免日后矛盾。

## 七、使用本指南的推荐流程

1. **动笔前**：根据本节内容确定要写的界域、时代、涉及势力与地点；打开对应权威文档（见 §三）扫一眼关键条目。  
2. **写含感情戏/床戏的章节时**：**必须先**阅读参考文（`references/入海.txt` 等）相关段落 + **`references/感情戏与床戏写法学习笔记.md`**，再动笔；不得跳过此步。**涉及床戏时**：宪法要求提供**两个版本**——**【PO18 版】**（露骨涩情）与**【过审版】**（唯美可过审），两版本**同时在正文中出现并标明**（见 `constitution.md` §8.10 床戏双版本规定）。详见该学习笔记开头的「使用前必读」。  
3. **写作中**：涉及地名、势力名、境界、通道时，优先用本指南 §六 快速表核对；不确定则查 §1.1 对应文件。  
4. **成稿后/修订时**：用 §四 清单逐项过一遍（地点归属、人物势力、力量规则、时间、命名与禁忌词）。  
5. **扩展设定时**：新地点/势力/通道在对应 world 文档中同步更新，并参考 §五 禁例与「世界观与时间线一致性检查与修订说明」的维护建议。  
6. **总览项目时**：执行 §八 总览与冲突检查，确保多文档表述一致。

## 八、总览项目与冲突检查

在**总览项目**、**大改设定**或**批量扩展**后，建议执行以下冲突检查，并在发现矛盾时以 §三 权威文档为准修正。

### 8.1 必查项

| 检查项 | 权威/对照 | 易冲突点 |
| :--- | :--- | :--- |
| 三界本质 | `worldview.md`、`world_map.md` | 凡、灵、浊**同处一方天地**，以结界与浓度区隔；**不得**写「浊界为独立位面」或「浊界为灵界的一部分」 |
| 通道归属 | `world_map.md` 重要节点表、§六 快速参考表 | 天门关＝凡界↔灵界（朔方州—烛阴国）；灵界↔浊界＝神农封印、极北裂缝；**不得**出现「浊界天门关」 |
| 地理归属 | `world_map.md` | 坠星湖在东莱州坠星郡；天门关在朔方州烛阴郡；天京在京兆郡 |
| 势力称谓 | `factions.md`、`power_systems.md` | 紫霄法脉为**法修**；玄门九宗属灵界 |
| 境界与寿命 | `power_systems.md` | 凡界练气大圆满上限；灵界元婴上限；压制规则一致 |

### 8.2 扩展设定文档一致性

| 文档 | 核对要点 |
| :--- | :--- |
| `境界机缘备选·凡灵妖.md` / `境界机缘备选·浊邪阴.md` | 机缘中的**地点、通道名**与 `world_map.md` 一致；已修正「浊界天门关」→「灵浊关口」 |
| `功法套路全览.md`、`法宝套路全览.md` | 适用境界与 `power_systems.md` 一致；势力名与 `factions.md` 一致 |
| `世界观与时间线一致性检查与修订说明.md` | §3 已执行修订清单中「浊界本质」表述为**同处一方天地**（非独立位面）；若后续修订请同步更新本指南 §五、§八 |

### 8.3 发现冲突时的处理

1. **确定权威**：按 §三 表确定该主题的权威文档。  
2. **修正非权威**：修改与权威矛盾的文档（或本指南禁例表），使表述与权威一致。  
3. **记录**：若属地理/通道/三界本质类，建议在 `世界观与时间线一致性检查与修订说明.md` §2 续增问题与 §3 续增修订项，并更新本指南 §五 禁例或 §八。
