# 感情戏与床戏写法学习笔记

**说明**：根据《入海》《夺妻》《魔界妖女她拿下正道之光了》三本参考书摘录与归纳，供写作/润色时查阅。**宪法要求（见 constitution.md §8.10）**：涉及床戏时须提供**两个版本**——**【PO18 版】**（具体、露骨、非常涩情）与**【过审版】**（唯美色气、艺术感、可过审），两版本**同时在正文中出现并标明**；感情戏**含蓄**，用神态、动作、环境反衬。

---

## 使用前必读：写含感情戏/床戏的章节时

**流程（必须遵守）**：**先读参考 + 本笔记 → 再动笔。**

1. **动笔前**：若本章会含感情戏或亲密戏/床戏，必须先完成：
   - 阅读 `references/入海.txt` 中相关感情/亲密段落（可搜索「吻」「相拥」「缠绵」「呼吸」等定位）；
   - 视需要阅读 `references/夺妻.txt`、`references/魔界妖女她拿下正道之光了（扶她高h）.txt` 中相关段落；
   - 通读本笔记全文，尤其是「可操作总结」表。
2. **写作时**：涉及床戏须写**两个版本**——【过审版】按本笔记要点与《入海》式含蓄；【PO18 版】可具体露骨、直写器官与行为细节。两版本在正文中并列并标明「【过审版】」「【PO18 版】」。
3. **第七章《乱葬岗超度》**不含床戏，已保持正文不动；本流程自后续章节起执行。

## 一、《入海》（首要参考·修真唯美）

### 1. 感情戏要点

- **少直白心理**：不写「她心想：我喜欢她」，而是用**神态、动作、环境**传达。
  - 例：蓝妩跑走后回来找季泠月，写「季泠月眼梢微红，快步上前握住她的手腕，急声问：你去哪里了？」——焦急与在乎全在动作和台词里。
  - 例：季泠月害羞时写「又很快撇开视线」「面皮发烫地垂下眼」「视线总是极快地从她身上扫过，扭扭捏捏」。
- **事后情绪**：亲密之后用**小动作、对话、身体反应**收束，不总结「她爱上了」。
  - 例：季泠月「泪眼朦胧」「对不起，把你弄脏了」；蓝妩「心道她怎么这么爱说对不起」「她这么善解人意，蓝妩倒更心虚了」。
- **节奏留白**：重要情节点前后可缓笔，用环境或日常一笔带过，再切入下一段。
  - 例：云消雨歇后「蓝妩抱着人泡进热水里洗了洗」「蓝妩躺在她旁边，却有些失眠」「身旁的人却翻了个身，把脑袋贴到了蓝妩肩膀上」「放轻呼吸，视线描摹着季泠月的五官轮廓」。

### 2. 亲密戏/床戏要点（偏唯美、可过审）

- **用体温、触感、呼吸代替直白器官名**：「滚烫的温度」「湿濡的腿心」「柔软的蚌肉」「穴肉黏糊糊裹在指节上」「小腹一片潮意」「潮湿的眼角」。
- **吻的层次**：先写「在她唇角/下巴/眼角亲了下」，再写「贴上自己的唇」「轻启红唇，小心翼翼舔舐起唇瓣」「舌尖钻入她口中，与她胡乱纠缠」；**接吻与心理挂钩**（如蓝妩「接吻这种事，相爱之人才能做」的纠结，再「罢了，都滚到床上了」的让步），增加重量。
- **节奏**：前戏（抚触、腿心、穴口）→ 进入/抽送（可略写「抽送」「插了几十来次」）→ **高潮用身体反应与声音**：「脚趾绷直又蜷起」「难耐地发出一声绵软的呻吟」「腿间倏地涌出一股热流」「眼泪啪嗒啪嗒落下」；**事后**必有一笔：擦拭、抱紧、对话（「还要吗？」「不要了……」），或一方「小心抱住蓝妩的腰肢，慢慢闭上眼睛」。
- **情绪与身体交织**：不只写「很爽」，写「季泠月难堪地呜咽一声，颤抖着缩到蓝妩怀里」「哽咽道：我不想吵醒你的」「蓝妩顿了一下，心里软的一塌糊涂，她低下头亲吻季泠月潮湿的眼角」。

## 二、《夺妻》（补充参考·古风权谋+亲密）

### 1. 感情戏要点

- **权力与占有**：强势方的欲望用**动作与台词**表达——「卿卿想碰我了？」「卿卿，我想你肏我了」；被动方从抗拒到动摇，用**身体诚实**（「沈静姝慢慢地竟然也开始回吻」）与**羞耻/愧疚**（「满面羞红地求助」「又羞又愧」）表现。
- **称呼与亲密**：小字「卿卿」、昵称「衿儿」反复出现，制造专属感；**请求与哄**（「舔舔我」「快帮我舔舔～」「我的小美人，好卿卿」）拉近距离。

### 2. 床戏要点（偏细节、可酌情收敛）

- **步骤分明**：从「捧住/刮乳尖→舌舔乳→指揉花核→指入穴」一层层写，**每步有对方反应**（「思不归即刻发出愉悦地喘息」「卿卿……学得好快～」）。
- **主动方指导被动方**：「用手捧着它，先用手指刮一刮前头」「用舌头舔舔，绕着凸起的地方打转」「来，揉揉它，重一点地揉它！」——适合「第一次/生涩」的剧情，既有教学感又带色气。
- **心理满足**：「沈静姝注视着对方脸上的潮红，突然也有些不一样的快感——来自心理的，取悦对方之后的满足」——**情感与身体双线**，可借鉴。
- **过审处理**：可多用「花处」「腿心」「小穴」「花核」「玉乳」「乳尖」等雅俗之间的词，极端露骨处改为「湿热」「紧致」「翕合」等概括，或略写「抽送」「进出」带过。

## 三、《魔界妖女她拿下正道之光了》（补充参考·仙侠+高h）

### 1. 感情戏要点

- **恨与欲交织**：女主梦境里「报复」与「占有」合一——「她身躯和心灵受尽折磨侮辱的痛苦又岂是一剑杀了她能偿还的？」「真美。就像她当时在溪水边，用神识看到的一样美」；**执念用动作释放**（掐脖、撕衣、吻、抚触），不先长篇心理。
- **台词短而有力**：「师妹原来这么喜欢啊？嗯？」「说谎！」「荡妇！」——权力感与情欲用少量对白完成。

### 2. 床戏要点（尺度大，仅作节奏与结构参考）

- **场景与身份**：牢房、锁链、嫁衣/里衣撕裂，**身份差**（师姐/师妹、施罚/受罚）强化张力。
- **顺序**：吻唇→颈/胸→腰→腿心→指入→（扶她设定略）；**每步有少女反应**（「呜咽」「娇泣」「锁链鸣响」「渐渐陷入意乱情迷」）。
- **过审借鉴**：若需「唯美过审」，可只取「节奏与层次」「从吻到胸到腰到腿心的推进」「用呻吟、颤抖、眼泪、余韵收束」，**具体器官与动作可换成《入海》式的体温、湿意、紧握、呼吸**。

## 四、可操作总结（写正文时直接用）

| 类型     | 建议 |
|----------|------|
| **感情戏** | 用神态、动作、环境反衬；少「她心想」；重要情绪用一句台词或一个动作点破（如「你去哪里了？」）；事后留白（沉默、抱紧、视线描摹）。 |
| **吻戏**   | 分层次：唇角/眼角/下巴→唇瓣相贴→舔舐→深吻/舌缠；可夹一句心理或犹豫（如「接吻这种事，相爱之人才能做」）增加重量。 |
| **前戏**   | 触感优先：体温、湿意、柔软、紧绷；用「腿心」「腰肢」「脊背」「指尖」等；每步带出对方反应（喘、颤、呜咽）。 |
| **正戏**   | 可略写「抽送」「进出」；高潮写**身体反应**（脚趾蜷起、腰拱起、颤抖、热流、眼泪）和**一声呻吟或半句台词**；避免器官直名与过细动作。 |
| **事后**   | 必有一笔：擦拭/抱紧/盖被；简短对话（「还要吗？」「睡吧」）；或一方默默抱腰/贴肩，收束情绪。 |
| **过审**   | 以《入海》的用词与留白为准；《夺妻》《魔界妖女》取节奏与层次，用词往「雅、略隐」靠拢。 |
| **双版本** | 宪法强制：床戏须同时提供【过审版】与【PO18 版】，正文中标明版本；【过审版】用上表要点，【PO18 版】可露骨直写器官与细节。 |
