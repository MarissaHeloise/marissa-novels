# 文风参考·引用与摘录

本文件夹放在**项目工作区内**，AI 可直接读取，用于文风与设定参考。

## 为什么之前读不到？

- 你添加的 references（`入海.txt` 等）在 Cursor 的 **「工作区外」**（路径是 `../references/`，即项目上一级目录）。
- 当前工具只能读取 **工作区根目录及其子目录**（`d:\novels\my-novel\` 下）的文件，所以无法直接打开外面的 `入海.txt`。

## 怎么才能让我直接读到？

把需要参考的内容放到**本文件夹**（或任意 `my-novel` 下的子目录）里即可：

1. **方式一：复制摘录进 Markdown（推荐）**  
   - 打开 `入海·文风摘录.md`，把《入海》里你喜欢的段落粘贴进去（可分批粘贴）。  
   - 我会按这个文件来学文风，不必整本书都贴。

2. **方式二：整本复制**  
   - 把《入海》全文贴进 `入海·全文.md`（或你新建的 .md），我也可以读，但文件会很大。

3. **方式三：把原 references 挪进项目**  
   - 把 `入海.txt` 等从 Cursor 的 references 位置**复制或移动**到本文件夹 `d:\novels\my-novel\references\`，我就能用路径直接读。

| 文件 | 用途 |
|------|------|
| `README.md` | 本说明 |

| （可选）`夺妻·摘录.md`、`魔界妖女…·摘录.md` | 另两本参考的摘录，需要时可自建 |
