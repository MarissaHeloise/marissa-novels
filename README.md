# marissa-novels
marissa-novels
<br>尝试了用开源的novel-tool-kit包来创建AI小说项目，但是效果不太行，需要人的大量重修。<br>
但是能够快速生成一个基本的大纲。<br>
其原理是创建cursor或者其他编辑器的slash命令集，或者是skills。<br>
建议直接编辑skills。<br>
缺点：生成的东西不太能看，如果有reference的话，就会好很多，很多情况下表现不如豆包。<br>
而豆包则有严重的上下文不一致的问题。<br>
cursor的会员要钱，trae目前是免费的，但是大白话文多，生成的内容质量也比较差。<br>
trae慢慢的也需要排队了……<br>
个人感觉如果想要生成精品，那还是需要作者的超级大量投入，不是程序员可以做的事。<br>
目前国内的平台也是，很多时候让人会心一笑的，不是设定或者故事本身，而是作者本身的见闻和幽默感。<br>
如果你自己都不觉得这个故事有意思，没有表达的欲望，那你的作品自然也不会精彩。<br>
如果只是为了赚钱，投送到番茄小说之类的平台上的话，大部分的AI生成小说或者比较正经的不抓眼球的小说，基本上没有什么流量。<br>
十几天之后，平台看你数据不行，还会突然把你流量全部砍掉，你白白给它贡献版权。<br>
至于其他尝试，比如说多文炼丹，我没有试过，感兴趣的同学可以尝试一下。<br>
目前个人使用者来说，感觉还是差点意思。<br>
