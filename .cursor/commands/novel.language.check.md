# 小说语言规范检查 (Language Compliance Check)

## User Input

```text
$ARGUMENTS
```

可选：传入要检查的路径，如 `chapters/volume1` 或单章路径。留空则检查全项目正文与设定中的用语。

## 执行步骤

1. **读取宪法语言规范**  
   从 `.novelkit/memory/constitution.md` 的 **8.4 语言规范** 中取得：
   - **严禁出现**：电压、监控、数据、逻辑核心、像素、代码、权限等级、洗浴中心、能量棒、赛博
   - **替代词汇**：灵压、天眼/神识、道则/本源、道痕碎片、尘埃/幻象、命理、天命眷顾/法印、灵脉/仙髓

2. **确定检查范围**  
   - 若 `$ARGUMENTS` 为空：检查 `chapters/`、`world/` 下所有 `.md`（不含 `.novelkit` 模板与命令本身）
   - 若传入路径：仅检查该路径下的 `.md` 文件

3. **扫描禁词**  
   在范围内搜索上述严禁词（含常见变体，如：全息、赛博朋克、数据流、监控全城、权限、能量棒等），并记录出现位置（文件路径、行号、当前句或片段）。

4. **生成报告**  
   - 列出所有命中项，并给出建议替换（按 8.4 替代词汇或修真向表述）
   - 若无命中，输出「未发现禁词，符合宪法 8.4 语言规范」

## 输出格式

```markdown
# 语言规范检查报告

**检查时间**：YYYY-MM-DD
**范围**：[全项目 / 或具体路径]

## 结果摘要
- 检查文件数：N
- 发现禁词/不当用语：M 处

## 问题列表（若有）

| 文件 | 行号 | 当前用语 | 建议替换 |
|------|------|----------|----------|
| chapters/volume1/chapter-001.md | 42 | 监控 | 观测/神识 |
| world/settings/world_map.md | 64 | 全息幻术 | 大型幻术/灵纹幻象 |

## 建议
（针对每类禁词的一两句替换建议，或「无问题」）
```

## 说明

- 本命令仅做**用语检查**，与 `/novel.constitution.check` 的全面合规检查互补。
- 检查时以宪法 8.4 为准；若宪法中 8.4 有更新，以最新内容为准。
- 对设定文档中的「严禁词」也应替换为修真向表述，以保持与正文一致。
